# Shared Blocks

This directory contains **shared blocks** that are used across ALL features/branches.

## What belongs here:
- ✅ Hero - Used by all features
- ✅ Content - Used by all features
- ✅ FAQ - Used by all features
- ✅ Features - Used by all features
- ✅ MediaBlock - Used by all features
- ✅ ContactFormBlock - Used by all features
- ✅ And other SHARED blocks

## What does NOT belong here:
- ❌ Ecommerce-specific blocks → `src/branches/ecommerce/blocks/`
- ❌ Construction-specific blocks → `src/branches/construction/blocks/`

See: `docs/SRC-CLEANUP-MASTER-PLAN.md` for architecture details.
