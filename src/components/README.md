# Shared Components

This directory contains **shared components** used across ALL features/branches.

## What belongs here:
- ✅ Header - Global site header
- ✅ Footer - Global site footer
- ✅ RichText - Shared rich text renderer
- ✅ Grid - Shared layout grid
- ✅ Analytics - Global analytics
- ✅ And other SHARED components

## What does NOT belong here:
- ❌ Ecommerce-specific components → `src/branches/ecommerce/components/`
- ❌ Platform-specific components → `src/branches/platform/components/`
- ❌ Construction-specific components → `src/branches/construction/components/`

See: `docs/SRC-CLEANUP-MASTER-PLAN.md` for architecture details.
