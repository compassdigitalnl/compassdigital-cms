Build Instructions
// ═══ BUILD INSTRUCTIONS FOR DEVELOPERS / CLAUDE ═══

1. DESIGN TOKENS
// Always use CSS custom properties from Chapter 1
// Never hardcode colors — use var(--teal), var(--navy), etc.
// Spacing must follow 4px grid: 4, 8, 12, 16, 24, 32, 48, 64, 80px

2. TYPOGRAPHY
// Primary body: 'Plus Jakarta Sans' (import from Google Fonts)
// Display headings: 'DM Serif Display' (serif contrast)
// Code/SKUs: 'JetBrains Mono'
// Never use font-size below 9px or above 36px

3. COMPONENTS
// Every component must have: hover state, focus state, disabled state
// Use transition: all var(--dur) var(--ease) for all interactive elements
// Cards: 1px border var(--grey), hover = shadow-md + translateY(-2px)
// Buttons: always border-radius: var(--r-sm), min-height 36px

4. RESPONSIVE
// Mobile-first approach
// Grids: 4-col → 2-col at 900px → 1-col at 640px
