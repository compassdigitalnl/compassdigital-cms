## Pagina-opbouw (Template)

Elke branche-hoofdpagina en niche-pagina volgt dezelfde conversie-geoptimaliseerde structuur:

### Sectie 1: Hero
- **Kop:** Branche-specifiek, zoekwoord in H1
- **Subkop:** Concreet resultaat/belofte
- **CTA:** "Bekijk demo" + "Plan een gesprek"
- **Visual:** Screenshot of mockup van de branche in actie
- **Trust badge:** "Geen technische kennis nodig" / "Live binnen 2 weken"

### Sectie 2: Pijnpunten (Problem Agitation)
- 3-4 herkenbare problemen van de doelgroep
- Bijv. Kapsalon: "Afspraken via WhatsApp? Klanten die niet opdagen?"
- Ikoon + korte tekst per probleem
- Emotionele trigger â†’ herkenning

### Sectie 3: Oplossing (Feature Highlights)
- 4-6 features specifiek voor deze branche
- Per feature: screenshot + korte uitleg + resultaat
- Bijv. Elektricien: "Online offerte-aanvragen â†’ 3x meer leads"
- Afwisselend links/rechts layout

### Sectie 4: KPI's / Resultaten
- 3-4 concrete cijfers
- Bijv. E-commerce: "Gem. 34% meer omzet" / "47% minder retourvragen"
- Bijv. Tandarts: "68% minder no-shows" / "24/7 online afspraken"
- Grote nummers, klein label â€” visueel sterk

### Sectie 5: Hoe het werkt (3-stappen)
1. "Kies je template" / "Plan een gesprek"
2. "Wij richten alles in" / "Content uploaden"
3. "Ga live" / "Eerste klant binnen"
- Eenvoud benadrukken â€” geen technische drempel

### Sectie 6: Pricing
- Relevante tier uitlichten (bijv. E-commerce â†’ B2C â‚¬149/mo)
- "Alles inbegrepen" benadrukken
- Vergelijking: "vs. Shopify + 15 plugins = â‚¬400/mo"
- CTA: "Start gratis proefperiode" of "Plan een demo"

### Sectie 7: Branche-specifieke Testimonial
- Citaat van klant in dezelfde branche
- Foto, naam, bedrijfsnaam, resultaat
- Indien geen echte klant: "Bekijk hoe [bedrijf] het doet" â†’ demo-link

### Sectie 8: Vergelijking met Alternatieven
- Tabel: CompassDigital vs. Shopify / WordPress / Wix / branche-specifiek
- Focus op: prijs, features, support, maatwerk
- Eerlijk maar voordeliger positioneren

### Sectie 9: FAQ
- 5-8 branche-specifieke vragen
- Structured data (FAQ schema) voor SEO
- Bijv. "Kan ik mijn WooCommerce shop migreren?" / "Hoe werkt het met mijn kassa?"

### Sectie 10: CTA (Afsluitend)
- Urgentie: "Start deze week nog"
- Social proof: "Sluit je aan bij 500+ bedrijven"
- Twee opties: "Gratis demo" + "Direct starten"

## SEO Strategie

### On-Page SEO (per pagina)

| Element | Implementatie |
|---------|--------------|
| **Title tag** | `[Niche] Website & Software - CompassDigital` (max 60 chars) |
| **Meta description** | Resultaatgericht, met CTA, max 155 chars |
| **H1** | Exacte zoekterm, 1x per pagina |
| **H2's** | Variaties en long-tail zoektermen |
| **URL** | Clean, kort, zoekwoord (`/bouw/elektricien`) |
| **Internal links** | Naar hoofdbranche, naar pricing, naar gerelateerde niches |
| **Image alt** | Beschrijvend met zoekwoord context |
| **Schema markup** | `FAQPage`, `Product` (voor pricing), `BreadcrumbList` |
| **Canonical** | Self-referencing canonical |
| **Open Graph** | Branche-specifieke social preview |

### Content Strategie

**Per niche-pagina:**
- 800-1.200 woorden unieke content
- Niet generiek â€” specifieke pijnpunten en features per niche
- Interne links naar:
  - Hoofdbranche pagina (parent)
  - Andere niches in dezelfde branche (siblings)
  - Pricing pagina
  - Demo aanvraag

**Ondersteunende content (blog):**
- Per branche 3-5 blogposts die linken naar de landing page
- Bijv. "5 Tips voor een Succesvolle Kapsalon Website" â†’ linkt naar `/beauty/kapsalon`
- Long-tail keywords vangen

## Conversie Strategie

### Primaire Conversie: Demo Aanvraag
- Formulier: Naam, Email, Telefoon, Branche (pre-filled), Bericht
- Tracking: UTM + branche + niche in form submission
- Follow-up: Automatische bevestigingsmail + telefonisch contact < 4 uur

### Secundaire Conversie: Gratis Proefperiode
- 14 dagen gratis, geen creditcard nodig
- Pre-configured met branche-template
- Onboarding wizard met branche-specifieke stappen

### Micro-conversies
- Nieuwsbrief aanmelding
- Whitepaper / brancherapport download
- "Bereken je besparing" calculator
- Live chat (chatbot)

### Tracking & Analytics

| Event | Tracking |

| Form abandonment | GA4 event op blur |

## Technische Implementatie

### Optie A: CMS-gebaseerd (aanbevolen)
Gebruik het eigen Payload CMS met `Pages` collection:

1. Maak een `branch-landing` page layout/template
2. Gebruik bestaande blocks: Hero, Features, Stats, Testimonials, FAQ, Pricing, CTA
3. Voeg toe: `ComparisonTable` block, `PainPoints` block, `HowItWorks` block
4. Elke pagina = een Page record in het CMS
5. Content bewerkbaar door marketing team zonder developer

**Voordelen:** Snel aanpasbaar, A/B testbaar, geen deploy nodig voor content wijzigingen.
