Step 1: Payload Block Configuration

Create src/branches/blocks/ecommerce/product/ProductGrid/config.ts with the following configuration:

// src/branches/blocks/ecommerce/product/ProductGrid/config.ts import { Block } from 'payload/types' export const ProductGrid: Block = { slug: 'productgrid', labels: { singular: 'Product Grid', plural: 'Product Grids', }, fields: [ // Section Title (optional) { name: 'title', type: 'text', label: 'Section Title', admin: { description: 'Optional heading boven de product grid (bijv: "Populaire Producten")', }, }, // Grid Configuration { type: 'row', fields: [ { name: 'columns', type: 'select', label: 'Aantal Kolommen', defaultValue: '4', options: [ { label: '2 kolommen', value: '2' }, { label: '3 kolommen', value: '3' }, { label: '4 kolommen (standaard)', value: '4' }, ], admin: { width: '33%', }, }, { name: 'limit', type: 'number', label: 'Maximum Aantal Producten', defaultValue: 8, min: 1, max: 24, admin: { width: '33%', description: 'Max 24 producten tegelijk', }, }, { name: 'sortBy', type: 'select', label: 'Sortering', defaultValue: 'manual', options: [ { label: 'Handmatig (volgorde selectie)', value: 'manual' }, { label: 'Nieuwste eerst', value: 'newest' }, { label: 'Prijs laag-hoog', value: 'price-asc' }, { label: 'Prijs hoog-laag', value: 'price-desc' }, { label: 'Populariteit', value: 'popular' }, ], admin: { width: '33%', }, }, ], }, // Product Source { name: 'source', type: 'radio', label: 'Product Bron', defaultValue: 'manual', options: [ { label: 'Handmatig selecteren', value: 'manual' }, { label: 'Categorie', value: 'category' }, { label: 'Tag', value: 'tag' }, { label: 'Nieuwste producten', value: 'recent' }, ], required: true, admin: { description: 'Kies hoe producten worden geselecteerd', }, }, // Manual Product Selection { name: 'products', type: 'relationship', label: 'Producten', relationTo: 'products', hasMany: true, admin: { condition: (data) => data.source === 'manual', description: 'Selecteer producten handmatig. Step 2: TypeScript Type Definitions

Create type-safe interfaces for the ProductGrid block:

// src/branches/blocks/ecommerce/product/ProductGrid/types.ts import { Product, Category, Tag } from '@/payload-types' export interface ProductGridBlock { blockType: 'productgrid' title?: string columns: '2' | '3' | '4' limit: number sortBy: 'manual' | 'newest' | 'price-asc' | 'price-desc' | 'popular' source: 'manual' | 'category' | 'tag' | 'recent' products?: (string | Product)[] category?: string | Category tag?: string | Tag showCta?: boolean ctaLabel?: string ctaUrl?: string } export interface ProductGridProps { block: ProductGridBlock }
Step 3: Usage in Next.js Components

React component implementation voor rendering van de product grid:

// src/branches/blocks/ecommerce/product/ProductGrid/Component.tsx import { ProductGridProps } from './types' import { getPayloadHMR } from '@payloadcms/next/utilities' import { ProductCard } from '@/components/ecommerce/ProductCard' export async function ProductGrid({ block }: ProductGridProps) { const payload = await getPayloadHMR({ config }) // Fetch products based on source type let products = [] if (block.source === 'manual') { // Use manually selected products products = await payload.find({ collection: 'products', where: { id: { in: block.products }, }, limit: block.limit, }) } else if (block.source === 'category') { // Fetch products from category products = await payload.find({ collection: 'products', where: { category: { equals: block.category }, }, limit: block.limit, sort: getSortField(block.sortBy), }) } else if (block.source === 'tag') { // Fetch products with tag products = await payload.find({ collection: 'products', where: { tags: { contains: block.tag }, }, limit: block.limit, sort: getSortField(block.sortBy), }) } else if (block.source === 'recent') { // Fetch recent products products = await payload.find({ collection: 'products', limit: block.limit, sort: '-createdAt', }) } return ( <section className="product-grid-block"> {block.title && ( <h2 className="section-title">{block.title}</h2> )} <div className="product-grid" data-columns={block.columns} > {products.docs.map((product) => ( <ProductCard key={product.id} product={product} /> ))} </div> {block.showCta && ( <a href={block.ctaUrl} className="cta-button" > {block.ctaLabel} </a> )} </section> ) } // Helper function for sort field mapping function getSortField(sortBy: string) { switch (sortBy) { case 'newest': return '-createdAt' case 'price-asc': return 'price' case 'price-desc': return '-price' case 'popular': return '-views' default: return '-createdAt' } }
Step 4: Admin Panel Structure

When you add this block to a Page's layout field, you will see:

📦 Product Grid Block
▸ Section Title (text, optional)
- Placeholder: "Populaire Producten"
▸ Grid instellingen (row met 3 velden)
- Aantal Kolommen: dropdown (2/3/4) → default: 4
- Maximum Aantal: number input (1-24) → default: 8
- Sortering: dropdown → default: manual
▸ Product Bron (radio buttons)
- ⚪ Handmatig selecteren
- ⚪ Categorie
- ⚪ Tag
- ⚪ Nieuwste producten
▸ Conditionele velden (afhankelijk van bron)
- Als "Handmatig": relationship field → select meerdere products
- Als "Categorie": relationship field → select 1 category
- Als "Tag": relationship field → select 1 tag
- Als "Nieuwste": geen extra velden
▸ CTA Button (checkbox + conditional row)
- ☑ Toon "Bekijk alle" knop
- Als checked: Button Tekst + Button Link velden verschijnen
Step 5: Testing Checklist
1. Block beschikbaar: Navigate to Pages → Edit page → Add block → "Product Grid" verschijnt in lijst
2. Manual selection werkt: Kies "Handmatig selecteren" → relationship field toont products → selecteer 4 producten
3. Category filter werkt: Kies "Categorie" → category dropdown verschijnt → select category → producten worden gefilterd
4. Column variants: Test 2, 3, en 4 kolommen → grid layout past aan op frontend
5. Responsive grid: Test mobile/tablet → 4 cols wordt 2 cols @900px, 1 col @640px
6. CTA button: Enable CTA → vul tekst en URL in → button verschijnt onder grid op frontend
7. Empty state: Kies category zonder producten → grid toont "Geen producten gevonden" message

Step 7: Common Issues & Solutions
Issue Solution
Block niet zichtbaar in admin Check payload.config.ts → blocks array in Pages collection → moet ProductGrid bevatten
Relationship fields tonen geen products Verify products collection bestaat en relationTo: 'products' slug klopt
Conditional fields verschijnen niet Check admin.condition functie → data.source moet exact matchen met radio value
Grid toont geen producten op frontend Check Component.tsx → payload.find() query → log products.docs om data te zien
TypeScript errors op block.products Regenerate types: npx payload generate:types → import ProductGridBlock type
Database error bij save Run migration: npx payload migrate → check migration file bevat productgrid block type
Responsive grid werkt niet Check CSS: @media (max-width: 900px) en 640px breakpoints → grid-template-columns values
