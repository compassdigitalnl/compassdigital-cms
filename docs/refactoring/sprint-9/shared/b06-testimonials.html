5 Testing Checklist
  • Star Rating Display: Test all ratings (1-5 stars). Verify filled stars are amber, empty stars are grey. Test fractional ratings if needed (3.5 stars).
  • Avatar Fallbacks: Test with avatar image, without avatar (initials show). Verify initials use first 2 letters of author name. Test teal background for fallback.
  • Grid Responsiveness: Test grid layout: 3 columns on desktop (>1024px), 2 columns on tablet (768-1023px), 1 column on mobile (<768px). Verify consistent card heights.
  • Carousel Navigation: Test prev/next buttons cycle through testimonials. Verify wrapping (last → first, first → last). Test pagination dots click to specific slide.
  • Featured Layout: Test featured variant shows first testimonial only. Verify dark navy gradient background, larger text (24-32px), white text color.
  • Long Quotes: Test with short quotes (50 chars), medium (150 chars), long (300+ chars). Verify cards maintain consistent heights in grid. Test text wrapping.
  • Accessibility: Test carousel navigation with keyboard (arrow keys). Verify ARIA labels on buttons. Test screen reader announces slide changes. Check focus states visible.

CREATE TABLE IF NOT EXISTS testimonials_blocks (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  block_id UUID NOT NULL REFERENCES blocks(id) ON DELETE CASCADE,

  title TEXT,
  variant VARCHAR(20) NOT NULL DEFAULT 'grid'
    CHECK (variant IN ('grid', 'carousel', 'featured')),

  created_at TIMESTAMPTZ DEFAULT NOW(),
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Create junction table for testimonials
CREATE TABLE IF NOT EXISTS testimonials_items (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  testimonials_block_id UUID NOT NULL REFERENCES testimonials_blocks(id) ON DELETE CASCADE,
  quote TEXT NOT NULL,
  author TEXT NOT NULL,
  role TEXT,
  avatar_id UUID REFERENCES media(id) ON DELETE SET NULL,
  rating INTEGER NOT NULL DEFAULT 5
    CHECK (rating >= 1 AND rating <= 5),
  order_index INTEGER NOT NULL DEFAULT 0,

  created_at TIMESTAMPTZ DEFAULT NOW(),

  CONSTRAINT unique_testimonial_order UNIQUE (testimonials_block_id, order_index)
);

-- Create indexes

7 Troubleshooting Guide
Avatar images not displaying
Cause: Next.js Image component requires width/height or domain whitelisting.
Fix: Verify avatar.url is absolute URL (https://...). Check next.config.js has image domain whitelisted. Test with width={40} height={40} props. Ensure avatar object exists before accessing .url.
Initials fallback showing wrong letters
Cause: author.slice(0, 2) takes first 2 characters, not initials.
Fix: For proper initials: author.split(' ').map(n => n[0]).join('').slice(0, 2). Test with "Jan Smit" → "JS", "Maria" → "MA". Handle empty names gracefully.
Carousel buttons not working
Cause: Client component not marked 'use client' or state not updating.
Fix: Verify TestimonialsCarousel.tsx has 'use client' at top. Check useState imported. Test prevSlide/nextSlide functions with console.log. Verify modulo math: (prev + 1) % length.
Grid cards have uneven heights
Cause: Different quote lengths cause varying card heights.
Fix: Add min-height to cards or use display: grid with grid-auto-rows: 1fr. Consider line-clamp for quotes (max 3-4 lines) with "Read more" expansion. Test with very short and very long quotes.
Stars not showing amber color
Cause: text-amber class not defined or Tailwind purging class.
Fix: Verify --amber: #F59E0B in CSS variables. Add to tailwind.config.js theme.colors. Test with text-[#F59E0B] if dynamic. Whitelist text-amber in safelist if purged.
Carousel navigation buttons outside viewport
Cause: translate-x-1/2 positions buttons half their width outside container.
Fix: Add mx-auto max-w-4xl wrapper with position: relative. Ensure parent has enough padding (px-6). Test on mobile (<640px) - consider hiding buttons, use swipe gestures instead.
Featured layout text too small on mobile
Cause: Fixed text-2xl/text-3xl not responsive enough.
Fix: Use responsive classes: text-xl md:text-2xl lg:text-3xl for quote. Test at 375px, 768px, 1024px widths. Reduce padding on mobile: p-6 md:p-10 lg:p-16.
Pagination dots not clickable
Cause: Buttons too small (8px) or z-index issues.
Fix: Add larger click area: p-2 to buttons (expands clickable area). Test touch targets min 44x44px for mobile. Verify onClick={() => setCurrentIndex(index)} fires. Check z-index relative to other elements.

Usage Guidelines
  • Quote Length: Keep quotes between 100-200 characters for optimal readability. Longer quotes work in featured layout. Short quotes (<50 chars) best for carousel.
  • Avatar Images: Upload square images (minimum 80x80px, recommended 200x200px). Supports JPG, PNG, WebP. If no avatar, initials show on colored background.
  • Star Ratings: Use 5 stars for excellent reviews. 4 stars acceptable. Avoid displaying 1-2 star reviews publicly (handle those privately). Ratings appear as filled (amber) or empty (grey) stars.
  • Layout Selection: Grid (default, best for 3-9 testimonials), Carousel (best for 5+ testimonials, interactive), Featured (single testimonial, high-impact hero section).
  • Author Details: Author name required. Role optional but recommended for credibility (include company). Format: "Name, Company" or "Name" and "Job Title, Company".
  • Grid Columns: Automatically responsive: 3 columns on desktop (>1024px), 2 on tablet (768-1023px), 1 on mobile (<768px). Cards have equal heights.
  • Carousel Features: Auto-cycles every 5 seconds (optional). Manual navigation with prev/next buttons. Pagination dots for direct access. Keyboard accessible (arrow keys).
  • SEO Benefits: Testimonials generate Review schema.org markup. Aggregate ratings shown in Google search. Improves trust signals and click-through rates.
