Font Families
Plus Jakarta Sans
Primary body font for Medisch B2B vertical. Modern geometric sans-serif. Weights: 400, 500, 600, 700, 800.
--font: 'Plus Jakarta Sans'
DM Serif Display
Display font for hero titles, section headers, and card titles. Elegant serif contrast.
--font-display: 'DM Serif Display'
JetBrains Mono
Monospace font for SKU codes, article numbers, pricing, and technical data.
--font-mono: 'JetBrains Mono'
Vertical Override: All verticals except Medisch B2B use DM Sans as body font instead of Plus Jakarta Sans. Theme system automatically handles this via data-theme attribute.

// src/types/theme.ts export interface ThemeTypography { // Font Families fontBody: string fontDisplay: string fontMono: string // Type Scale (sizes in px) heroSize: number sectionSize: number cardTitleSize: number bodyLgSize: number bodySize: number smallSize: number labelSize: number microSize: number } export interface Theme { id: string colors?: ThemeColors typography?: ThemeTypography // ← ADD THIS updatedAt: string createdAt: string }
Step 3: Usage in Next.js Components

Example: Fetch theme and generate CSS variables in your layout:

// app/layout.tsx import { getPayloadHMR } from '@payloadcms/next/utilities' import configPromise from '@payload-config' import { Theme } from '@/types/theme' export default async function RootLayout({ children }) { const payload = await getPayloadHMR({ config: configPromise }) // Fetch theme from Payload Global const theme = await payload.findGlobal({ slug: 'theme', }) as Theme // Generate CSS custom properties for typography const typographyVars = theme.typography ? ` --font: ${theme.typography.fontBody}; --font-display: ${theme.typography.fontDisplay}; --font-mono: ${theme.typography.fontMono}; --text-hero: ${theme.typography.heroSize}px; --text-section: ${theme.typography.sectionSize}px; --text-card-title: ${theme.typography.cardTitleSize}px; --text-body-lg: ${theme.typography.bodyLgSize}px; --text-body: ${theme.typography.bodySize}px; --text-small: ${theme.typography.smallSize}px; --text-label: ${theme.typography.labelSize}px; --text-micro: ${theme.typography.microSize}px; ` : '' return ( <html lang="nl"> <head> {typographyVars && ( <style dangerouslySetInnerHTML={{ __html: `:root { ${typographyVars} }` }} /> )} </head> <body>{children}</body> </html> ) }

Step 5: Testing Checklist
1. Typography tab exists: Navigate to /admin/globals/theme → Typography tab visible
2. All 11 fields visible: 3 font families + 8 type scale sizes
3. Default values set: Plus Jakarta Sans, DM Serif Display, JetBrains Mono + sizes
4. Number validation works: Try setting heroSize to 100 → should accept (within min/max)
5. CSS variables generated: Inspect <html> → see :root --font, --text-hero, etc.
6. Font changes reflect: Change heroSize to 48 → save → check frontend hero titles
7. TypeScript types work: No TS errors in components using theme.typography

Step 7: Common Issues & Solutions
Issue Solution
Typography tab not showing Check src/globals/Theme.ts → tabs array includes Typography import
Fields not showing in admin Restart dev server (npm run dev)
Fonts not loading on frontend Check Google Fonts CDN link in app/layout.tsx or add to public/fonts/
CSS variables not applying Check app/layout.tsx has typography CSS variable generation code
TypeScript errors on theme.typography Check src/types/theme.ts → ThemeTypography interface exists
Database error on save Run migration: npx payload migrate
Step 8: Vertical Theme Overrides

Multi-tenant font handling: All verticals except Medisch B2B use DM Sans as body font:

// styles/verticals.css // All verticals except medisch-b2b override the body font [data-theme="beauty"], [data-theme="bouw"], [data-theme="horeca"], [data-theme="zorg"], [data-theme="makelaar"], [data-theme="automotive"], [data-theme="professional"], [data-theme="events"], [data-theme="accommodatie"] { --font: 'DM Sans', system-ui, sans-serif; /* Override body font */ /* Display and mono fonts remain the same */ } // Medisch B2B keeps Plus Jakarta Sans (default) [data-theme="medisch-b2b"] { --font: 'Plus Jakarta Sans', 'DM Sans', system-ui, sans-serif; }
