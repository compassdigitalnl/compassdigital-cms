Average Score Calculation:

const calculateAverage = (distribution) => {
  const { fiveStar, fourStar, threeStar, twoStar, oneStar } = distribution;
  const total = fiveStar + fourStar + threeStar + twoStar + oneStar;
  const sum = (fiveStar * 5) + (fourStar * 4) + (threeStar * 3) + (twoStar * 2) + (oneStar * 1);
  return (sum / total).toFixed(1);
};

STEP 7: Implementation Notes

Real-time Data Fetching (Polling):

const [viewerCount, setViewerCount] = useState(0);

useEffect(() => {
  const fetchViewers = async () => {
    const res = await fetch(`/api/products/${productId}/viewers`);
    const data = await res.json();
    setViewerCount(data.count);
  };

  // Initial fetch
  fetchViewers();

  // Poll every 30 seconds
  const interval = setInterval(fetchViewers, 30000);

  return () => clearInterval(interval);
}, [productId]);

Websocket Alternative (Real-time):

useEffect(() => {
  const ws = new WebSocket('wss://api.example.com/live-stats');

  ws.onmessage = (event) => {
    const data = JSON.parse(event.data);
    if (data.productId === productId) {
      setViewerCount(data.viewers);
    }
  };

  return () => ws.close();
}, [productId]);
