B30

CallToAction (Inline)

Neutral

Inline CTA variant (simpler dan B3). Titel, beschrijving, 1 button. Light of teal background. Gebruik voor mid-page conversie punten.

  blockType: 'calltoaction'

⚛️ React Component

components/blocks/CallToAction.tsx 
import React from 'react'
import { CallToActionBlock } from '@/types/blocks'

export const CallToAction: React.FC<CallToActionBlock> = ({
  title,
  description,
  buttonLabel,
  buttonLink,
  backgroundColor = 'grey',
}) => {
  return (
    <div className={`cta-inline cta-inline--${backgroundColor}`}>
      <h3 className="cta-inline__title">{title}</h3>
      {description && (
        <p className="cta-inline__description">{description}</p>
      )}
      <a href={buttonLink} className="cta-inline__button">
        {buttonLabel}
      </a>
    </div>
  )
}

💡 Tip: Voor Next.js gebruik Link component in plaats van <a> voor client-side routing.

📚 Usage Guidelines

✅ When to Use

  • Mid-page conversions: Na sections met uitleg/features een actie trigger
  • Soft CTAs: Niet-agressieve calls-to-action (vs. B3 die hero-achtig is)
  • Content breaks: Tussen lange content pagina's om engagement te behouden
  • Form follow-ups: Na contactforms: "Of bel ons direct: ..."
  • Alternative actions: Secundaire conversie routes aanbieden

❌ When NOT to Use

  • Hero sections: Gebruik B1 Hero Block met integrated CTA
  • Page-ending CTAs: Gebruik B3 CTA Block (gradient, dramatisch)
  • Multiple buttons: Dit block heeft 1 button (B3 heeft primary+secondary)
  • Complex layouts: Dit is text-only, gebruik B9 TwoColumn voor split layouts

Content Guidelines

Field Length Best Practices
Title 3-8 woorden • Gebruik actieve taal ("Start vandaag", "Krijg toegang")
• Maak het specifiek ("Word klant" vs "Aan de slag")
• Test vragende vorm ("Klaar om te groeien?")
Description 1-2 zinnen (max 120 karakters) • Focus op benefit, niet feature
• Optioneel: verwijder weerstand ("Geen creditcard", "Gratis")
• Houd het scanbaar (korte zinnen)
Button Label 2-4 woorden • Gebruik eerste persoon ("Start mijn trial" > "Start trial")
• Wees specifiek ("Download gids" > "Meer info")
• Test urgentie ("Probeer nu gratis")

Background Selection Guide

  • White: Op gekleurde page backgrounds (navy, teal sections)
  • Grey: Default op white page backgrounds (subtiel, niet invasief)
  • Teal: High-priority CTAs die extra aandacht verdienen

⚠️ Let op: Gebruik max. 1-2 inline CTAs per pagina. Te veel verwateren de conversie rates.
