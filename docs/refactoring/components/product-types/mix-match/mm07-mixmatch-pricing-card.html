📦 MM07: MixMatchPricingCard — Complete Sidebar

Purpose: Complete sticky sidebar for Mix & Match product pages. Combines box visualization, item list, pricing breakdown, box size selector, CTA, and trust signals.

Key Features:

  • Box Visual Header: Dark gradient bg with emoji slots (6 slots, filled/empty states, pulse animation)
  • Items List: Scrollable (max 260px), hover states, remove buttons (MM05 embedded)
  • Pricing Breakdown: Subtotal, discount, total (large bold), per-item average
  • CTA Button: 2 states - incomplete (grey + lock icon) / ready (teal + cart icon)
  • Box Size Selector: Radio options (Small/Medium/Large) with prices
  • Trust Signals: 3 icons (truck, leaf, recycle) with labels

Structure (Top to Bottom):

  1. .box-visual — Dark header with title + 6 emoji slots
  2. .box-items — Scrollable selected items list (MM05)
  3. .box-pricing — Grey card with price breakdown
  4. .box-cta — Primary action button
  5. .box-note — Helper text below button
  6. .box-options — Box size radio selector
  7. .box-trust — Trust signals row

TypeScript Interface:

interface MixMatchPricingCardProps {
  // Selected items
  items: Array<{ id: string; emoji: string; name: string; detail: string; quantity: number; price: number }>;
  maxItems: number;

  // Pricing
  subtotal: number;
  discount: number;
  totalPrice: number;

  // Box options
  boxSizes: Array<{ id: string; name: string; itemCount: number; description: string; price: number }>;
  selectedBoxSize: string;
  onBoxSizeChange: (sizeId: string) => void;

  // Actions
  onRemoveItem: (itemId: string) => void;
  onCheckout: () => void;

  // State
  isComplete: boolean;

  // Trust signals (optional)
  trustSignals?: Array<{ icon: string; text: string }>;
}

Design Tokens (Key):

  • Card: white bg, 1.5px grey border, 20px radius, shadow-md
  • Visual header: navy gradient, 20px padding, teal radial glow (120px circle)
  • Slots: 6-column grid, 4px gap, 8px radius, aspect-ratio 1:1
  • Filled slot: teal-glow bg, solid border rgba(0,137,123,0.3)
  • Empty slot: transparent bg, dashed border, pulse animation
  • Body padding: 16px
  • Pricing card: grey-light bg, 12px padding, 12px radius
  • Total row: 18px Plus Jakarta Sans 800, 2px navy border-top
  • CTA: 52px height, 12px radius, 16px font
  • CTA ready: teal bg, shadow, hover navy + translateY(-1px)
  • CTA incomplete: grey bg, grey text, cursor not-allowed
  • Radio option: 8px padding, 1.5px border, 8px radius
  • Radio selected: teal border + teal-glow bg

Responsive: Sticky on desktop (position: sticky; top: 90px), static on mobile, full-width below 768px.
