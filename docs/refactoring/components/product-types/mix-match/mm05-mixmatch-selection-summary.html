📦 MM05: MixMatchSelectionSummary

Purpose: Vertical list of selected items in Mix & Match box sidebar. Shows emoji, name, details, quantity, price, and remove button per item.

Key Features:

  • Scrollable list (max-height: 260px)
  • Hover background on each row
  • Remove button (coral X icon on hover)
  • Emoji/icon + name (ellipsis overflow)
  • Meta details (calories, dietary info)
  • Quantity badge (×1, ×2, etc.)
  • Price per item (JetBrains Mono)

Design Tokens:

  • Row padding: 6px 8px, gap: 8px, border-radius: 8px
  • Hover bg: #F1F4F8 (grey-light)
  • Icon: 18px emoji
  • Name: 12px 700 DM Sans, ellipsis overflow
  • Detail: 10px grey-mid
  • Qty badge: 11px JetBrains Mono 700, grey-light bg, 2/6px padding, 4px radius
  • Price: 11px JetBrains Mono 700
  • Remove button: 20×20px, transparent → coral-light bg on hover, coral icon

TypeScript Interface:

interface MixMatchSelectionSummaryProps {
  items: Array<{
    id: string;
    emoji: string;
    name: string;
    detail: string;
    quantity: number;
    price: number;
  }>;
  onRemove: (id: string) => void;
  maxHeight?: number; // Default: 260px
}

React Example:

export const MixMatchSelectionSummary = ({ items, onRemove }) => (
  <div className="box-items">
    {items.map((item) => (
      <div key={item.id} className="bi">
        <div className="bi-icon">{item.emoji}</div>
        <div className="bi-info">
          <div className="bi-name">{item.name}</div>
          <div className="bi-detail">{item.detail}</div>
        </div>
        <div className="bi-qty">×{item.quantity}</div>
        <div className="bi-price">€{item.price.toFixed(2)}</div>
        <button className="bi-remove" onClick={() => onRemove(item.id)}>
          <X size={12} />
        </button>
      </div>
    ))}
  </div>
);
