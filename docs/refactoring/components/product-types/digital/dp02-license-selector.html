📖 DP02: LicenseSelector — Complete Documentation

STEP 1: Component Overview

Component Name: DP02: LicenseSelector

Category: Digital Product Components (Product Types)

Usage: License/pricing tier selection for digital products (software, SaaS, templates, courses, e-books)

Description:

LicenseSelector displays pricing tiers with radio button selection, feature lists, and pricing information. Used for digital product pages where users need to choose between different license types (Personal, Professional, Team) or subscription tiers (Starter, Pro, Enterprise).

Key Features:

  • Radio button selection with custom styled radio circles
  • License name + optional badge (Popular, Recommended, New)
  • Description text (short benefit statement)
  • Feature list with checkmark icons
  • Price display with period (eenmalig, per maand, per jaar)
  • Old price + savings badge for discounts
  • Selected state visual feedback (teal border + glow + background)
  • Hover states for all options
  • 3 variants: Default (full features), Compact (inline description), Vertical (centered layout)
  • Responsive mobile layout (stacks price below content)

STEP 2: When to Use This Component

✅ Use LicenseSelector when:

  • Selling digital products with multiple license tiers (Personal, Pro, Team)
  • Offering SaaS subscription plans (Starter, Pro, Enterprise)
  • Displaying pricing options for templates, courses, or e-books (Basic, Premium, Bundle)
  • Showing upgrade paths from free to paid tiers
  • Comparing features across different pricing levels
  • User needs to select ONE pricing option before checkout

❌ Don't use when:

  • Selling physical products → use Variable Product components instead
  • Only one pricing option exists → just show the price, no selector needed
  • Comparing features without purchase → use ComparisonTable (DP05) instead
  • Multiple add-ons can be combined → use checkbox multi-select, not radio

💡 Common Use Cases:

  • Software licenses: Personal (€39), Professional (€79), Team (€399/year)
  • SaaS subscriptions: Free, Starter (€9/mo), Pro (€29/mo), Enterprise (custom)
  • Course bundles: Single course (€49), Course bundle (€199), Lifetime access (€499)
  • Template packs: Single template (€19), Template pack (€79), Unlimited (€199/year)
  • E-book editions: PDF only (€9.99), PDF + ePub (€14.99), Complete bundle (€24.99)

STEP 4: Props & Configuration

TypeScript Interface:

interface LicenseSelectorProps {
  title?: string;                    // Section title (e.g., "Kies je licentie")
  description?: string;               // Section description (optional)
  options: LicenseOption[];           // Array of license tiers
  selectedValue?: string;             // Currently selected license value
  onChange: (value: string) => void;  // Selection change handler
  name: string;                       // Radio group name (for form)
  variant?: 'default' | 'compact' | 'vertical';  // Display variant
  disabled?: boolean;                 // Disable all options
}

interface LicenseOption {
  value: string;                      // Unique identifier (e.g., "professional")
  name: string;                       // Display name (e.g., "Professional")
  description: string;                // Short benefit statement
  badge?: {
    text: string;                     // Badge text (e.g., "Populairst")
    variant: 'popular' | 'new' | 'recommended';  // Badge style
  };
  features?: string[];                // Feature list (checkmarks)
  price: {
    amount: string | number;          // Price (e.g., 79 or "Op aanvraag")
    period?: string;                  // Billing period (e.g., "eenmalig", "per maand")
    oldPrice?: number;                // Original price (for discount)
    save?: string;                    // Savings text (e.g., "Bespaar €20")
  };
  disabled?: boolean;                 // Disable this option
}

STEP 5: Layout Rules

Container Structure:

  • Container: .license-selector — Flex column with 12px gap
  • Title: Optional section heading (14px, 600 weight)
  • Description: Optional helper text (12px, grey-mid)
  • Options: Vertical stack of .lic-option cards

Option Card Layout:

  • Default: flex row: [radio] [body] [price]
  • Compact: Same as default, but smaller padding + no features
  • Vertical: flex-direction: column, radio absolutely positioned, centered content

Grid Breakdown (Default):

  • Radio: 20px circle, flex-shrink: 0, margin-top: 2px (alignment)
  • Body: flex: 1 (expands to fill space)
  • Price: flex-shrink: 0, text-align: right

Responsive Behavior:

  • Desktop (≥641px): Horizontal layout (radio → body → price)
  • Mobile (≤640px): Vertical layout:
    • Radio + body stay horizontal (top section)
    • Price moves below with border-top separator
    • Price becomes horizontal row (amount ↔ period)

Integration Context:

  • Product page sidebar: Use default variant with features
  • Modal/quick view: Use compact variant (saves space)
  • Comparison page: Use vertical variant (side-by-side columns)
  • Checkout flow: Use compact variant (user already knows details)

  • Teal on white: 4.2:1 (AA for large text)
  • Badge text on colored backgrounds: minimum 4.5:1

STEP 7: Implementation Notes

Selection State Management (React):

const [selectedLicense, setSelectedLicense] = useState<string | null>(null);

function handleLicenseChange(value: string) {
  setSelectedLicense(value);

  // Update parent component (e.g., add to cart data)
  onLicenseSelect?.(value);

  // Analytics tracking
  gtag('event', 'select_license', {
    license_type: value,
    product_id: productId,
  });
}

Dynamic Pricing Calculation:

// Calculate total with selected license
const selectedOption = options.find(opt => opt.value === selectedLicense);
const licensePrice = selectedOption?.price.amount || 0;
const totalPrice = basePrice + licensePrice;

// Show savings if old price exists
const savings = selectedOption?.price.oldPrice
  ? selectedOption.price.oldPrice - licensePrice
  : 0;

Form Integration:

// Add to cart with selected license
function addToCart() {
  if (!selectedLicense) {
    toast.error('Selecteer eerst een licentie');
    return;
  }

  const cartItem = {
    productId,
    productName,
    license: selectedLicense,
    price: selectedOption.price.amount,
    quantity: 1,
  };

  addItemToCart(cartItem);
}

Validation:

// Require license selection before checkout
function validateCheckout() {
  const errors: string[] = [];

  if (!selectedLicense) {
    errors.push('Selecteer een licentie om verder te gaan');
  }

  return errors;
}
