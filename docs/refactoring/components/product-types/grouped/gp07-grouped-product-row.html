GP07: Grouped Product Row

List view variant for grouped product selection — horizontal row layout with 6-column grid

Step 1: Component Overview

What is GP07?

GP07 (Grouped Product Row) is the list view variant for displaying grouped product items in a horizontal row layout. It provides a denser, more information-rich view compared to the grid card layout, ideal for users who want to compare multiple attributes at a glance.

Key Insight: List view is preferred by B2B buyers and power users who need to scan multiple products quickly with all key details visible (SKU, stock, price) without interaction.

Step 2: When to Use

Use GP07 when:

  • User toggles to List view via GP04 (Toolbar)
  • Displaying 6-12+ products in a grouped set (dense information layout)
  • B2B context where SKU, stock, and pricing need to be immediately visible
  • Users need to quickly compare multiple product attributes without clicking
  • Desktop/tablet screens (≥768px) where horizontal space is available
  • Implementing bulk selection (checkboxes + "Select all" actions)

Don't use when:

  • Mobile screens (<640px) — use GP01 (ProductCard) instead
  • Fewer than 4 products — grid view is more visually appealing
  • Product images are the primary decision factor (use grid view)

Step 4: Props & Configuration

React/TypeScript Interface 

interface GroupedProductRowProps {
  // Product data
  productId: string
  name: string
  sku: string
  icon?: string           // Emoji or image URL
  image?: string          // Product image URL

  // Pricing
  price: number
  oldPrice?: number       // Show strike-through if present
  currency?: string       // Default: 'EUR'

  // Stock status
  stock: 'in-stock' | 'low-stock' | 'out-of-stock'
  stockCount?: number     // For "Nog 3 stuks"
  stockText?: string      // Custom stock message

  // Badges
  badges?: Array<{
    type: 'popular' | 'new' | 'sale' | 'pro'
    label: string
    icon?: string
  }>

  // Selection state
  isSelected?: boolean
  isDisabled?: boolean    // Disable checkbox + greyscale

  // Callbacks
  onSelect?: (productId: string, selected: boolean) => void
  onAddToCart?: (productId: string) => void

  // Display options
  showCheckbox?: boolean  // Default: true
  ctaText?: string        // Default: "Voeg toe"
  ctaDisabledText?: string // Default: "Uitverkocht"
}

Step 5: Layout Rules

Grid Structure (6 Columns) 

grid-template-columns: 40px 60px 1fr auto auto auto;

Column 1 (40px)   → Checkbox (20×20px centered)
Column 2 (60px)   → Product icon/image (60×60px)
Column 3 (1fr)    → Product info (name, SKU, badges) — flexible width
Column 4 (auto)   → Price display (right-aligned)
Column 5 (auto)   → Stock status (right-aligned)
Column 6 (auto)   → Add to cart button (fixed width)

Responsive Behavior

Breakpoint Layout Change
≥1024px 6-column grid (default layout)
≤1024px 4-column grid → Price/Stock/CTA wrap to 2nd row (full width)
≤640px 2-column grid → Icon (col 1) + Info (col 2), everything else wraps

State Variations

  • Default: White bg, grey border (1.5px), hover → teal border + shadow
  • Selected: Teal border + light teal bg (rgba(0,137,123,0.02)), checkbox checked
  • Out of Stock: Opacity 0.6, greyscale filter 0.5, disabled checkbox, disabled CTA
  • Hover: Border color → teal, box-shadow → var(--shadow-md), CTA lift effect
Step 6: Accessibility

WCAG 2.1 AA Compliance

ARIA Attributes

  • role="checkbox" on checkbox element
  • aria-checked="true|false" for checkbox state
  • aria-disabled="true" for out-of-stock products
  • tabindex="0" on checkbox for keyboard navigation

Keyboard Navigation

Key Action
Tab Navigate between checkbox → button
Space Toggle checkbox selection
Enter Toggle checkbox OR activate button

Color Contrast

  • Product name (#0A1628) on white — 18.5:1 ✓
  • Price (#0A1628) on white — 18.5:1 ✓
  • Stock text (green/amber/coral) — 4.8:1+ ✓
  • Button text (grey-dark) on white — 7.2:1 ✓

Screen Reader Support: Checkbox announces "not checked" or "checked" state. Disabled products announce "disabled". Button text is read correctly ("Voeg toe" / "Uitverkocht").

Step 7: Implementation Notes

React Component Example 

import { ShoppingCart, Clock, Check, Star, Sparkles } from 'lucide-react'

export function GroupedProductRow({
  productId,
  name,
  sku,
  icon,
  price,
  oldPrice,
  stock = 'in-stock',
  stockCount,
  stockText,
  badges = [],
  isSelected = false,
  isDisabled = false,
  onSelect,
  onAddToCart,
  ctaText = 'Voeg toe',
  ctaDisabledText = 'Uitverkocht',
}: GroupedProductRowProps) {
  const handleCheckboxClick = () => {
    if (isDisabled) return
    onSelect?.(productId, !isSelected)
  }

  const handleAddClick = () => {
    if (stock === 'out-of-stock') return
    onAddToCart?.(productId)
  }

  const stockConfig = {
    'in-stock': { text: stockText || 'Op voorraad', class: 'in-stock' },
    'low-stock': { text: stockText || `Nog ${stockCount || 3} stuks`, class: 'low-stock' },
    'out-of-stock': { text: 'Tijdelijk uitverkocht', class: 'out-of-stock' },
  }

  const badgeIcons = {
    popular: Star,
    new: Sparkles,
  }

  return (
    <div
      className={cn(
        'product-row',
        isSelected && 'selected',
        stock === 'out-of-stock' && 'out-of-stock'
      )}
      data-product-id={productId}
      data-price={price}
    >
      {/* Checkbox */}
      <div className="product-checkbox-wrapper">
        <div
          className="product-checkbox"
          role="checkbox"
          aria-checked={isSelected}
          aria-disabled={isDisabled}
          tabIndex={0}
          onClick={handleCheckboxClick}
          onKeyDown={(e) => {
            if (e.key === ' ' || e.key === 'Enter') {
              e.preventDefault()
              handleCheckboxClick()
            }
          }}
        >
          {isSelected && <Check size={12} />}
        </div>
      </div>

      {/* Icon/Image */}
      <div className="product-icon">{icon}</div>

      {/* Product Info */}
      <div className="product-info">
        {badges.length > 0 && (
          <div className="product-badges">
            {badges.map((badge, i) => {
              const Icon = badgeIcons[badge.type]
              return (
                <span key={i} className={`badge ${badge.type}`}>
                  {Icon && <Icon size={11} />}
                  {badge.label}
                </span>
              )
            })}
          </div>
        )}
        <h3 className="product-name">{name}</h3>
        <p className="product-sku">SKU: {sku}</p>
      </div>

      {/* Price */}
      <div className="product-price">
        <div className="price-row">
          <span className="price-current">€{price.toFixed(2)}</span>
          {oldPrice && (
            <span className="price-old">€{oldPrice.toFixed(2)}</span>
          )}
        </div>
      </div>

      {/* Stock */}
      <div className={`product-stock ${stockConfig[stock].class}`}>
        <span className="stock-dot" />
        {stockConfig[stock].text}
      </div>

      {/* CTA */}
      <button
        className="product-cta"
        disabled={stock === 'out-of-stock'}
        onClick={handleAddClick}
      >
        {stock === 'out-of-stock' ? (
          <>
            <Clock size={16} />
            {ctaDisabledText}
          </>
        ) : (
          <>
            <ShoppingCart size={16} />
            {ctaText}
          </>
        )}
      </button>
    </div>
  )
}

return (
