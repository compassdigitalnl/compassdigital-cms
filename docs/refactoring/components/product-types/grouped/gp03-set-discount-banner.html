GP03: Set Discount Banner

Promotional banner component highlighting the complete set discount offer. Features purple styling with gift icon to draw attention to the savings opportunity.

1 Component Overview

Purpose

The Set Discount Banner is a high-visibility promotional banner that appears below the hero section on grouped product pages. It clearly communicates the discount customers receive when purchasing the complete set of items.

Key Features

  • Purple Styling: Light purple background (#EDE9FE) with dark purple border (#7C3AED) for high contrast
  • Gift Icon: 24px Lucide icon (purple color) to symbolize savings/discount
  • Bold Headline: "Complete set voordeel" in bold purple text
  • Discount Amount: Monospace font (JetBrains Mono), 800 weight, 16px for emphasis
  • Flexible Text: Supports custom messaging (item count, percentage, amount)
  • Responsive Design: Stacks vertically on mobile (<640px)

2 When to Use

✅ Use this component when:

  • Product page offers a grouped product with complete set discount
  • You want to highlight savings opportunity prominently
  • Discount is automatic when all items are selected
  • You need to increase conversion by emphasizing value

❌ Don't use when:

  • There is no discount for buying the complete set
  • Discount requires a coupon code (use coupon banner instead)
  • Product is a fixed bundle (discount already applied in base price)
  • Banner would clutter the page (e.g., too many promotional messages)

React/TypeScript Component
interface SetDiscountBannerProps {
  headline?: string
  message: string
  discountAmount: number
  discountPercent?: number
  icon?: string
}

export const SetDiscountBanner = ({
  headline = 'Complete set voordeel:',
  message,
  discountAmount,
  discountPercent,
  icon = 'gift'
}: SetDiscountBannerProps) => {
  return (
    <div className="set-discount-banner">
      <LucideIcon name={icon} />
      <div className="set-discount-text">
        {headline && <strong>{headline}</strong>}{' '}
        {message}{' '}
        <span className="discount-amount">
          €{discountAmount} korting
        </span>
        {discountPercent && ` (${discountPercent}%)`}
      </div>
    </div>
  )
}

6 Accessibility

♿ Accessibility Checklist

  • Color Contrast: Purple text on light purple bg = 4.5:1 ratio (AA compliant)
  • Semantic HTML: Use standard div with descriptive class names
  • Icon Purpose: Decorative only (gift symbolizes discount visually)
  • Reading Order: Icon → Headline → Message → Amount (logical flow)
  • Font Legibility: 14px body text, 16px discount amount (easily readable)

ARIA Attributes

Optional: Add role="complementary" and aria-label="Set discount information" if banner contains critical promotional info.

7 Implementation Notes

Dynamic Content Example

// Calculate discount dynamically
const itemCount = groupedProduct.items.length
const discountPercent = groupedProduct.setDiscount
const individualTotal = items.reduce((sum, item) => sum + item.price, 0)
const discountAmount = individualTotal * (discountPercent / 100)

const message = `Selecteer alle ${itemCount} items en ontvang automatisch`

<SetDiscountBanner
  message={message}
  discountAmount={discountAmount}
  discountPercent={discountPercent}
/>

Common Pitfalls

  • Message too long: Keep message concise (max 2 lines) to avoid banner becoming too tall
  • Missing currency symbol: Always include € (or locale currency) before discount amount
  • Percentage without amount: Show BOTH amount (€45) AND percentage (15%) for clarity
  • Wrong icon: Use gift for discounts, not percent (reserved for general promotions)
