STEP 1: Component Overview

Component Name: BB05: BundleTotalCalculator

Category: Product Types › Bundle Builder

Version: 1.0.0

Last Updated: 25 Februari 2026

Purpose

The BundleTotalCalculator component displays a comprehensive price breakdown for bundle purchases. It calculates and shows the subtotal, bundle discount, optional shipping/tax, final total, and a motivational savings badge. This component provides pricing transparency and reinforces the value proposition of bundle buying.

Key Features

  • Price Breakdown: Clear line-item display of subtotal, discount, and total
  • Real-time Calculation: Updates automatically as bundle items change
  • Green Discount Highlight: Negative amount in green with minus sign (− €31,40)
  • Bold Total: 20px Plus Jakarta Sans 800 with 2px navy border for emphasis
  • Savings Badge: Green badge showing total savings amount with icon
  • Optional Lines: Shipping, tax, or other fees can be included
  • Currency Formatting: Locale-aware number formatting (€157,00)

Visual Hierarchy

  1. Total Line: Boldest (20px, 800 weight, 2px border) — most prominent
  2. Savings Badge: Green background — second most prominent
  3. Discount Line: Green text with minus — third (reinforces savings)
  4. Subtotal Line: Grey text — fourth (reference point)
  5. Optional Lines: Smallest/lightest (shipping, tax) — least prominent

Calculation Logic

Subtotal = Sum of all item prices
Discount = Subtotal × (discount percentage / 100)
Shipping = Based on shipping method (or "Gratis" if free)
Tax = (Subtotal - Discount + Shipping) × (tax rate / 100)
Total = Subtotal - Discount + Shipping (+ Tax if applicable)
Savings = Discount amount

STEP 2: When to Use

Primary Use Cases

  • Bundle Builder Sidebar: Main use case — shows total below bundle items and discount tiers
  • Cart Summary: Price breakdown for bundle products in cart
  • Checkout Summary: Final price review before payment
  • Order Confirmation: Receipt showing bundle savings
  • Quote/Offerte Summary: Price breakdown for B2B quotes

When NOT to Use

  • Single Product: Use standard OrderSummary (EC07) for non-bundle products
  • Simple Line Total: If no discount/tax/shipping, use simpler total display
  • Complex Invoices: Use dedicated invoice template for detailed breakdowns
  • Empty Bundle: Hide or show placeholder when no items selected

Recommended Pairings

  • BB03: BundleItemRow — Selected products list (appears above this component)
  • BB04: BundleDiscountTiers — Discount tiers hint (appears above this component)
  • BB06: BundleProgressBar — Progress indicator (appears at top of sidebar)
  • EC09: TrustSignals — Trust badges (appears below this component)
  • CTA Button: "Bundel bestellen" or "Toevoegen aan winkelwagen" (appears below)

Placement Guidelines

  • Bundle Sidebar: Bottom section, above CTA button (optimal position)
  • Cart: Right sidebar or below line items on mobile
  • Checkout: Right sidebar, sticky on scroll
  • Order Confirmation: Within order summary card

Calculation Functions

function calculateDiscount(subtotal: number, percentage: number): number {
  return subtotal * (percentage / 100);
}

function calculateTax(amount: number, taxRate: number): number {
  return amount * (taxRate / 100);
}

function calculateTotal(
  subtotal: number,
  discount: number,
  shipping: number,
  tax?: number
): number {
  const afterDiscount = subtotal - discount;
  const withShipping = afterDiscount + shipping;
  const taxAmount = tax ? calculateTax(withShipping, tax) : 0;
  return withShipping + taxAmount;
}

STEP 7: Implementation Notes

React/Next.js Component Example

'use client';

import { BadgePercent } from 'lucide-react';

interface BundleTotalCalculatorProps {
  subtotal: number;
  discount: number;
  itemCount: number;
  shipping?: number | 'free';
  tax?: number;
  showSavingsBadge?: boolean;
  size?: 'compact' | 'default';
  currency?: string;
  locale?: string;
}

export function BundleTotalCalculator({
  subtotal,
  discount,
  itemCount,
  shipping,
  tax,
  showSavingsBadge = true,
  size = 'default',
  currency = 'EUR',
  locale = 'nl-NL',
}: BundleTotalCalculatorProps) {
  // Calculate amounts
  const discountAmount = subtotal * (discount / 100);
  const afterDiscount = subtotal - discountAmount;

  const shippingCost = shipping === 'free' ? 0 : (shipping || 0);
  const beforeTax = afterDiscount + shippingCost;

  const taxAmount = tax ? beforeTax * (tax / 100) : 0;
  const total = beforeTax + taxAmount;

  // Format currency
  const formatCurrency = (amount: number) => {
    return new Intl.NumberFormat(locale, {
      style: 'currency',
      currency,
    }).format(amount);
  };

  return (
    <>
      <div
        className={`pb ${size !== 'default' ? size : ''}`}
        role="region"
        aria-label="Bundle price breakdown"
      >
        {/* Subtotal */}
        <div className="pb-line sub" role="row">
          <span role="rowheader">
            Subtotaal ({itemCount} item{itemCount !== 1 ? 's' : ''})
          </span>
          <span role="cell">{formatCurrency(subtotal)}</span>
        </div>

        {/* Discount */}
        {discount > 0 && (
          <div className="pb-line discount" role="row">
            <span role="rowheader">
              Bundelkorting (−{discount}%)
            </span>
            <span role="cell">− {formatCurrency(discountAmount)}</span>
          </div>
        )}

        {/* Shipping */}
        {shipping !== undefined && (
          <div
            className={`pb-line shipping ${shipping === 'free' ? 'free' : ''}`}
            role="row"
          >
            <span role="rowheader">Verzendkosten</span>
            <span role="cell">
              {shipping === 'free' ? 'Gratis' : formatCurrency(shippingCost)}
            </span>
          </div>
        )}

        {/* Tax */}
        {tax && tax > 0 && (
          <div className="pb-line tax" role="row">
            <span role="rowheader">BTW ({tax}%)</span>
            <span role="cell">{formatCurrency(taxAmount)}</span>
          </div>
        )}

        {/* Total */}
        <div className="pb-total" role="row" aria-label="Total price">
          <span role="rowheader">
            Totaal{tax ? ' (incl. BTW)' : ''}
          </span>
          <span role="cell">{formatCurrency(total)}</span>
        </div>
      </div>

      {/* Savings badge */}
      {showSavingsBadge && discount > 0 && (
        <div
          className="pb-save-badge"
          role="status"
          aria-live="polite"
        >
          <BadgePercent size={14} aria-hidden="true" />
          Je bespaart {formatCurrency(discountAmount)} met deze bundel!
        </div>
      )}
    </>
  );
}

State Management with React

// Parent component managing bundle
function BundleBuilder() {
  const [bundleItems, setBundleItems] = useState<BundleItem[]>([]);
  const [activeTier, setActiveTier] = useState<DiscountTier | null>(null);

  // Calculate subtotal
  const subtotal = bundleItems.reduce((sum, item) => sum + item.price, 0);

  // Get active discount percentage
  const discountPercentage = activeTier?.discount || 0;

  // Determine if free shipping (e.g., bundle total > €60)
  const shipping = subtotal > 60 ? 'free' : 4.95;

  return (
    <aside className="bundle-sidebar">
      <BundleItemList items={bundleItems} onRemove={handleRemove} />
      <BundleDiscountTiers tiers={tiers} currentItemCount={bundleItems.length} />

      <BundleTotalCalculator
        subtotal={subtotal}
        discount={discountPercentage}
        itemCount={bundleItems.length}
        shipping={shipping}
      />

      <button className="buy-btn">Bundel bestellen</button>
    </aside>
  );
}

Integration with Payload CMS

// Fetch bundle pricing from Payload CMS
async function calculateBundlePrice(bundleId: string) {
  const bundle = await payload.findByID({
    collection: 'bundles',
    id: bundleId,
    depth: 2, // Populate product prices
  });

  const subtotal = bundle.bundleItems.reduce(
    (sum, item) => sum + item.product.price,
    0
  );

  const activeTier = getActiveTier(bundle.bundleItems.length, bundle.discountTiers);
  const discountPercentage = activeTier?.discount || 0;
  const discountAmount = subtotal * (discountPercentage / 100);

  const total = subtotal - discountAmount;

  return {
    subtotal,
    discountPercentage,
    discountAmount,
    total,
    itemCount: bundle.bundleItems.length,
  };
}

Dynamic Savings Badge Text

// Generate dynamic badge text based on savings amount
function getSavingsBadgeText(discountAmount: number, percentage: number) {
  const formatted = formatCurrency(discountAmount);

  if (percentage >= 30) {
    return `🎉 Je bespaart ${formatted} (${percentage}%)!`;
  } else if (percentage >= 20) {
    return `Je bespaart ${formatted} met deze bundel!`;
  } else {
    return `Bundel & bespaar ${formatted}`;
  }
}
