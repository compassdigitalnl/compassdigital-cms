STEP 1: Component Overview

Component Name: BB04: BundleDiscountTiers

Category: Product Types › Bundle Builder

Version: 1.0.0

Last Updated: 25 Februari 2026

Purpose

The BundleDiscountTiers component displays a visual representation of volume-based discount tiers for bundle purchases. It motivates users to add more items to their bundle by clearly showing the savings they can unlock at different quantity thresholds.

Key Features

  • Visual Discount Ladder: 4-tier horizontal grid showing increasing discounts
  • Active State Highlighting: Current tier highlighted with teal border and background
  • Progress Indicator: Checkmark (✓) on active tier for visual confirmation
  • Max Tier Emphasis: Highest discount shown in green for maximum motivation
  • Compact Design: Fits within bundle sidebar without overwhelming the interface
  • Motivational Header: "Meer toevoegen = meer korting" encourages action
  • Real-time Updates: Active tier changes as bundle size changes

Visual Hierarchy

  1. Header Text: 12px bold with teal icon (primary message)
  2. Active Tier: Teal background + border (most prominent)
  3. Max Tier: Green discount percentage (second most prominent)
  4. Inactive Tiers: Grey text on white (tertiary, future goals)

Psychological Design

This component leverages the Goal Gradient Effect — users are more motivated to complete a goal as they get closer to it. By visualizing tiers, users can see:

  • Their current progress (active tier)
  • How close they are to the next tier (motivation)
  • The maximum possible savings (aspirational goal)

STEP 2: When to Use

Primary Use Cases

  • Bundle Builder Sidebar: Main use case — shows discount tiers between bundle items list and total calculator
  • Product Configurator: Displays tiered pricing for component packages
  • Subscription Upsells: Shows discount tiers for multi-month subscriptions
  • Volume Pricing: Motivates bulk purchases with visual tier representation
  • Gift Card Bundles: Shows bonus percentages at different purchase amounts

When NOT to Use

  • Single-Tier Discounts: Use simple badge/banner for flat discounts (no tiers needed)
  • Complex Pricing Tables: Use StaffelCalculator (C4) for detailed volume pricing with calculations
  • Simple Progress Bars: Use BB06 (BundleProgressBar) for linear progress without discount tiers
  • Non-Bundle Contexts: Don't force tier display if pricing isn't tiered

Recommended Pairings

  • BB03: BundleItemRow — Shows selected products (appears above this component)
  • BB05: BundleTotalCalculator — Price breakdown showing applied discount (appears below)
  • BB06: BundleProgressBar — Alternative/additional progress indicator (appears at top of sidebar)
  • BB02: BundleProductCard — Selectable products that add to bundle (drives tier progression)

Placement Guidelines

  • Bundle Sidebar: Between item list and total calculator (optimal position)
  • Sticky Footer: Fixed at bottom on mobile during product selection
  • Cart Summary: Below line items, above checkout button
  • Product Page: Near add-to-cart for bundle products

Active Tier Logic

function getActiveTier(currentItemCount: number, tiers: DiscountTier[]) {
  // Find the highest tier the user has unlocked
  const sortedTiers = [...tiers].sort((a, b) => b.itemCount - a.itemCount);
  return sortedTiers.find(tier => currentItemCount >= tier.itemCount);
}

STEP 7: Implementation Notes

React/Next.js Component Example

'use client';

import { TrendingUp } from 'lucide-react';

interface DiscountTier {
  itemCount: number;
  label: string;
  discount: number;
}

interface BundleDiscountTiersProps {
  tiers: DiscountTier[];
  currentItemCount: number;
  headerText?: string;
  showIcon?: boolean;
  size?: 'compact' | 'default';
}

export function BundleDiscountTiers({
  tiers,
  currentItemCount,
  headerText = 'Meer toevoegen = meer korting',
  showIcon = true,
  size = 'default',
}: BundleDiscountTiersProps) {
  // Find active tier
  const activeTier = getActiveTier(currentItemCount, tiers);
  const maxTier = tiers[tiers.length - 1];

  // Determine next tier for screen reader
  const nextTier = tiers.find(t => t.itemCount > currentItemCount);

  return (
    <div
      className={`bdt-container ${size !== 'default' ? size : ''}`}
      role="region"
      aria-label="Bundle discount tiers"
    >
      <div className="bdt-header" id="bdt-header">
        {showIcon && (
          <TrendingUp size={13} aria-hidden="true" />
        )}
        {headerText}
      </div>

      <div
        className="bdt-tiers"
        role="list"
        aria-labelledby="bdt-header"
      >
        {tiers.map((tier) => {
          const isActive = activeTier?.itemCount === tier.itemCount;
          const isMax = tier.itemCount === maxTier.itemCount;

          return (
            <div
              key={tier.itemCount}
              className={`bdt-tier ${isActive ? 'active' : ''} ${isMax ? 'max' : ''}`}
              role="listitem"
              aria-label={`${tier.label}: ${tier.discount}% discount${isActive ? '. Currently active.' : ''}`}
              aria-current={isActive ? 'true' : undefined}
            >
              <span className="bdt-tier-label">{tier.label}</span>
              <span className="bdt-tier-value">{tier.discount}%</span>
            </div>
          );
        })}
      </div>

      {/* Screen reader announcement */}
      <div className="sr-only" aria-live="polite">
        {activeTier && (
          `You have ${currentItemCount} items in your bundle and unlocked ${activeTier.discount}% discount.`
        )}
        {nextTier && (
          ` Add ${nextTier.itemCount - currentItemCount} more item${nextTier.itemCount - currentItemCount > 1 ? 's' : ''} to unlock ${nextTier.discount}% discount.`
        )}
        {!nextTier && currentItemCount >= maxTier.itemCount && (
          ' Maximum discount tier unlocked!'
        )}
      </div>
    </div>
  );
}

function getActiveTier(currentItemCount: number, tiers: DiscountTier[]) {
  const sortedTiers = [...tiers].sort((a, b) => b.itemCount - a.itemCount);
  return sortedTiers.find(tier => currentItemCount >= tier.itemCount);
}

// Fetch discount tiers from Payload CMS
async function getDiscountTiers(bundleConfigId: string) {
  const config = await payload.findByID({
    collection: 'bundleConfigurations',
    id: bundleConfigId,
  });

  return config.discountTiers.map(tier => ({
    itemCount: tier.itemCount,
    label: tier.label,
    discount: tier.discount,
  }));
}

Dynamic Header Text

function getDynamicHeaderText(currentItemCount: number, nextTier?: DiscountTier) {
  if (!nextTier) {
    return 'Maximale korting bereikt!';
  }

  const itemsNeeded = nextTier.itemCount - currentItemCount;
  return `Voeg ${itemsNeeded} item${itemsNeeded > 1 ? 's' : ''} toe voor ${nextTier.discount}% korting`;
}
