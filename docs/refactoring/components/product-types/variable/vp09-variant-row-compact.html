Component Documentation

STEP 1: Component Overview

VP09: VariantRowCompact is a horizontal list row layout for multi-variant selection. It provides an alternative to VP08's vertical grid cards, optimized for displaying many variants in a scannable list format.

Key Features:

  • 6-column grid layout (checkbox, image, info, total, qty stepper, actions)
  • Larger 24×24px checkbox vs VP08's 18×18px
  • Optional image thumbnail (80×80px, can be hidden)
  • Combined info section (name + description + stock + price)
  • Total price calculation display (qty × unit price)
  • Reserved column for future actions (favorite, compare)
  • Compact vertical padding (10px vs VP08's 12px)
  • Mobile responsive: collapses to 2-column layout

File Size: ~940 lines

Component ID: VP09

Related: VP08 (grid card), VP10 (sidebar), VP11 (toolbar), VP13 (list container CSS)

STEP 2: When to Use

✅ Use VP09 (List Row) When:

  • Displaying 10+ variants in a scannable format
  • User preference is set to "List View" (toggle with VP08)
  • Variants have long descriptive names (more horizontal space)
  • Need to display variant images alongside details
  • Comparing prices across variants side-by-side
  • B2B context where users scan SKUs/specs quickly
  • Desktop-first experience (rows work better on wide screens)

❌ Don't Use VP09 When:

  • Fewer than 6 variants (VP08 grid is better for small sets)
  • Mobile-first experience (VP08 grid cards stack better)
  • Variants are visual (colors, patterns) — VP08 shows images larger
  • Single-variant selection (use VP01-VP05 instead)
  • No need for image thumbnails (use simplified list without column 2)

STEP 5: Layout Rules

6-Column Grid System

grid-template-columns: 24px 80px 1fr auto auto 110px;
Column Width Content
1 24px Checkbox (24×24px, static position)
2 80px Image thumbnail (optional, can hide with .hidden)
3 1fr Info (name, description, price, stock badge)
4 auto Total price (calculated: qty × unit price)
5 auto Quantity stepper (140px total: 40+60+40)
6 110px Reserved (action buttons: favorite, compare)

Mobile Responsive (≤768px)

Behavior:

  • Collapse to 2-column grid: 24px 1fr
  • All elements after checkbox wrap to column 2
  • Image thumbnail hidden (display: none)
  • Quantity stepper expands to full width
  • Action buttons left-aligned (full width available)

STEP 7: Implementation Notes

JavaScript Behavior

// 1. Checkbox Toggle
checkbox.addEventListener('click', () => {
  const isChecked = checkbox.getAttribute('aria-checked') === 'true';

  if (!isChecked) {
    checkbox.classList.add('checked');
    checkbox.setAttribute('aria-checked', 'true');
    row.classList.add('selected');

    // Auto-set qty to 1 if 0
    if (parseInt(qtyInput.value) === 0) {
      qtyInput.value = 1;
      updateTotal();
    }
  } else {
    checkbox.classList.remove('checked');
    checkbox.setAttribute('aria-checked', 'false');
    row.classList.remove('selected');
    qtyInput.value = 0;
    updateTotal();
  }
});

// 2. Quantity Change
qtyInput.addEventListener('change', () => {
  const qty = parseInt(qtyInput.value) || 0;

  if (qty > 0) {
    // Auto-check checkbox
    checkbox.classList.add('checked');
    checkbox.setAttribute('aria-checked', 'true');
    row.classList.add('selected');
  } else {
    // Auto-uncheck checkbox
    checkbox.classList.remove('checked');
    checkbox.setAttribute('aria-checked', 'false');
    row.classList.remove('selected');
  }

  updateTotal();
});

// 3. Update Total Price
function updateTotal() {
  const qty = parseInt(qtyInput.value) || 0;
  const unitPrice = parseFloat(row.dataset.price);
  const total = (qty * unitPrice).toFixed(2);

  totalDisplay.textContent = `€${total.replace('.', ',')}`;

  // Enable/disable minus button
  minusBtn.disabled = qty === 0;
}

// 4. Favorite Button Toggle
favoriteBtn.addEventListener('click', () => {
  favoriteBtn.classList.toggle('active');
  // Save to localStorage or API
});

React/Next.js Example

import { useState } from 'react';
import { Check, Heart, BarChart } from 'lucide-react';

interface VariantRowProps {
  id: string;
  name: string;
  fullName: string;
  price: number;
  stockStatus: 'in-stock' | 'low-stock' | 'pre-order' | 'out-of-stock';
  stockText: string;
  image?: string;
  showImage?: boolean;
}

export function VariantRowCompact({
  id, name, fullName, price, stockStatus, stockText,
  image, showImage = false
}: VariantRowProps) {
  const [selected, setSelected] = useState(false);
  const [qty, setQty] = useState(0);
  const [isFavorite, setIsFavorite] = useState(false);

  const total = (qty * price).toFixed(2);

  const handleCheckbox = () => {
    const newSelected = !selected;
    setSelected(newSelected);
    if (newSelected && qty === 0) setQty(1);
    if (!newSelected) setQty(0);
  };

  const handleQtyChange = (newQty: number) => {
    setQty(Math.max(0, newQty));
    setSelected(newQty > 0);
  };

  return (
    <div
      className={`variant-row-compact ${selected ? 'selected' : ''}`}
      data-variant-id={id}
      data-price={price}
    >
      <!-- Column 1: Checkbox -->
      <div
        className={`variant-checkbox-compact ${selected ? 'checked' : ''}`}
        role="checkbox"
        aria-checked={selected}
        tabIndex={0}
        onClick={handleCheckbox}
      >
        {selected && <Check className="w-3.5 h-3.5" />}
      </div>

      <!-- Column 2: Image (optional) -->
      {showImage && (
        <div className="variant-image-row">
          {image || '🧤'}
        </div>
      )}
      {!showImage && <div className="variant-image-row hidden" />}

      <!-- Column 3: Info -->
      <div className="variant-info-row">
        <div className="variant-name-row">{name}</div>
        <div className="variant-fullname-row">{fullName}</div>
        <div className="variant-meta-row">
          <span className="variant-price-row">€{price.toFixed(2).replace('.', ',')} p/st</span>
          <div className={`stock-badge-inline ${stockStatus}`}>
            <span className="stock-dot-inline" />
            {stockText}
          </div>
        </div>
      </div>

      <!-- Column 4: Total -->
      <div className="variant-total-row">
        <span className="total-value">€{total.replace('.', ',')}</span>
        <span className="unit-label">totaal</span>
      </div>

      <!-- Column 5: Qty Stepper -->
      <div className="qty-stepper-inline">
        <button
          className="qty-btn-inline minus"
          onClick={() => handleQtyChange(qty - 1)}
          disabled={qty === 0}
        >
          −
        </button>
        <input
          type="number"
          className="qty-input-inline"
          value={qty}
          onChange={(e) => handleQtyChange(parseInt(e.target.value) || 0)}
          min={0}
          max={999}
        />
        <button
          className="qty-btn-inline plus"
          onClick={() => handleQtyChange(qty + 1)}
        >
          +
        </button>
      </div>

      <!-- Column 6: Actions -->
      <div className="variant-actions-row">
        <button
          className={`action-btn-row ${isFavorite ? 'active' : ''}`}
          onClick={() => setIsFavorite(!isFavorite)}
          aria-label="Toevoegen aan favorieten"
        >
          <Heart className="w-4 h-4" />
        </button>
        <button
          className="action-btn-row"
          aria-label="Vergelijken"
        >
          <BarChart className="w-4 h-4" />
        </button>
      </div>
    </div>
  );
}

Performance Considerations

  • Virtualization: For 50+ variants, use react-window or react-virtual to render only visible rows
  • Debounce qty input: Update totals/state after 300ms typing pause to avoid excessive re-renders
  • Image lazy loading: Use loading="lazy" on thumbnails to defer off-screen images
  • Memoization: Memoize total price calculation to avoid recalculating on every render
