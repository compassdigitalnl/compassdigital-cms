Premium UI Kit — Setup & Installatie

Volg deze stappen om het Premium UI Kit in je project te installeren. Gemiddelde setup tijd: 10-15 minuten.

1

Download en pak het ZIP bestand uit

Download het volledige pakket en pak alle bestanden uit naar een werkdirectory op je computer.

Terminal 
# Unzip het gedownloade bestand
unzip premium-ui-kit-v4.2.0.zip -d ~/Desktop/premium-ui-kit

# Navigeer naar de directory
cd ~/Desktop/premium-ui-kit
Tip: Bewaar het originele ZIP bestand voor backup doeleinden.
2

Installeer de vereiste dependencies

Het UI Kit vereist Node.js 18+ en NPM 9+. Installeer alle benodigde packages met NPM.

Terminal 
# Controleer Node.js versie (moet 18+ zijn)
node --version

# Installeer dependencies
npm install

# Of met Yarn
yarn install
Node.js versie 18 of hoger is vereist. Download de laatste versie op nodejs.org
3

Configureer je project

Kopieer het voorbeeld configuratiebestand en pas de instellingen aan voor jouw project.

Terminal 
# Kopieer voorbeeld config
cp .env.example .env

# Open config bestand
nano .env  # Of gebruik je favoriete editor
.env 
# Project configuratie
PROJECT_NAME="Mijn Project"
API_URL="https://api.example.com"
THEME_COLOR="#00897B"
4

Start de development server

Start de lokale development server en open je browser op localhost:3000.

Terminal 
# Start development server
npm run dev

# Server start op http://localhost:3000

Screenshot: Development server draait op localhost:3000

Gelukt! Je UI Kit is nu geïnstalleerd. Bekijk de documentatie voor component voorbeelden.

Veel Voorkomende Problemen

❌ Error: "Node version too old"

Update Node.js naar versie 18 of hoger via nodejs.org. Controleer je versie met node --version.

❌ Error: "Port 3000 already in use"

Een andere service gebruikt al poort 3000. Stop die service of gebruik een andere poort: PORT=3001 npm run dev.

❌ Components worden niet correct weergegeven

Clear je browser cache en NPM cache: npm cache clean --force, verwijder node_modules en draai npm install opnieuw.

Platform Setup

Installatie per Platform

Kies je besturingssysteem voor platform-specifieke installatie instructies.

1

Download het Windows installatieprogramma

Download het .exe installatieprogramma en voer het uit als administrator.

PowerShell 
# Download via PowerShell
Invoke-WebRequest -Uri "https://example.com/installer.exe" -OutFile "installer.exe"

# Voer installer uit
.\installer.exe
2

Voeg toe aan Windows PATH

Voeg de installatie directory toe aan je systeem PATH variabele.

Ga naar Systeem → Geavanceerde systeeminstellingen → Omgevingsvariabelen en voeg C:\Program Files\UIKit toe aan PATH.
1

Installeer via Homebrew

De makkelijkste manier om te installeren op macOS is via Homebrew.

Terminal 
# Installeer via Homebrew
brew tap company/uikit
brew install premium-ui-kit

# Verifieer installatie
uikit --version
2

Configureer macOS permissions

Geef de applicatie toegang tot je bestanden via System Preferences.

Ga naar System Preferences → Security & Privacy → Files and Folders en geef UIKit toegang.
1

Installeer via Package Manager

Gebruik je distributie's package manager (apt, yum, pacman).

Bash 
# Ubuntu/Debian
sudo apt-get update
sudo apt-get install premium-ui-kit

# Fedora/RHEL
sudo yum install premium-ui-kit

# Arch Linux
sudo pacman -S premium-ui-kit
