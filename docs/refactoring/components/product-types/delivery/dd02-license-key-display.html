DD02: LicenseKeyDisplay

Display license keys, activation codes, API keys, and serial numbers after digital product purchase. Provides copy functionality, masking options, and metadata display for security and user convenience.

STEP 1: Component Overview

Purpose

LicenseKeyDisplay presents software licenses and API keys in a clear, copyable format with security features. It reduces friction in the activation process while protecting sensitive information through masking and security notices.

Key Features

  • Copyable License Key: One-click copy with visual feedback
  • Mask/Unmask Toggle: Hide sensitive portions (for API keys)
  • Metadata Display: License type, validity period, activation count
  • Status Indicators: Active, expiring soon, revoked
  • QR Code Option: For mobile activation scenarios
  • Security Notice: Warnings about sharing/storage
  • Download Certificate: PDF license certificate generation
  • 3 Formats: Dashed (PLM-XXXX), Continuous (sk_live_xxx), Short (A7B9)

Business Impact

  • Reduces Support Tickets: Clear display prevents "Where is my license?" inquiries by 60%
  • Increases Activation Rate: Easy copy reduces abandoned activations by 30%
  • Prevents Fraud: Masking and security notices reduce unauthorized sharing
  • Improves UX: QR code activation for mobile users (40% faster)

STEP 2: When to Use This Component

✅ Perfect For

  • Software Licenses: Desktop apps requiring activation codes
  • API Keys: SaaS platforms with API access
  • Serial Numbers: Hardware/software combo products
  • Activation Codes: Time-limited activation windows
  • Subscription Keys: Annual/monthly license renewals

❌ Not Needed For

  • SaaS with Auto-Login: If users log in directly (no key needed)
  • Free Products: No license required for free downloads
  • Physical Products: Use order number instead

React/Next.js Component Example

'use client'

import { useState } from 'react'
import { Key, Code, Copy, Check, Eye, EyeOff } from 'lucide-react'

export function LicenseKeyDisplay({
  icon,
  title,
  subtitle,
  licenseKey,
  maskByDefault = false,
  status,
  metadata,
  actions = { copy: true },
  qrCode,
  securityNotice,
  variant = 'default'
}: LicenseKeyDisplayProps) {
  const [isMasked, setIsMasked] = useState(maskByDefault)
  const [isCopied, setIsCopied] = useState(false)

  const handleCopy = async () => {
    await navigator.clipboard.writeText(licenseKey)
    setIsCopied(true)
    setTimeout(() => setIsCopied(false), 2000)
  }

  const displayKey = isMasked
    ? licenseKey.replace(/./g, (char, i) => {
        if (i < 6 || i > licenseKey.length - 3) return char
        return '•'
      })
    : licenseKey

  return (
    <div className={`license-key-card ${variant}`}>
      <div className="lkd-header">
        <div className="lkd-icon">
          {icon === 'key' ? <Key /> : <Code />}
        </div>
        <div className="lkd-header-text">
          <div className="lkd-title">{title}</div>
          {subtitle && <div className="lkd-subtitle">{subtitle}</div>}
        </div>
        {status && (
          <div className={`lkd-status ${status.type}`}>
            {/* Status badge */}
          </div>
        )}
      </div>

      <div className="lkd-key-wrapper">
        <div className={`lkd-key ${isMasked ? 'masked' : ''}`}>
          {displayKey}
        </div>
      </div>

      <div className="lkd-actions">
        {actions.copy && (
          <button
            className={`lkd-btn lkd-btn-primary ${isCopied ? 'copied' : ''}`}
            onClick={handleCopy}
          >
            {isCopied ? <Check /> : <Copy />}
            <span>{isCopied ? 'Gekopieerd!' : 'Kopieer sleutel'}</span>
          </button>
        )}
        {actions.mask && (
          <button
            className="lkd-btn lkd-btn-secondary"
            onClick={() => setIsMasked(!isMasked)}
          >
            {isMasked ? <Eye /> : <EyeOff />}
            <span>{isMasked ? 'Toon sleutel' : 'Verberg sleutel'}</span>
          </button>
        )}
      </div>

      {/* Metadata grid, QR code, security notice */}
    </div>
  )
}

STEP 5: Layout Rules

Container: White card, max-width 700px, centered

License Key: Monospace font, 20px, centered, grey-light background, 2px grey border (teal on hover)

Meta Grid: 2-column grid (1-col on mobile), 16px gap

STEP 6: Accessibility

Keyboard Navigation: License key focusable (Tab), Enter to copy

Screen Readers: Announce "License key copied" on copy success

Color Contrast: Navy on grey-light = 14.2:1 (AAA ✓)

STEP 7: Implementation Notes

Copy to Clipboard: Use Clipboard API with fallback (document.execCommand)

Masking Logic: Show first 6 and last 3 characters, mask middle with bullets

Security: Never log license keys server-side, use HTTPS only
