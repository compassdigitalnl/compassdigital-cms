📋 STEP 1: COMPONENT OVERVIEW

Component: C7 - Cookie Consent Banner

Category: Legal / GDPR Compliance

Complexiteit: Medium (toggle states, localStorage persistence, 3 action buttons)

Purpose:

GDPR-compliant cookie consent banner with granular controls. Allows users to accept all, essential only, or customize per category (essential, analytics, marketing).

Key Features:

  • Fixed bottom-center positioning
  • 3 cookie categories: Essential (locked on), Analytics, Marketing
  • Toggle switches for each category
  • 3 action buttons: "Alles accepteren", "Voorkeuren opslaan", "Alleen noodzakelijk"
  • Slide-up animation (0.5s bounce)
  • LocalStorage persistence (cookie_consent)
  • Link to privacy policy

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • All websites serving EU visitors (GDPR requirement)
  • Sites using analytics (Google Analytics, Matomo)
  • Sites with marketing cookies (Facebook Pixel, Google Ads)

Best Practices:

  • Show on first visit (check localStorage)
  • Don't block essential functionality
  • Load analytics/marketing scripts AFTER consent
  • Provide easy access to re-open preferences (footer link)
  • Make "Reject" as easy as "Accept" (same visual weight)
  • Store consent in localStorage + cookie (365 days)

    // Enable selected cookies

    if (prefs.marketing) enableMarketing();
  },
  onEssentialOnly: () => {
    // Disable all optional cookies
    disableAnalytics();
    disableMarketing();
  },

💡 STEP 7: IMPLEMENTATION NOTES

// React Component
export function CookieBanner() {
  const [isVisible, setIsVisible] = useState(false);
  const [analytics, setAnalytics] = useState(false);
  const [marketing, setMarketing] = useState(false);

  useEffect(() => {
    const consent = localStorage.getItem('cookie_consent');
    if (!consent) {
      setIsVisible(true);
    } else {
      const prefs = JSON.parse(consent);
      setAnalytics(prefs.analytics);
      setMarketing(prefs.marketing);
      applyPreferences(prefs);
    }
  }, []);

  const saveConsent = (prefs) => {
    localStorage.setItem('cookie_consent', JSON.stringify({
      ...prefs,
      timestamp: Date.now(),
      expiresAt: Date.now() + (365 * 24 * 60 * 60 * 1000)
    }));
    setIsVisible(false);
  };

  const handleAcceptAll = () => {
    saveConsent({ essential: true, analytics: true, marketing: true });
    enableAnalytics();
    enableMarketing();
  };

  const handleSavePreferences = () => {
    saveConsent({ essential: true, analytics, marketing });
    if (analytics) enableAnalytics();
    if (marketing) enableMarketing();
  };

  const handleEssentialOnly = () => {
    saveConsent({ essential: true, analytics: false, marketing: false });
  };

  return (
    <div className={`cookie-banner ${isVisible ? 'show' : ''}`}>
      <div className="cb-body">
        <div className="cb-title"><span>🍪</span> Cookie voorkeuren</div>
        <div className="cb-text">
          Wij gebruiken cookies om uw ervaring te verbeteren. U kunt hieronder uw voorkeuren instellen.{' '}
          <a href="/privacy">Lees meer</a>
        </div>
        <div className="cb-toggles">
          <div className="cb-toggle">
            <div className="cb-toggle-info">
              <Shield size={16} />
              <span className="cb-toggle-name">Essentieel</span>
              <span className="cb-toggle-req">(verplicht)</span>
            </div>
            <div className="cb-switch locked on" />
          </div>
          <div className="cb-toggle">
            <div className="cb-toggle-info">
              <BarChart2 size={16} />
              <span className="cb-toggle-name">Analytisch</span>
            </div>
            <div className={`cb-switch ${analytics ? 'on' : ''}`} onClick={() => setAnalytics(!analytics)} />
          </div>
          <div className="cb-toggle">
            <div className="cb-toggle-info">
              <Megaphone size={16} />
              <span className="cb-toggle-name">Marketing</span>
            </div>
            <div className={`cb-switch ${marketing ? 'on' : ''}`} onClick={() => setMarketing(!marketing)} />
          </div>
        </div>
        <div className="cb-actions">
          <button className="cb-btn essential" onClick={handleEssentialOnly}>Alleen noodzakelijk</button>
          <button className="cb-btn save" onClick={handleSavePreferences}>Voorkeuren opslaan</button>
          <button className="cb-btn accept" onClick={handleAcceptAll}>Alles accepteren</button>
        </div>
      </div>
    </div>
  );
}
