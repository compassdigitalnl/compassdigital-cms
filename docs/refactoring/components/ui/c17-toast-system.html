C17-ToastSystem

Temporary notification system with 4 variants: success, info, warning, and error

STEP 1: Component Overview

C17-ToastSystem provides non-intrusive, temporary notifications that appear in the top-right corner of the screen. Toasts automatically dismiss after 5 seconds and can be manually closed by users.

Key features:

  • 4 semantic variants: success (green), info (blue), warning (amber), error (coral)
  • Auto-dismiss: 5-second timer with animated progress bar
  • Manual close: X button for immediate dismissal
  • Stack management: Multiple toasts stack vertically with 8px gap
  • Slide-in animation: Smooth translateX(120%) → translateX(0) entrance
  • Action links: Optional CTA links within toasts (e.g., "View cart", "Retry")
  • Responsive: Adapts width on mobile (<640px)

Critical feedback component: Toasts provide immediate visual feedback for user actions (add to cart, form submission, errors) without interrupting workflow.

STEP 2: When to Use

Use C17-ToastSystem when:

  • Confirming successful actions (product added to cart, order placed, settings saved)
  • Providing informational updates (subscription reminder, shipping update)
  • Warning users about important conditions (low stock, session expiring)
  • Alerting users to errors (payment failed, form validation errors)
  • You need non-blocking feedback (user can continue working while toast is visible)
  • The message is temporary and doesn't require user action to proceed

Don't use when:

  • Information is critical and must be acknowledged (use modal instead)
  • User must make a decision before continuing (use confirmation dialog)
  • Content is lengthy or requires reading time (use alert banner or modal)
  • Showing permanent status (use status badge or label)
  • Multiple related messages appear rapidly (consolidate into single toast or use notification center)

STEP 5: Layout Rules

Positioning

  • Fixed position: position: fixed; top: 20px; right: 20px;
  • Z-index: 1000 (above content, below modals at 1100)
  • Pointer events: Container has pointer-events: none to allow clicks through gaps, toasts re-enable with pointer-events: auto

Stacking Behavior

  • Vertical stack: Flexbox column with 8px gap
  • New toasts append to bottom: Latest toast appears below existing ones
  • Max visible: No hard limit, but consider limiting to 3-4 for UX
  • Auto-removal: Toasts dismiss after 5.5 seconds (5s animation + 0.5s buffer)

STEP 6: Accessibility

ARIA Live Regions

<!-- Toast stack as live region -->
<div class="toast-stack"
     id="toastStack"
     role="status"
     aria-live="polite"
     aria-atomic="false">
</div>

<!-- Individual toast -->
<div class="toast show"
     role="alert"
     aria-live="assertive">
  <!-- Toast content -->
</div>

Screen Reader Announcements

  • role="alert" on individual toasts — screen readers immediately announce new toasts
  • aria-live="assertive" for errors/warnings — interrupts current announcement
  • aria-live="polite" for success/info — waits for current announcement to finish
  • aria-atomic="true" — announces entire toast content, not just changes

Keyboard Navigation

  • Close button focusable: Users can Tab to close button and press Enter/Space to dismiss
  • Action links focusable: If toast has action link, it's keyboard accessible
  • ESC key support: Optionally dismiss all toasts with ESC key

Focus Management

function showToast(type) {
  const toast = createToast(type);
  toastStack.appendChild(toast);

  // Don't steal focus from current element
  // Toasts are non-modal and shouldn't interrupt workflow

  // Optional: If action link exists, make it focusable
  const actionLink = toast.querySelector('.toast-action');
  if (actionLink) {
    actionLink.setAttribute('tabindex', '0');
    actionLink.setAttribute('role', 'button');
  }
}

STEP 7: Implementation Notes

Complete Implementation

// Toast system initialization
let toastId = 0;

const toastConfig = {
  success: { icon: 'check-circle', bg: 'var(--green-light)', color: 'var(--green)', title: '...', text: '...', action: '...' },
  info: { icon: 'info', bg: 'var(--blue-light)', color: 'var(--blue)', title: '...', text: '...', action: '...' },
  warning: { icon: 'alert-triangle', bg: 'var(--amber-light)', color: 'var(--amber)', title: '...', text: '...', action: null },
  error: { icon: 'x-circle', bg: 'var(--coral-light)', color: 'var(--coral)', title: '...', text: '...', action: '...' }
};

function showToast(type) {
  const config = toastConfig[type];
  const id = ++toastId;

  // Create toast element
  const toast = document.createElement('div');
  toast.className = 'toast';
  toast.id = 'toast-' + id;
  toast.setAttribute('role', 'alert');
  toast.setAttribute('aria-live', type === 'error' || type === 'warning' ? 'assertive' : 'polite');
  toast.setAttribute('aria-atomic', 'true');

  toast.innerHTML = `
    <div class="toast-icon" style="background: ${config.bg}">
      <i data-lucide="${config.icon}" style="color: ${config.color}"></i>
    </div>
    <div class="toast-body">
      <div class="toast-title">${config.title}</div>
      <div class="toast-text">${config.text}</div>
      ${config.action ? `<span class="toast-action">${config.action}</span>` : ''}
    </div>
    <button class="toast-close" onclick="removeToast(${id})" aria-label="Sluiten">
      <i data-lucide="x"></i>
    </button>
    <div class="toast-progress">
      <div class="toast-progress-bar" style="background: ${config.color}"></div>
    </div>
  `;

  // Append to stack
  document.getElementById('toastStack').appendChild(toast);

  // Initialize Lucide icons
  lucide.createIcons();

  // Trigger slide-in animation
  requestAnimationFrame(() => {
    toast.classList.add('show');
  });

  // Auto-dismiss after 5.5 seconds
  setTimeout(() => {
    removeToast(id);
  }, 5500);
}

function removeToast(id) {
  const toast = document.getElementById('toast-' + id);
  if (!toast) return;

  // Slide out
  toast.classList.remove('show');

  // Remove from DOM after animation
  setTimeout(() => {
    toast.remove();
  }, 400);
}

Custom Toast Messages

For dynamic content (e.g., product name, error details), extend the function:

function showCustomToast(type, title, text, action = null) {
  const config = toastConfig[type];
  const id = ++toastId;

  // Use provided title/text instead of config defaults
  const toast = document.createElement('div');
  toast.className = 'toast';
  toast.id = 'toast-' + id;

  toast.innerHTML = `
    <div class="toast-icon" style="background: ${config.bg}">
      <i data-lucide="${config.icon}" style="color: ${config.color}"></i>
    </div>
    <div class="toast-body">
      <div class="toast-title">${title}</div>
      <div class="toast-text">${text}</div>
      ${action ? `<span class="toast-action">${action}</span>` : ''}
    </div>
    <button class="toast-close" onclick="removeToast(${id})">
      <i data-lucide="x"></i>
    </button>
    <div class="toast-progress">
      <div class="toast-progress-bar" style="background: ${config.color}"></div>
    </div>
  `;

  document.getElementById('toastStack').appendChild(toast);
  lucide.createIcons();
  requestAnimationFrame(() => toast.classList.add('show'));
  setTimeout(() => removeToast(id), 5500);
}

// Usage:
showCustomToast('success', 'Product toegevoegd', 'Welch Allyn DuraShock DS45 (2×) toegevoegd', 'Bekijk winkelwagen');

Action Link Handlers

To make action links functional:

// Add click handler to action links
const actionLink = toast.querySelector('.toast-action');
if (actionLink) {
  actionLink.addEventListener('click', (e) => {
    e.preventDefault();

    // Perform action (e.g., navigate to cart)
    if (config.action === 'Bekijk winkelwagen') {
      window.location.href = '/cart';
    } else if (config.action === 'Opnieuw proberen') {
      retryPayment();
    }

    // Close toast after action
    removeToast(id);
  });
}

Stack Limit

To prevent toast spam, limit max visible toasts:

function showToast(type) {
  // Remove oldest toast if 3+ already visible
  const toasts = document.querySelectorAll('.toast');
  if (toasts.length >= 3) {
    const oldestId = toasts[0].id.replace('toast-', '');
    removeToast(parseInt(oldestId));
  }

  // ... continue with normal toast creation
}
