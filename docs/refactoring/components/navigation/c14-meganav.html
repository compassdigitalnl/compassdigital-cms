📋 STEP 1: COMPONENT OVERVIEW

Component ID: C14 - MegaNavigation (Complete Header System)

Category: Navigation / Layout

Complexiteit: Hoog (3-laags layered navigation met 10+ sub-componenten)

Inclusieve Sub-Componenten:

  • Topbar: USPs, Login, Language Switcher (36px height)
  • Header: Logo, Search Bar, B2B/B2C Toggle, Action Buttons (72px height)
  • Navbar: Main Navigation Links, Contact Info (48px height)
  • Flyout Menu: 3-Level Layered Navigation (L1: Categories, L2: Subcategories, L3: Sub-sub + Promo)
  • Branches Dropdown: Industry-specific product navigation
  • Mobile Drawer: Full mobile navigation drawer met nested accordions

Total Height (Desktop): 156px (topbar 36px + header 72px + navbar 48px)

Positioning: Header sticky top:0, Navbar sticky top:72px, Flyout fixed top:120px

Note: Topbar is NOT sticky (scrolls away), flyout positions below sticky header+navbar only

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • E-commerce platforms met 100+ productcategorieën
  • B2B webshops met complexe navigatiestructuur
  • Multi-industry platforms (branches dropdown)
  • Sites met internationale doelgroep (language switcher)
  • Platforms met B2B/B2C prijsweergave toggle

Niet geschikt voor:

  • Kleine sites met <20 producten (te complex)
  • Content-focused sites zonder e-commerce
  • Apps waar schermruimte kritiek is (156px header height)

Real-world Implementaties:

Dit component is gebaseerd op best practices van leading B2B medische webshops zoals Plastimed, Mediq, en Manutan.

💡 STEP 7: IMPLEMENTATION NOTES

JavaScript State Management:

/* Menu toggle with backdrop sync */
function toggleMenu() {
  const menu = document.getElementById('flyoutMenu');
  const backdrop = document.getElementById('flyoutBackdrop');
  const trigger = document.getElementById('menuTrigger');
  const isOpen = menu.classList.toggle('open');
  backdrop.classList.toggle('open', isOpen);
  trigger.classList.toggle('open', isOpen);
  // Update aria-expanded
  trigger.setAttribute('aria-expanded', isOpen);
}

/* L2 panel switching (hover-based) */
function showL2(categoryId) {
  // Remove active from all L1 items
  document.querySelectorAll('.fly-col.l1 .fly-item')
    .forEach(el => el.classList.remove('active'));
  // Add active to hovered item
  event.currentTarget.classList.add('active');
  // Show corresponding L2 panel
  document.querySelectorAll('.fly-col.l2 .fly-panel')
    .forEach(el => el.classList.remove('active'));
  document.getElementById('l2-' + categoryId)?.classList.add('active');
  // Hide all L3 panels
  document.querySelectorAll('.fly-col.l3 .fly-panel')
    .forEach(el => el.classList.remove('active'));
}

Performance Optimizations:

  • CSS Transitions: Only animate opacity/transform (GPU-accelerated)
  • Lazy Icon Loading: lucide.createIcons() called after menu open
  • Event Delegation: Single event listener on parent vs individual listeners
  • Backdrop Blur: backdrop-filter: blur(3px) (may disable on low-end devices)

Mobile Drawer Behavior:

/* Nested accordion expansion */
function toggleSub(element) {
  element.classList.toggle('expanded');
  const sub = element.nextElementSibling;
  if (sub?.classList.contains('md-sub')) {
    sub.classList.toggle('open');
  }
}

/* Prevent body scroll when drawer open */
document.getElementById('mobileDrawer').addEventListener('transitionend', (e) => {
  if (e.target.classList.contains('open')) {
    document.body.style.overflow = 'hidden';
  } else {
    document.body.style.overflow = '';
  }
});

/* Update cart count dynamically */

  const badge = document.querySelector('.cart-count');
  badge.textContent = count;
  badge.style.animation = 'pulse 0.3s ease';
  setTimeout(() => badge.style.animation = '', 300);
}
