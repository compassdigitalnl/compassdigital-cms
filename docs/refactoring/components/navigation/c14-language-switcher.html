📋 STEP 1: COMPONENT OVERVIEW

Component: C14 - Language Switcher

Category: Navigation / Header

Gebruik: Language selection (typically used in topbar)

Varianten: 3 variants

Variants:

  • Button Group: 2-5 languages, horizontal buttons (NL/EN/DE)
  • With Flags: Same as button group but with flag emojis (🇳🇱 NL / 🇬🇧 EN)
  • Dropdown: 6+ languages, saves space, full language names with flags

🎯 STEP 2: WHEN TO USE

When to Use Each Variant:

  • Button Group: 2-5 languages, clean minimal look, topbar placement
  • With Flags: 2-5 languages, more visual/recognizable, helps visual learners
  • Dropdown: 6+ languages, saves horizontal space, footer or mobile

General Use Cases:

  • Multi-language websites requiring language selection
  • B2B/B2C platforms serving international customers
  • Typically placed in topbar (top-right corner)
  • E-commerce sites with multiple country/region support

Niet geschikt voor:

  • Single-language sites (obviously!)
  • Mobile-only apps (topbar hidden on mobile, use dropdown in footer instead)
  • Sites where language is auto-detected and rarely changed

📐 STEP 5: LAYOUT RULES

  • Always displayed inline (flex container)
  • 4px gap between buttons
  • Typically placed in .c14-topbar-right
  • Should be preceded by login link + divider
  • Buttons have equal width (determined by content)

♿ STEP 6: ACCESSIBILITY

  • Buttons must have aria-label (e.g., "Switch to English")
  • Active button has aria-current="true"
  • All buttons keyboard focusable (native button element)
  • Focus state should be visible (add focus ring if needed)
  • Language codes should use lang attribute

💡 STEP 7: IMPLEMENTATION NOTES

State Management

Active language is stored in cookies/localStorage and applied on page load

JavaScript Behavior

// Click handler
buttons.forEach(btn => {
  btn.addEventListener('click', (e) => {
    const langCode = e.target.textContent;
    setActiveLang(langCode);
    window.location.href = `/${langCode.toLowerCase()}${window.location.pathname}`;
  });
});

Integration with Topbar

Always placed inside .c14-topbar-right after login link and divider
