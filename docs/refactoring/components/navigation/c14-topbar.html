📋 STEP 1: COMPONENT OVERVIEW

Component: C14 - Topbar

Parent: C14-MegaNavigation (Top section)

Category: Navigation / Header

Complexiteit: Low (horizontal layout with USPs + links)

Purpose:

Top bar displaying key USPs (unique selling points), login links, and language switcher. Scrolls away (NOT sticky).

Structure:

  • Left Side: USPs met icons (4 items, flex gap 24px)
  • Right Side: Login link + separator + B2B portaal + separator + language switcher

Integrated Sub-Components:

  • C14-LanguageSwitcher: NL/EN/DE buttons (already extracted)
  • USP Icons: Lucide icons (14px, teal-light color)

Visual Details:

  • Navy background (#0A1628)
  • 36px height
  • 12.5px font size, 500 weight
  • White text with 60% opacity (rgba(255,255,255,0.6))
  • Icons in teal-light (#26A69A)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • E-commerce websites met duidelijke USPs (shipping, returns, guarantees)
  • B2B platformen met separate login portalen
  • Multi-language websites (language switcher)
  • Sites waar brand trust belangrijk is (trust badges via USPs)

Niet geschikt voor:

  • Mobile-first designs (topbar hidden <768px)
  • Sites met minimale header (topbar voegt 36px height toe)
  • Content sites zonder e-commerce (USPs niet relevant)

Common Patterns:

  • E-commerce: Shipping, returns, payment USPs
  • B2B: "Dealer login", "Request quote", "Volume pricing"
  • SaaS: "Free trial", "No credit card", "Cancel anytime"
  • Service: "24/7 support", "Free consultation", "Licensed professionals"

Dynamic USP Selection:

// Auto-select USPs based on user location/status
function getUSPs(user, location) {
  const usps = [];

  // Free shipping threshold varies by country
  if (location.country === 'NL') {
    usps.push({ icon: 'truck', text: 'Gratis verzending vanaf €150' });
  } else if (location.country === 'BE') {
    usps.push({ icon: 'truck', text: 'Gratis verzending vanaf €200' });
  }

  // B2B users see different USPs
  if (user.isB2B) {
    usps.push({ icon: 'badge-check', text: 'Voordelige B2B prijzen' });
    usps.push({ icon: 'file-text', text: 'Factuur op naam' });
  } else {
    usps.push({ icon: 'shield-check', text: 'Veilig & achteraf betalen' });
  }

  return usps;
}

💡 STEP 7: IMPLEMENTATION NOTES

Language Switcher Handler:

function setLang(btn) {
  // Remove active from all buttons
  btn.parentElement.querySelectorAll('.lang-btn')
    .forEach(b => {
      b.classList.remove('active');
      b.setAttribute('aria-pressed', 'false');
    });

  // Activate clicked button
  btn.classList.add('active');
  btn.setAttribute('aria-pressed', 'true');

  // Get language code
  const lang = btn.textContent.trim();

  // Store in localStorage
  localStorage.setItem('language', lang);

  // Redirect or update content
  window.location.href = `/${lang.toLowerCase()}${window.location.pathname}`;
}

Dynamic USP Loading:

// Fetch USPs from CMS/API
async function loadUSPs() {
  const response = await fetch('/api/settings/topbar-usps');
  const usps = await response.json();

  const container = document.querySelector('.topbar-usps');
  container.innerHTML = usps.map(usp => `
    <div class="topbar-usp">
      <i data-lucide="${usp.icon}" aria-hidden="true"></i>
      ${usp.text}
    </div>
  `).join('');

  // Reinitialize Lucide icons
  lucide.createIcons();
}

Sticky Header Adjustment:

// Adjust header position when topbar scrolls away
const topbar = document.querySelector('.topbar');
const header = document.querySelector('.header');

window.addEventListener('scroll', () => {
  const topbarHeight = topbar.offsetHeight;
  const scrolled = window.scrollY;

  if (scrolled >= topbarHeight) {
    // Topbar scrolled away, header sticky at top:0
    header.style.top = '0';
  } else {
    // Topbar visible, header below topbar
    header.style.top = `${topbarHeight - scrolled}px`;
  }
});
