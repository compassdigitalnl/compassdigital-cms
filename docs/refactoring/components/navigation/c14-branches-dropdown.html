📋 STEP 1: COMPONENT OVERVIEW

Component: C14 - Branches Dropdown

Parent: C14-MegaNavigation (Navbar)

Category: Navigation / Dropdown

Complexiteit: Low (grid of cards)

Purpose:

Industry-specific navigation dropdown showing product categories organized by professional branches (e.g., doctors, dentists, physiotherapists).

Structure:

  • Title: "Assortiment per branche" with building icon
  • 2-Column Grid: Branch cards in responsive grid
  • Branch Cards: Icon (emoji), name, product count

Visual Features:

  • White background with teal top border (2px)
  • Rounded bottom corners (16px)
  • Shadow elevation (--shadow-lg)
  • Colored icon backgrounds (teal, blue, coral, amber, green)
  • Hover effect (teal border + teal-glow background)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • Multi-industry B2B platforms (medical, construction, horeca)
  • Webshops serving multiple professional verticals
  • Sites with industry-specific product catalogs
  • Platforms where users identify by profession/industry

Common Industries/Branches:

  • Medical: Huisartsen, Tandartsen, Fysiotherapeuten, Verloskundigen
  • Construction: Electricians, Plumbers, Contractors, Architects
  • Horeca: Restaurants, Cafes, Hotels, Catering
  • Beauty: Salons, Spas, Barbershops, Nail studios
  • Automotive: Garages, Dealerships, Body shops, Detailing

Niet geschikt voor:

  • Single-industry sites (no need for branch navigation)
  • B2C-only platforms (consumers don't identify by profession)
  • Sites with <4 branches (not worth a dropdown)

Dynamic Product Count:

// Fetch real-time product counts
async function loadBranchCounts() {
  const response = await fetch('/api/branches/counts');
  const counts = await response.json();

  Object.keys(counts).forEach(branchId => {
    const countEl = document.querySelector(
      `[data-branch="${branchId}"] .branch-count`
    );
    if (countEl) {
      countEl.textContent = `${counts[branchId].toLocaleString()} producten`;
    }
  });
}

💡 STEP 7: IMPLEMENTATION NOTES

Hover/Click Toggle:

// Toggle dropdown on click (mobile-friendly)
const navLink = document.querySelector('.nav-link[aria-haspopup="true"]');
const dropdown = navLink.querySelector('.branch-dropdown');

navLink.addEventListener('click', (e) => {
  e.preventDefault();

  const isOpen = navLink.getAttribute('aria-expanded') === 'true';

  if (isOpen) {
    navLink.setAttribute('aria-expanded', 'false');
    dropdown.style.opacity = '0';
    dropdown.style.visibility = 'hidden';
  } else {
    navLink.setAttribute('aria-expanded', 'true');
    dropdown.style.opacity = '1';
    dropdown.style.visibility = 'visible';
  }
});

// Close on outside click
document.addEventListener('click', (e) => {
  if (!navLink.contains(e.target)) {
    navLink.setAttribute('aria-expanded', 'false');
    dropdown.style.opacity = '0';
    dropdown.style.visibility = 'hidden';
  }
});
