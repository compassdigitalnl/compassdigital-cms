📋 STEP 1: COMPONENT OVERVIEW

Component: C19 - Breadcrumbs

Category: Navigation

Complexiteit: Low (simple navigation trail)

Purpose:

Provide users with a hierarchical navigation trail showing their current location within the site structure, with clickable links to parent pages.

Variants:

  • Default: Simple text links with chevron (›) icon separators
  • Pill-style: Links styled as background chips with arrow (›) separators
  • Icon-enriched: Category icons next to each link for better visual hierarchy

Key Features:

  • Current page always navy (#0A1628) + bold (not clickable)
  • Links grey-mid (#94A3B8), hover teal (#00897B)
  • Semantic HTML (<ol> for ordered list)
  • SEO-friendly (helps search engines understand site structure)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • Product detail pages (Home › Category › Subcategory › Product)
  • Category/archive pages (Home › Products › Gloves)
  • Blog posts (Home › Blog › Article)
  • Account pages (Home › Account › Orders › Order #123)
  • Multi-level content pages (Home › Knowledge Base › Guides › Getting Started)

Niet geschikt voor:

  • Homepage (geen parent pagina's)
  • Standalone marketing pages zonder hiërarchie
  • Login/registratie flows (geen navigatie context)
  • Checkout process (gebruik progress steps, zie C25)

Variant Selection Guide:

  • Default: Most common, clean, minimalist
  • Pill-style: Voor sites met speelse/moderne aesthetic, ecommerce focus
  • Icon-enriched: Voor complexe site structuren, visuele hiërarchie belangrijk

Auto-Generation from URL:

// Generate breadcrumbs from pathname
function generateBreadcrumbs(pathname) {
  const segments = pathname.split('/').filter(Boolean);

  const items = [{ label: 'Home', href: '/' }];

  let path = '';
  segments.forEach((segment, i) => {
    path += `/${segment}`;
    const label = segment
      .replace(/-/g, ' ')
      .replace(/\b\w/g, l => l.toUpperCase());

    items.push({
      label,
      href: i < segments.length - 1 ? path : null  // Last = current
    });
  });

  return items;
}

♿ STEP 6: ACCESSIBILITY

Semantic HTML:

<nav aria-label="Breadcrumb">
  <ol class="breadcrumb default">
    <li><a href="/">Home</a></li>
    <li class="separator" aria-hidden="true">
      <i data-lucide="chevron-right"></i>
    </li>
    <li><a href="/producten">Producten</a></li>
    <li class="separator" aria-hidden="true">
      <i data-lucide="chevron-right"></i>
    </li>
    <li aria-current="page" class="current">
      Peha-soft Nitrile Fino
    </li>
  </ol>
</nav>

Key Accessibility Features:

  • <nav aria-label="Breadcrumb">: Identifies navigation landmark
  • <ol>: Ordered list semantically correct (hierarchical order)
  • aria-hidden="true": Hide decorative separators from screen readers
  • aria-current="page": Indicates current page to assistive tech

Keyboard Navigation:

  • Tab: Focus each breadcrumb link
  • Enter: Activate focused link
  • Current page (not a link) is not focusable

Screen Reader Behavior:

  • Announces: "Breadcrumb navigation"
  • Reads items in order: "Home, link, Producten, link, current page: Peha-soft"
  • Separators ignored (aria-hidden)

Focus Management:

/* Clear focus state */
.breadcrumb a:focus {
  outline: 2px solid var(--teal);
  outline-offset: 2px;
  border-radius: 4px;
}

💡 STEP 7: IMPLEMENTATION NOTES

React Component (TypeScript):

import { ChevronRight } from 'lucide-react';

interface BreadcrumbItem {
  label: string;
  href?: string;
  icon?: string;
}

interface BreadcrumbProps {
  items: BreadcrumbItem[];
  variant?: 'default' | 'pill' | 'icon';
}

export function Breadcrumb({ items, variant = 'default' }: BreadcrumbProps) {
  return (
    <nav aria-label="Breadcrumb">
      <ol className={`breadcrumb ${variant}`}>
        {items.map((item, i) => (
          <Fragment key={i}>
            <li>
              {i < items.length - 1 ? (
                <a href={item.href}>
                  {variant === 'icon' && item.icon && (
                    <Icon name={item.icon} />
                  )}
                  {item.label}
                </a>
              ) : (
                <span className="current" aria-current="page">
                  {variant === 'icon' && item.icon && (
                    <Icon name={item.icon} />
                  )}
                  {item.label}
                </span>
              )}
            </li>
            {i < items.length - 1 && (
              <li className="separator" aria-hidden="true">
                {variant === 'default' ? (
                  <ChevronRight size={13} />
                ) : (
                  '›'
                )}
              </li>
            )}
          </Fragment>
        ))}
      </ol>
    </nav>
  );
}

function useBreadcrumbs() {
  const router = useRouter();
  const { asPath } = router;

  const segments = asPath.split('?')[0].split('/').filter(Boolean);

  const items = [{ label: 'Home', href: '/' }];

}
