📋 STEP 1: COMPONENT OVERVIEW

Component: C14 - Search Bar

Parent: C14-MegaNavigation (Header section)

Category: Navigation / Search

Complexiteit: Medium (3 integrated elements)

Integrated Elements:

  • Search Field: Text input met focus states
  • Search Icon: Left-positioned icon (18px, reacts to focus)
  • Price Toggle: B2B/B2C switch (embedded, right side)
  • Keyboard Shortcut: ⌘K hint (right side, disabled state)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • E-commerce platforms met product search
  • B2B/B2C dual-pricing webshops
  • Sites met keyboard-first navigation (⌘K shortcut)
  • Desktop-first designs (mobile shows simplified version)

Niet geschikt voor:

  • Mobile-only designs (too wide, 600px max-width)
  • B2C-only sites (price toggle redundant)
  • Content sites without product search

Integration Points:

  • C1-InstantSearch: Opens on input focus/⌘K
  • Search API: Debounced autocomplete calls
  • Local Storage: Recent searches persistence

💡 STEP 7: IMPLEMENTATION NOTES

Price Toggle State Management:

function setPriceMode(btn) {
  // Remove active from all buttons
  btn.parentElement.querySelectorAll('.price-toggle-btn')
    .forEach(b => b.classList.remove('active'));

  // Add active to clicked button
  btn.classList.add('active');

  // Update aria-pressed
  btn.parentElement.querySelectorAll('button')
    .forEach(b => b.setAttribute('aria-pressed', 'false'));
  btn.setAttribute('aria-pressed', 'true');

  // Trigger price mode change event
  const mode = btn.textContent;
  document.dispatchEvent(new CustomEvent('pricemodechange', { detail: { mode } }));
}

Keyboard Shortcut Detection:

// Detect OS and show correct shortcut
const isMac = navigator.platform.toUpperCase().indexOf('MAC') >= 0;
const shortcutText = isMac ? '⌘K' : 'Ctrl+K';
document.querySelector('.search-shortcut').textContent = shortcutText;

// Listen for keyboard shortcut
document.addEventListener('keydown', (e) => {
  const isShortcut = isMac
    ? e.metaKey && e.key === 'k'
    : e.ctrlKey && e.key === 'k';

  if (isShortcut) {
    e.preventDefault();
    document.querySelector('.search-field').focus();
  }
});

InstantSearch Integration:

const searchField = document.querySelector('.search-field');

// Open InstantSearch on focus
searchField.addEventListener('focus', () => {
  document.getElementById('instantSearchOverlay')?.classList.add('open');
});

// Debounced autocomplete
let debounceTimer;
searchField.addEventListener('input', (e) => {
  clearTimeout(debounceTimer);
  debounceTimer = setTimeout(() => {
    if (e.target.value.length >= 2) {
      fetchAutocomplete(e.target.value);
    }
  }, 300);
});

Performance:

  • Debounce autocomplete (300ms) to reduce API calls
  • Use autocomplete="off" to prevent browser dropdown interference
  • Lazy-load search results (only fetch when needed)
