C16-Newsletter

Newsletter signup component with 2 variants: banner and inline

STEP 1: Component Overview

C16-Newsletter provides email signup forms to grow your mailing list. It comes in two variants optimized for different placement contexts.

Key features:

  • 2 variants: Banner (full-width, above footer) and Inline (compact, sidebar)
  • Banner variant: Teal gradient background, decorative glow, badge + heading + description
  • Inline variant: White card, minimal design, GDPR hint with shield icon
  • Prominent CTA: Send icon + "Aanmelden" button with hover animation
  • Fully responsive: Banner stacks vertically on mobile

STEP 2: When to Use

Use C16-Newsletter when:

  • Banner variant: Above footer, between content sections, as page-width CTA
  • Inline variant: Sidebar, product page, blog post, modal
  • You want to grow email list with prominent signup form
  • Promoting specific benefits (discounts, tips, early access)

Don't use when:

  • Newsletter is not primary CTA (use simpler link instead)
  • Already have prominent signup in header (avoid duplication)

STEP 5: Layout Rules

Banner Variant

  • Flexbox: align-items: center; justify-content: space-between;
  • Content left, form right
  • Form width: 280px input + button (flex-shrink: 0)
  • Mobile: Stack vertically, form full-width

Inline Variant

  • Vertical stack: title → text → form → hint
  • Form flexbox: input (flex: 1) + button (flex-shrink: 0)

STEP 7: Implementation Notes

Form Submission

// Example: Submit to email service
document.querySelector('.nl-form').addEventListener('submit', async (e) => {
  e.preventDefault();
  const email = e.target.querySelector('.nl-input').value;

  try {
    const res = await fetch('/api/newsletter', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ email })
    });

    if (res.ok) {
      showToast('success', 'Bedankt!', 'U bent aangemeld voor de nieuwsbrief.');
      e.target.reset();
    }
  } catch (err) {
    showToast('error', 'Fout', 'Er ging iets mis. Probeer opnieuw.');
  }
});
