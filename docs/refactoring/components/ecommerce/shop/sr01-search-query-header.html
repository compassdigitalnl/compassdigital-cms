SR01: SearchQueryHeader

Component Identity: Search results page header displaying the search query and result count

Category: Navigation / Search

Source File: ef6-search-results.html (lines 14-30)

STEP 1: Component Overview

Purpose

The SearchQueryHeader displays the user's search query and the number of results found. It provides immediate feedback on search success and helps users understand what they're looking at.

Key Features

  • Search icon: 28×28px teal search icon for visual context
  • Query display: Large (28px) bold text with quotation marks
  • Result count: Prominent teal number with "producten" unit
  • Clear labels: "Zoekresultaten voor" and "Gevonden" context
  • No results state: Grey count number when 0 results

Layout

The header uses a two-column flex layout: query on the left (flexible), result count on the right (fixed ~120px). On mobile (<640px), it stacks vertically.

STEP 2: When to Use

Use This Component When:

  • Displaying search results (any search page)
  • Users need confirmation of what they searched for
  • Showing result count is important (e.g., filtering, search)
  • You want to provide search context at page top

Don't Use This Component When:

  • Showing instant search results (use C1 InstantSearch instead)
  • Displaying category archives (use EC02 CategoryHero instead)
  • No search query exists (static pages)

Content Guidelines

  • Query display: Wrap in quotation marks ("query") to indicate exact match
  • Labels: Use "Zoekresultaten voor" (not "Results for") for Dutch
  • Unit: Use "producten" (plural) even for 1 result (or conditionally: "product" vs "producten")
  • Count formatting: No thousands separators for counts <1000 (e.g., "287" not "287")

// Highlight matched terms in query display
function highlightQuery(query: string) {
  const terms = query.split(' ')

  return terms.map((term, i) => (
    <span key={i}>
      {i > 0 && ' '}
      <mark style={{ background: 'rgba(0,137,123,0.15)', color: '#00897B' }}>
        {term}
      </mark>
    </span>
  ))
}
