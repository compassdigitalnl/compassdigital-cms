=Ë STEP 1: COMPONENT OVERVIEW

Component: C21 - Filter Sidebar

Category: E-commerce / Product Discovery

Complexiteit: High (collapsible sections, checkboxes, price slider, active filters)

Purpose:

Comprehensive filtering system for product category pages, search results, and archives. Allows users to narrow down large product catalogs using multiple criteria (brand, size, price range, material, rating, availability).

Key Features:

  • Collapsible filter cards (smooth expand/collapse animation)
  • Checkbox filters with item counts (grey badge)
  • Dual-handle price range slider with min/max inputs
  • Star rating filter (5-star, 4+ stars, 3+ stars)
  • Active filter chips (removable tags with X icon)
  • Reset all filters link
  • Sticky positioning (top: 90px, follows scroll)
  • Icon-enriched section headers (Lucide icons in teal)
  • Responsive: Horizontal scroll on mobile (<768px)

<Ż STEP 2: WHEN TO USE

Ideaal voor:

  • Product category pages (>20 products)
  • Search results pages (multi-faceted search)
  • B2B catalogs with technical specifications
  • Blog/article archives (category, author, date filters)
  • Admin data tables with complex filtering needs

Best Practices:

  • Show max 6-8 filter categories (avoid overwhelming users)
  • Always display item counts (shows availability before filtering)
  • Start with most-used filters open (brand, price, size)
  • Update URL with filter params (?brand=hartmann&size=s,m)
  • Show active filters as removable chips above product grid
  • Preserve filters when navigating back (localStorage/URL)
  • Show "No results" state when filters return 0 products
  • Auto-apply checkbox filters (no "Apply" button needed)
  • Require "Apply" button for price slider (prevents excessive API calls)

Don't Use When:

  • Product catalog has <20 items (not worth filtering)
  • Only 1-2 filter criteria available (use dropdown instead)
  • Mobile-first design (use modal/drawer filter panel)

React/TypeScript Props:

interface FilterSidebarProps {
  filters: FilterGroup[];           // Array of filter categories
  activeFilters: ActiveFilter[];    // Currently selected filters
  onFilterChange: (filters: ActiveFilter[]) => void;
  onResetAll: () => void;
  sticky?: boolean;                 // Enable sticky positioning
  stickyTop?: number;               // Sticky top offset (default: 90px)
  defaultOpen?: string[];           // Filter IDs to open by default
}

interface FilterGroup {
  id: string;                       // 'brand', 'size', 'price', etc.
  label: string;                    // Display name
  icon: string;                     // Lucide icon name
  type: 'checkbox' | 'range' | 'rating';
  options?: FilterOption[];         // For checkbox/rating filters
  range?: { min: number; max: number; step: number; }; // For price slider
  defaultOpen?: boolean;
}

interface FilterOption {
  value: string;
  label: string;
  count: number;                    // Product count for this option
}

interface ActiveFilter {
  groupId: string;
  label: string;                    // Display label for chip
  values: string[];                 // Selected values
}

Price Slider Implementation:

// Dual-handle range slider (vanilla JS example)
const track = document.getElementById('price-slider');
const thumbMin = document.getElementById('thumb-min');
const thumbMax = document.getElementById('thumb-max');
const fill = document.getElementById('price-fill');

let isDragging = null;
const min = 0, max = 100;

thumbMin.addEventListener('mousedown', () => isDragging = 'min');
thumbMax.addEventListener('mousedown', () => isDragging = 'max');

document.addEventListener('mousemove', (e) => {
  if (!isDragging) return;

  const rect = track.getBoundingClientRect();
  const percent = Math.max(0, Math.min(100,
    (e.clientX - rect.left) / rect.width * 100
  ));

  if (isDragging === 'min') {
    const maxPercent = parseFloat(thumbMax.style.left);
    if (percent < maxPercent) {
      thumbMin.style.left = percent + '%';
      fill.style.left = percent + '%';
      updatePriceInput('min', percent);
    }
  } else {
    const minPercent = parseFloat(thumbMin.style.left);
    if (percent > minPercent) {
      thumbMax.style.left = percent + '%';
      fill.style.right = (100 - percent) + '%';
      updatePriceInput('max', percent);
    }
  }
});

document.addEventListener('mouseup', () => isDragging = null);

Filter Logic (API Integration):

// Build API query from active filters
function buildFilterQuery(filters: ActiveFilter[]) {
  const query: any = {};

  filters.forEach(filter => {
    switch(filter.groupId) {
      case 'brand':
        query.brand = filter.values; // ['hartmann', 'medline']
        break;
      case 'size':
        query.size = filter.values;
        break;
      case 'price':
        query.price_min = filter.values[0];
        query.price_max = filter.values[1];
        break;
      case 'rating':
        query.rating_min = parseInt(filter.values[0]);
        break;
      case 'stock':
        query.in_stock = filter.values.includes('in-stock');
        break;
    }
  });

  return query;
}

// Example API call
const fetchProducts = async (filters: ActiveFilter[]) => {
  const query = buildFilterQuery(filters);
  const response = await fetch('/api/products?' + new URLSearchParams(query));
  const data = await response.json();
  return data.products;
};

Performance Optimization:

  • Debounce price slider: Only trigger API call 300ms after user stops dragging
  • Virtual scrolling: If >50 filter options, use react-window for checkbox lists
  • Lazy load counts: Fetch item counts async after initial page load
  • Sticky positioning: Use CSS position: sticky (no JS scroll listeners needed)

LocalStorage Persistence:

// Save filters to localStorage
function saveFiltersToStorage(filters: ActiveFilter[]) {
  localStorage.setItem('shop-filters', JSON.stringify(filters));
}

// Load filters on page load
function loadFiltersFromStorage(): ActiveFilter[] {
  const saved = localStorage.getItem('shop-filters');
  return saved ? JSON.parse(saved) : [];
}

// Clear filters after 24 hours
const filterTimestamp = localStorage.getItem('shop-filters-timestamp');
if (Date.now() - parseInt(filterTimestamp) > 86400000) {
  localStorage.removeItem('shop-filters');
}
