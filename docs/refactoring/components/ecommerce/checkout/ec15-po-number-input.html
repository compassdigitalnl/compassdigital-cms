Purpose

B2B purchase order number input field for checkout and quote requests. Allows business customers to add their internal reference number (PO number) which will appear on invoices and packing slips for internal administration and accounting.

Visual Pattern

  • Label with file-text icon + optional "B2B" badge
  • Monospace input field (JetBrains Mono font)
  • Uppercase text transformation (automatic)
  • Optional helper text below input
  • Teal focus state with glow ring
  • Non-required field (no red asterisk)

Key Features

  • ✅ Monospace font (JetBrains Mono) for code-like numbers
  • ✅ Automatic uppercase transformation
  • ✅ Letter-spacing for readability
  • ✅ Optional field (B2B convenience, not required)
  • ✅ Helper text explaining usage
  • ✅ B2B badge indicator (optional)
  • ✅ Icon variants (label icon or input icon)
  • ✅ Compact variant for inline forms
  • ✅ Max length validation (50 characters)

STEP 2: When to Use

✅ USE this component when:

  • ✅ B2B checkout — collect PO number for invoice reference
  • ✅ Quote requests — link quote to customer's internal order
  • ✅ Account order forms — recurring orders with PO tracking
  • ✅ Invoice payment — match payment to specific PO
  • ✅ Return/RMA forms — reference original PO number

❌ DON'T use this component when:

  • ❌ B2C checkout — consumers don't use PO numbers
  • ❌ Guest checkout — requires B2B account
  • ❌ Digital products — no physical invoice/pakbon
  • ❌ Admin panel — use standard input

Integration Points

  • Checkout location: After address form (EC14), before payment selection
  • Invoice display: Shows on PDF invoices and packing slips
  • Order details: Visible in order history and admin panel
  • Related components:
    • EC14: AddressForm (B2B company info section)
    • EC07: OrderSummary (may show PO number in summary)
    • Invoice templates (displays PO number)

STEP 7: Implementation Notes

React/Next.js Component

'use client';

import { FileText, Info } from 'lucide-react';

export function PONumberInput({
  name = 'poNumber',
  label = 'Referentienummer / PO-nummer',
  placeholder = 'Uw interne referentie voor op de factuur',
  helperText = 'Dit nummer verschijnt op uw factuur en pakbon voor interne administratie.',
  maxLength = 50,
  variant = 'default',
  showIcon = true,
  iconPosition = 'label',
  showB2BBadge = false,
  value,
  onChange,
  disabled = false,
  required = false,
  className = '',
}: PONumberInputProps) {
  const handleChange = (e: React.ChangeEvent<HTMLInputElement>) => {
    // Auto-uppercase transformation
    const uppercased = e.target.value.toUpperCase();
    onChange?.(uppercased);
  };

  return (
    <div className={`po-number-group ${variant} ${className}`}>
      <label className="po-label" htmlFor={name}>
        {showIcon && iconPosition === 'label' && (
          <FileText size={16} aria-hidden="true" />
        )}
        {label}
        {!required && <span className="optional">(optioneel)</span>}
        {showB2BBadge && <span className="b2b-badge">B2B</span>}
      </label>

      {iconPosition === 'input' ? (
        <div className="po-input-wrapper">
          <FileText size={16} className="po-input-icon" aria-hidden="true" />
          <input
            className="po-input"
            type="text"
            id={name}
            name={name}
            placeholder={placeholder}
            value={value}
            onChange={handleChange}
            disabled={disabled}
            required={required}
            maxLength={maxLength}
            aria-describedby={helperText ? `${name}-helper` : undefined}
          />
        </div>
      ) : (
        <input
          className="po-input"
          type="text"
          id={name}
          name={name}
          placeholder={placeholder}
          value={value}
          onChange={handleChange}
          disabled={disabled}
          required={required}
          maxLength={maxLength}
          aria-describedby={helperText ? `${name}-helper` : undefined}
        />
      )}

      {helperText && (
        <div className="po-helper" id={`${name}-helper`}>
          <Info size={14} aria-hidden="true" />
          <span>{helperText}</span>
        </div>
      )}
    </div>
  );
}

const PO_NUMBER_PATTERN = /^[A-Z0-9-]+$/; // Alphanumeric + dashes only

const validatePONumber = (value: string): boolean => {
  if (!value) return true; // Optional field
