STEP 1: Component Overview

ID: EC11
Name: ShippingMethodCard
Category: E-commerce / Checkout
Source: ef3-checkout.html (lines 51-86)
Priority: 🔴 Critical (Phase 1 - Checkout Flow)
Complexity: Medium

Selecteerbare verzendmethode kaarten voor de checkout flow. Radio button card met icon, naam, levertijd en prijs. Ondersteunt 3 standaard methoden: Standaard verzending, Express verzending, en Afhalen (gratis).

STEP 2: When to Use

✅ Use when:

  • Checkout flow met verzendmethode selectie (stap 2 van 4)
  • 3+ verzendopties beschikbaar (Standaard/Express/Afhalen)
  • Visuele verschillen tussen methoden belangrijk zijn (icon + prijs)
  • Mobile-friendly selectie gewenst is (grote klikbare kaarten)
  • B2B en B2C orders (beide gebruiken verzending)

❌ Don't use when:

  • Alleen 1 verzendmethode beschikbaar (gebruik gewoon tekst)
  • Digitale producten (geen verzending nodig)
  • Quote request flow (nog geen checkout)
  • Meer dan 5 opties (wordt te druk, gebruik dropdown)

React Component Example:

'use client'

import { Truck, Zap, Package } from 'lucide-react'

const iconMap = {
  truck: Truck,
  zap: Zap,
  package: Package
}

const iconColorMap = {
  truck: 'text-teal',
  zap: 'text-amber',
  package: 'text-teal'
}

export function ShippingMethodCard({
  method,
  selected,
  onSelect,
  disabled = false
}: ShippingMethodCardProps) {
  const Icon = iconMap[method.icon]
  const iconColor = iconColorMap[method.icon]

  const isFree = method.price === 0 || method.isFree

  return (
    <label
      className={`
        shipping-method
        ${selected ? 'shipping-method--selected' : ''}
        ${disabled ? 'shipping-method--disabled' : ''}
      `}
    >
      <input
        type="radio"
        name="shipping-method"
        value={method.id}
        checked={selected}
        onChange={() => onSelect(method.id)}
        disabled={disabled}
        className="shipping-method__radio"
      />

      <div className="shipping-method__header">
        <Icon className={`shipping-method__icon ${iconColor}`} />
        <div className="shipping-method__name">{method.name}</div>
      </div>

      <div className="shipping-method__time">{method.deliveryTime}</div>

      <div className={`
        shipping-method__price
        ${isFree ? 'shipping-method__price--free' : ''}
      `}>
        {isFree ? 'Gratis' : `€ ${method.price.toFixed(2)}`}
      </div>
    </label>
  )
}

STEP 6: Accessibility

ARIA & Semantics:

  • <label> wraps entire card (click anywhere selects)
  • type="radio" native radio input (keyboard accessible)
  • name="shipping-method" groups all options
  • Icons have descriptive class names (screenreader context via label text)

Keyboard Navigation:

  • Tab: Focus op radio buttons
  • Space/Enter: Select optie
  • Arrow keys: Navigate tussen radio buttons
  • Focus ring: outline: 3px solid rgba(0,137,123,0.12)

Screen Readers:

  • Announces: "Standaard, radio button, 1-2 werkdagen, € 6,95, checked/not checked"
  • Label text provides full context (icon is decorative)
  • Price and time are read as part of label

STEP 7: Implementation Notes

React/Next.js Integration:

  • Use useState voor selected shipping method ID
  • Fetch shipping methods van Payload CMS Site Settings
  • Filter op isActive: true
  • Calculate free shipping dynamically (cart total vs freeThreshold)
  • Update Order Summary wanneer methode wijzigt

State Management:

const [selectedShipping, setSelectedShipping] = useState<string | null>(null)

// Auto-select first method on mount
useEffect(() => {
  if (!selectedShipping && methods.length > 0) {
    setSelectedShipping(methods[0].id)
  }
}, [methods, selectedShipping])

// Update cart total when shipping changes
useEffect(() => {
  const method = methods.find(m => m.id === selectedShipping)
  if (method) {
    updateCartTotal({ shippingCost: method.price })
  }
}, [selectedShipping])

Important Details:

  • ⚠️ Always show at least 1 shipping method (default: Standard)
  • ⚠️ Disable Express/Same-Day voor bepaalde postcodes (rural areas)
  • ⚠️ Show "Gratis" in green voor free shipping (price = 0)
  • ⚠️ Hover state: teal border + subtle shadow
  • ⚠️ Mobile: Stack kaarten verticaal (1 per rij)
  • ⚠️ Delivery date calculation: Add estimatedDays to today (skip weekends)

API Validation Example:

// app/api/checkout/validate-shipping/route.ts
export async function POST(req: Request) {
  const { methodId, postalCode, cartTotal } = await req.json()

  const settings = await payload.findGlobal({ slug: 'site-settings' })
  const method = settings.shippingMethods.find(m => m.slug === methodId)

  if (!method || !method.isActive) {
    return Response.json({
      valid: false,
      message: 'Deze verzendmethode is niet beschikbaar'
    }, { status: 400 })
  }

  // Check free threshold
  const isFree = method.freeThreshold && cartTotal >= method.freeThreshold
  const finalPrice = isFree ? 0 : method.price

  return Response.json({
    valid: true,
    price: finalPrice,
    isFree,
    estimatedDelivery: calculateDeliveryDate(method.estimatedDays)
  })
}
