Purpose

Success message hero section displayed at the top of the order confirmation page after successful checkout. Features a green-to-teal gradient background with checkmark icon animation, success message, and prominent order number display. Provides immediate positive feedback to the customer.

Visual Pattern

  • Green (#00C853) to teal (#00897B) gradient background (135deg)
  • Decorative white radial glow (top-right corner)
  • Centered layout with white text
  • Large checkmark icon in frosted glass circle (80×80px)
  • Order number badge with backdrop blur
  • Staggered fade-in animations (icon → title → description → badge)

Key Features

  • ✅ Green gradient (success color psychology)
  • ✅ Checkmark icon animation (scale-in bounce)
  • ✅ Staggered content animations (fadeInUp)
  • ✅ Frosted glass effect (backdrop-filter: blur)
  • ✅ Monospace order number (JetBrains Mono)
  • ✅ Responsive design (smaller on mobile)
  • ✅ Decorative radial glow overlay
  • ✅ High contrast white text on gradient

STEP 2: When to Use

✅ USE this component when:

  • ✅ Order confirmation page — after successful checkout
  • ✅ Payment success — after payment completion
  • ✅ Quote submission — after quote request sent
  • ✅ Registration success — after account creation
  • ✅ Subscription activation — after subscription signup

❌ DON'T use this component when:

  • ❌ Payment failed — use error hero instead
  • ❌ Order pending — use warning/info hero
  • ❌ Toast notifications — too prominent for small actions
  • ❌ Admin panel — use standard success alerts

Integration Points

  • Page location: Top of order confirmation page (after checkout redirect)
  • After payment: Redirect to /order-confirmation/[orderId]
  • Email trigger: Success hero display triggers confirmation email
  • Related components:
    • OC02: OrderDetailsCard (below hero)
    • OC03: OrderItemsSummary (order details)
    • OC04: NextStepsCTA (action buttons)
    • OC05: EmailConfirmationBanner (email note)

STEP 7: Implementation Notes

React/Next.js Component

'use client';

import { Check } from 'lucide-react';

export function SuccessHero({
  orderNumber,
  title = 'Bedankt voor je bestelling!',
  description = 'Je bestelling is succesvol geplaatst en wordt zo snel mogelijk verwerkt.',
  orderNumberLabel = 'Ordernummer:',
  enableAnimation = true,
  variant = 'default',
  className = '',
}: SuccessHeroProps) {
  return (
    <section
      className={`success-hero ${variant} ${className}`}
      role="status"
      aria-live="polite"
    >
      <div className="success-content">
        <!-- Checkmark icon -->
        <div className="success-icon" aria-hidden="true">
          <Check size={48} />
        </div>

        <!-- Title -->
        <h1 className="success-title">{title}</h1>

        <!-- Description -->
        <p className="success-description">{description}</p>

        <!-- Order number badge -->
        <div className="order-number-badge">
          <span className="order-number-label">{orderNumberLabel}</span>
          <span
            className="order-number"
            aria-label={`Order nummer ${orderNumber.replace(/-/g, ' ')}`}
          >
            #{orderNumber}
          </span>
        </div>
      </div>
    </section>
  );
}

Order Confirmation Page Integration

// /app/order-confirmation/[orderId]/page.tsx

export default async function OrderConfirmationPage({
  params,
}: {
  params: { orderId: string };
}) {
  const order = await getOrder(params.orderId);

  if (!order) {
    notFound();
  }

  return (
    <div className="order-confirmation-page">
      <!-- Success Hero -->
      <SuccessHero orderNumber={order.orderNumber} />

      <!-- Order Details Cards (OC02) -->
      <OrderDetailsCard order={order} />

      <!-- Order Items Summary (OC03) -->
      <OrderItemsSummary items={order.items} />

      <!-- Next Steps CTAs (OC04) -->
      <NextStepsCTA orderId={order.id} />

      <!-- Email Confirmation Banner (OC05) -->
      <EmailConfirmationBanner email={order.email} />
    </div>
  );
}

Animation Control (Optional)

// Disable animations on repeat visits

const [showAnimation, setShowAnimation] = useState(true);

  } else {
