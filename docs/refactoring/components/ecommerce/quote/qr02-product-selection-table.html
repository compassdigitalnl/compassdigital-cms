📋 Component Documentation

STEP 1: Component Overview

Component ID: QR02

Component Name: ProductSelectionTable

Category: E-commerce / Quote Request

Priority: 🟢 Low

Source File: ef7-offerte.html

Output Location: /refactoring/components/ecommerce/qr02-product-selection-table.html

Purpose:
ProductSelectionTable displays and manages the list of products included in a B2B quote request. Users can add products, adjust quantities, and remove items. It serves as the central product management interface for the quote request form, allowing bulk selection from cart or catalog.

Key Features:

  • Header with title + "Product toevoegen" button
  • Product rows with thumbnail, name, SKU, qty input, delete button
  • Quantity input (100px wide, monospace font, number type)
  • Delete button with trash icon (hover: coral border + bg)
  • Empty state when no products selected
  • Responsive: rows stack on mobile
  • Product name truncation for long names
  • Add button hover: teal bg + white text

When to Use:

  • ✅ Quote request (offerte) forms
  • ✅ B2B bulk order forms
  • ✅ Project/tender product selection
  • ✅ RFQ (Request for Quote) flows
  • ❌ Don't use for B2C cart (use CartLineItem instead)
  • ❌ Don't use for single-product pages
  • ❌ Don't use for product comparison (different layout)

STEP 2: When to Use

Use Cases:

  • B2B Quote Requests: Customer selects 5-10 products for bulk pricing
  • Project Orders: Construction/medical project with specific product list
  • Tender Submissions: Government procurement with defined product specs
  • Wholesale Orders: Distributor ordering from manufacturer
  • Custom Bundles: Creating custom product bundles for clients

Product Addition Methods:

  • Import from cart (user adds to cart first, then requests quote)
  • Search/autocomplete modal (click "Product toevoegen")
  • Import from saved bestellijsten
  • CSV upload (bulk import via QO04 pattern)

Anti-patterns:

  • ❌ Allowing 0 quantity (min should be 1)
  • ❌ No validation on delete (at least 1 product required)
  • ❌ Tiny quantity inputs on mobile (hard to tap)
  • ❌ No empty state (confusing when table is empty)
  • ❌ Missing SKU (hard to identify products)

Guidelines:

  • Require at least 1 product before allowing quote submission
  • Validate quantities (min 1, max based on stock/MOQ)
  • Show SKU for product identification
  • Provide "Add from cart" shortcut if cart has items
  • Consider adding price column if available (optional)
  • Track changes for analytics (products added/removed)

STEP 4: Props & Configuration

TypeScript Interface:

interface Product {
  id: string
  name: string
  sku: string
  thumbnail?: string         // Image URL or emoji
  quantity: number
  price?: number             // Optional: show price column
  maxQuantity?: number       // Stock limit or MOQ
}

interface ProductSelectionTableProps {
  products: Product[]
  onProductAdd?: () => void             // Opens product search modal
  onProductRemove?: (id: string) => void
  onQuantityChange?: (id: string, qty: number) => void
  showPrices?: boolean                   // Show price column
  emptyMessage?: string
}

React Component Example:

'use client'
import { Plus, Trash2, PackageX } from 'lucide-react'

export function ProductSelectionTable({
  products,
  onProductAdd,
  onProductRemove,
  onQuantityChange,
  emptyMessage = 'Nog geen producten geselecteerd'
}: ProductSelectionTableProps) {
  if (products.length === 0) {
    return (
      <div className="product-selection-table">
        <div className="product-selection-header">
          <div className="product-selection-title">Geselecteerde producten</div>
          <button className="product-add-button" onClick={onProductAdd}>
            <Plus className="product-add-icon" />
            Product toevoegen
          </button>
        </div>
        <div className="product-selection-empty">
          <PackageX className="product-selection-empty-icon" />
          <div className="product-selection-empty-text">{emptyMessage}</div>
          <div className="product-selection-empty-hint">
            Klik op "Product toevoegen" om te beginnen
          </div>
        </div>
      </div>
    )
  }

  return (
    <div className="product-selection-table">
      <div className="product-selection-header">
        <div className="product-selection-title">Geselecteerde producten</div>
        <button className="product-add-button" onClick={onProductAdd}>
          <Plus className="product-add-icon" />
          Product toevoegen
        </button>
      </div>
      <div className="product-selection-body">
        {products.map(product => (
          <div key={product.id} className="product-row">
            <div className="product-thumbnail">
              {product.thumbnail ? (
                <img src={product.thumbnail} alt={product.name} />
              ) : (
                '📦'
              )}
            </div>
            <div className="product-info">
              <div className="product-name">{product.name}</div>
              <div className="product-sku">SKU: {product.sku}</div>
            </div>
            <input
              type="number"
              min="1"
              max={product.maxQuantity}
              value={product.quantity}
              onChange={(e) => onQuantityChange?.(product.id, parseInt(e.target.value))}
              className="product-qty-input"
            />
            <button
              className="product-delete-button"
              onClick={() => onProductRemove?.(product.id)}
              aria-label="Verwijder product"
            >
              <Trash2 className="product-delete-icon" />
            </button>
          </div>
        ))}
      </div>
    </div>
  )
}

STEP 7: Implementation Notes

State Management:

const [products, setProducts] = useState<Product[]>([])

const handleAddProduct = (newProduct: Product) => {
  setProducts([...products, newProduct])
}

const handleRemoveProduct = (id: string) => {
  if (products.length === 1) {
    alert('Een offerte moet minimaal 1 product bevatten')
    return
  }
  setProducts(products.filter(p => p.id !== id))
}

const handleQuantityChange = (id: string, qty: number) => {
  if (qty < 1) return // Validate minimum
  setProducts(products.map(p =>
    p.id === id ? { ...p, quantity: qty } : p
  ))
}

Product Addition Modal Integration:

const [showProductModal, setShowProductModal] = useState(false)

<ProductSelectionTable
  products={products}
  onProductAdd={() => setShowProductModal(true)}
  onProductRemove={handleRemoveProduct}
  onQuantityChange={handleQuantityChange}
/>

{showProductModal && (
  <ProductSearchModal
    onSelect={(product) => {
      handleAddProduct(product)
      setShowProductModal(false)
    }}
    onClose={() => setShowProductModal(false)}
  />
)}

Validation:

function validateQuoteRequest(products: Product[]): string[] {
  const errors: string[] = []

  if (products.length === 0) {
    errors.push('Selecteer minimaal 1 product')
  }

  products.forEach(p => {
    if (p.quantity < 1) {
      errors.push(`${p.name}: hoeveelheid moet minimaal 1 zijn`)
    }
    if (p.maxQuantity && p.quantity > p.maxQuantity) {
      errors.push(`${p.name}: maximaal ${p.maxQuantity} beschikbaar`)
    }
  })

  return errors
}
