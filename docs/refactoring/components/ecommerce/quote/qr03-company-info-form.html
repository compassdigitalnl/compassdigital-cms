STEP 1: Component Overview

Component Name: CompanyInfoForm (QR03)

Category: Forms / B2B

Usage Context: Quote request flow, B2B order forms, company profile pages

Purpose:

Displays company information for authenticated B2B users in a read-only form. Company details are auto-filled from the user's account and cannot be edited (verified business data). Optionally includes editable contact person fields for the specific quote/order.

Key Features:

  • ✅ Auto-filled company data (name, KVK, BTW, address)
  • ✅ Read-only verified fields with grey background
  • ✅ "Geverifieerd" (Verified) badge for trust
  • ✅ Optional editable contact person fields
  • ✅ Helper text explaining data source
  • ✅ 2-column responsive grid (1-col on mobile)
  • ✅ Building icon + shield-check badge for visual hierarchy

STEP 2: When to Use

Use this component when:

  • ✅ B2B user is requesting a quote (offerte)
  • ✅ Placing an order that requires invoice details
  • ✅ Displaying company profile information
  • ✅ Company data is verified and should not be changed
  • ✅ You need to show trust signals (verified badge)

Do NOT use when:

  • ❌ User is editing their company profile (use editable form)
  • ❌ During registration/onboarding (use editable fields)
  • ❌ For B2C users (they don't have company data)
  • ❌ Company data is unverified (remove badge, allow edits)

Component Guidelines:

  • Always show "Geverifieerd" badge if company is KVK-verified
  • Use grey background (#F5F7FA) for read-only fields to indicate non-editable state
  • Include helper text explaining why fields are locked
  • Link to account settings if user wants to update company data
  • Validate editable fields (contact person) in real-time

STEP 4: Props & Configuration

TypeScript Interface:

interface CompanyInfo {
  name: string
  kvk: string              // Chamber of Commerce number
  btw: string              // VAT number (NL123456789B01)
  address: string
  verified: boolean        // Show verified badge
  contactPerson?: {
    name: string
    phone: string
    email: string
  }
}

interface CompanyInfoFormProps {
  companyInfo: CompanyInfo
  showContactFields?: boolean      // Show editable contact person
  onContactChange?: (field: keyof CompanyInfo['contactPerson'], value: string) => void
  helperText?: string              // Custom helper text
  showEditButton?: boolean         // Show "Wijzig contactpersoon" button
  className?: string
}

STEP 7: Implementation Notes

React/Next.js Component:

'use client'

import { Building2, ShieldCheck, Info } from 'lucide-react'
import { CompanyInfoFormProps } from '@/types/components'

export function CompanyInfoForm({
  companyInfo,
  showContactFields = false,
  onContactChange,
  helperText = 'Deze gegevens zijn gekoppeld aan uw bedrijfsaccount en kunnen niet worden aangepast.',
  className = '',
}: CompanyInfoFormProps) {
  return (
    <div className={`company-info-form ${className}`}>
      <div className="company-info-header">
        <h3 className="company-info-title">
          <Building2 className="company-info-title-icon" />
          Bedrijfsgegevens
        </h3>
        {companyInfo.verified && (
          <span className="company-info-badge">
            <ShieldCheck className="company-info-badge-icon" />
            Geverifieerd
          </span>
        )}
      </div>

      <div className="company-info-fields">
        {/* Company Name */}
        <div className="company-info-field">
          <label className="company-info-label">Bedrijfsnaam</label>
          <input
            type="text"
            className="company-info-input"
            value={companyInfo.name}
            readOnly
          />
        </div>

        {/* KVK + BTW */}
        <div className="company-info-row">
          <div className="company-info-field">
            <label className="company-info-label">KVK-nummer</label>
            <input
              type="text"
              className="company-info-input"
              value={companyInfo.kvk}
              readOnly
            />
          </div>
          <div className="company-info-field">
            <label className="company-info-label">BTW-nummer</label>
            <input
              type="text"
              className="company-info-input"
              value={companyInfo.btw}
              readOnly
            />
          </div>
        </div>

        {/* Address */}
        <div className="company-info-field">
          <label className="company-info-label">Adres</label>
          <input
            type="text"
            className="company-info-input"
            value={companyInfo.address}
            readOnly
          />
        </div>

        {/* Contact fields (if enabled) */}
        {showContactFields && companyInfo.contactPerson && (
          <>
            <div className="company-info-row">
              <div className="company-info-field">
                <label className="company-info-label">Contactpersoon</label>
                <input
                  type="text"
                  className="company-info-input editable"
                  value={companyInfo.contactPerson.name}
                  onChange={(e) => onContactChange?.('name', e.target.value)}
                />
              </div>
              <div className="company-info-field">
                <label className="company-info-label">Telefoonnummer</label>
                <input
                  type="tel"
                  className="company-info-input editable"
                  value={companyInfo.contactPerson.phone}
                  onChange={(e) => onContactChange?.('phone', e.target.value)}
                />
              </div>
            </div>
            <div className="company-info-field">
              <label className="company-info-label">E-mailadres</label>
              <input
                type="email"
                className="company-info-input editable"
                value={companyInfo.contactPerson.email}
                onChange={(e) => onContactChange?.('email', e.target.value)}
              />
            </div>
          </>
        )}

        {/* Helper text */}
        <p className="company-info-helper">
          <Info className="company-info-helper-icon" />
          {helperText}
        </p>
      </div>
    </div>
  )
}

Integration with Payload CMS:

// app/(frontend)/offerte/page.tsx
import { CompanyInfoForm } from '@/components/forms/CompanyInfoForm'
import { getPayloadHMR } from '@payloadcms/next/utilities'

export default async function OffertePage() {
  const payload = await getPayloadHMR({ config })

  // Get company from authenticated user
  const user = await payload.auth()
  const company = await payload.findByID({
    collection: 'companies',
    id: user.company,
  })

  return (
    <main>
      <CompanyInfoForm
        companyInfo={{
          name: company.name,
          kvk: company.kvk,
          btw: company.btw,
          address: company.address,
          verified: company.verified,
        }}
        showContactFields={true}
      />
    </main>
  )
}

Performance Optimization:

  • ✅ Read-only inputs: Add readOnly and tabindex="-1" to prevent focus
  • ✅ Contact fields: Debounce onChange handlers (300ms)
  • ✅ Icons: Use Lucide React with tree-shaking
  • ✅ Verified badge: Only render if verified === true
