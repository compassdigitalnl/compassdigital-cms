📋 STEP 1: Component Overview

Component ID: EC01

Component Name: ProductCard

Category: E-commerce

Source File: /ecommerce-flows/ef1-shop-archive.html

Target Folder: /refactoring/components/ecommerce/

Description

Product card component with two layout variants (Grid and List). Displays product information including image, brand, title, SKU, pricing (regular + sale + volume discounts), star rating, stock status, and add-to-cart button. Supports product badges (Sale, Nieuw, Pro, Popular) and hover effects.

Key Features

  • ✅ Two variants: Grid (vertical) and List (horizontal) layouts
  • ✅ Product badges: 4 types (Sale/Nieuw/Pro/Popular) with color coding
  • ✅ Pricing display: Current price + old price (sale) + unit text + volume discount hints
  • ✅ Star rating: 5-star visual rating + review count
  • ✅ Stock indicators: 3 states (in-stock/low/out) with colored dots
  • ✅ Hover effect: Elevation animation (translateY + shadow + border color)
  • ✅ Add-to-cart CTA: Circular button with teal background
  • ✅ Accessibility: Semantic HTML (<a> wrapper) for full card clickability

🎯 STEP 2: When to Use

Use this component when:

  • ✅ Displaying products in shop archive / category pages
  • ✅ Search results pages
  • ✅ Related products sections
  • ✅ Product recommendations / upsells
  • ✅ Wishlist / favorites pages
  • ✅ Compare products view

Grid vs List Variant:

  • Grid Variant: Default view for shop archives (3-column layout on desktop)
  • List Variant: Alternative view when user toggles to list mode (horizontal layout showing more product details)

Don't use this component for:

  • ❌ Full product detail pages (use dedicated product page template)
  • ❌ Cart line items (use CartLineItem component - EC06)
  • ❌ Quick order tables (use QuickOrderRow component - QO03)

📐 STEP 5: Layout Rules

Grid Variant (Vertical)

  • Container: flex-direction: column
  • Image: 200px fixed height, 100% width, centered emoji/product image
  • Body: 18px padding, flex-grow to fill available space
  • Footer: margin-top: auto to push to bottom
  • Badge: Absolute positioned top-left (12px offset)
  • Stock indicator: Below footer with top border separator

List Variant (Horizontal)

  • Container: flex-direction: row
  • Image: 180px fixed width, auto height
  • Body: flex-direction: row with 24px gap
  • Info section: flex: 1 to take available space
  • Footer: flex-direction: column, aligned right, min-width 180px
  • Stock indicator: Next to add-to-cart button (no border separator)

Responsive Behavior

  • Desktop (≥1024px): 3-column grid for grid variant
  • Tablet (768-1023px): 2-column grid
  • Mobile (<768px): 1-column grid, list variant switches to vertical

💻 STEP 7: Implementation Notes

React/Next.js Component Example

'use client'

import { ShoppingCart, Star, TrendingDown } from 'lucide-react'
import Image from 'next/image'
import Link from 'next/link'
import { Product } from '@/types'

interface ProductCardProps {
  product: Product
  variant?: 'grid' | 'list'
  onAddToCart?: (productId: string) => void
}

export function ProductCard({
  product,
  variant = 'grid',
  onAddToCart
}: ProductCardProps) {
  const {
    id, name, slug, sku, brand, image, price, compareAtPrice,
    unit, volumePricing, rating, reviewCount, stock, stockStatus, badges
  } = product

  const handleAddToCart = (e: React.MouseEvent) => {
    e.preventDefault()
    onAddToCart?.(id)
  }

  const getStockClass = () => {
    switch (stockStatus) {
      case 'in-stock': return 'product-card__stock--in-stock'
      case 'low': return 'product-card__stock--low'
      case 'out': return 'product-card__stock--out'
      default: return ''
    }
  }

  const getStockText = () => {
    switch (stockStatus) {
      case 'in-stock': return `Op voorraad (${stock} stuks)`
      case 'low': return `Laag op voorraad (${stock} stuks)`
      case 'out': return 'Tijdelijk uitverkocht'
      default: return ''
    }
  }

  const bestVolumeTier = volumePricing?.[0]

  return (
    <Link
      href={`/products/${slug}`}
      className={`product-card ${variant === 'list' ? 'product-card--list' : ''}`}
      aria-label={`${name} door ${brand.name}, €${price.toFixed(2)}`}
    >
      {/* Badge */}
      {badges?.length > 0 && (
        <span
          className={`product-card__badge product-card__badge--${badges[0].type}`}
          aria-label={`${badges[0].type} product`}
        >
          {badges[0].label || badges[0].type}
        </span>
      )}

      {/* Image */}
      <div className="product-card__image">
        {image ? (
          <Image
            src={image.url}
            alt={image.alt || name}
            fill
            style={{ objectFit: 'contain' }}
          />
        ) : (
          <span>🧤</span>
        )}
      </div>

      {/* Body */}
      <div className="product-card__body">
        <div className={variant === 'list' ? 'product-card__info' : ''}>
          <div className="product-card__brand">{brand.name}</div>
          <div className="product-card__title">{name}</div>
          <div className="product-card__sku">SKU: {sku}</div>

          {/* Rating */}
          {rating && (
            <div
              className="product-card__rating"
              role="img"
              aria-label={`${rating} van 5 sterren, ${reviewCount} reviews`}
            >
              {[...Array(5)].map((_, i) => (
                <Star
                  key={i}
                  size={13}
                  fill={i < Math.floor(rating) ? '#F59E0B' : '#E8ECF1'}
                  color={i < Math.floor(rating) ? '#F59E0B' : '#E8ECF1'}
                />
              ))}
              <span className="product-card__rating-count">({reviewCount})</span>
            </div>
          )}
        </div>

        {/* Footer */}
        <div className="product-card__footer">
          <div className="product-card__price-wrapper">
            <div className="product-card__price">
              € {Math.floor(price)}<small>,{(price % 1).toFixed(2).slice(2)}</small>
              {compareAtPrice && (
                <span className="product-card__price--old">
                  € {compareAtPrice.toFixed(2)}
                </span>
              )}
            </div>
            {unit && <div className="product-card__unit">{unit}</div>}
            {bestVolumeTier && (
              <div className="product-card__staffel-hint">
                <TrendingDown size={12} />
                Vanaf {bestVolumeTier.minQty} stuks: € {bestVolumeTier.price.toFixed(2)} (−{bestVolumeTier.discountPercent}%)
              </div>
            )}
          </div>

          {/* Add to cart */}
          <button
            className="product-card__add-to-cart"
            onClick={handleAddToCart}
            aria-label={`Voeg ${name} toe aan winkelwagen`}
          >
            <ShoppingCart size={18} />
          </button>
        </div>

        {/* Stock */}
        <div
          className={`product-card__stock ${getStockClass()}`}
          role="status"
          aria-live="polite"
        >
          <div className="product-card__stock-dot"></div>
          {getStockText()}
        </div>
      </div>
    </Link>
  )
}

Important Implementation Details

  • ✅ Prevent default on add-to-cart: Use e.preventDefault() to prevent card navigation when clicking button
  • ✅ Image optimization: Use Next.js <Image> with fill and objectFit: 'contain'
  • ✅ Stock status auto-calculation: Calculate stockStatus based on stock value in Payload hooks:
    • stock === 0 → 'out'
    • stock > 0 && stock < 100 → 'low'
    • stock >= 100 → 'in-stock'
  • ✅ Volume pricing: Show lowest tier as hint ("Vanaf X: €Y (−Z%)")
  • ✅ Badge priority: If multiple badges, show first badge only (or implement badge stacking)

Payload CMS Hook: Auto Stock Status

// In /collections/Products.ts
{
  slug: 'products',
  hooks: {
    beforeChange: [
      ({ data }) => {
        // Auto-calculate stock status
        if (typeof data.stock === 'number') {
          if (data.stock === 0) {
            data.stockStatus = 'out'
          } else if (data.stock > 0 && data.stock < 100) {
            data.stockStatus = 'low'
          } else {
            data.stockStatus = 'in-stock'
          }
        }
        return data
      }
    ]
  },
  fields: [
    // ... fields
  ]
}
