📋 STEP 1: COMPONENT OVERVIEW

Component: C8 - Back-in-Stock Notification

Category: E-commerce / Inventory Management

Complexiteit: Low (email input, success state)

Purpose:

Email signup widget for out-of-stock products. Captures email addresses and sends notifications when product is back in stock. Reduces lost sales from temporary stockouts.

Key Features:

  • Coral "Tijdelijk uitverkocht" label with dot
  • Clear title and description
  • Email input with focus state
  • Submit button with bell icon
  • Green success confirmation
  • Form validation (email format)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • Product pages with stock: 0
  • Popular products that frequently sell out
  • Pre-launch products (coming soon)
  • Seasonal products (out of season)

Best Practices:

  • Replace add-to-cart button with this component
  • Send email notification within 24h of restock
  • Include direct link to product in notification email
  • Optional: Reserve stock for 24h for notified customers
  • Show estimated restock date if known
  • GDPR: Include privacy notice and unsubscribe option

💡 STEP 7: IMPLEMENTATION NOTES

// React Component
export function BackInStockNotification({ product, onSubmit }) {
  const [email, setEmail] = useState('');
  const [submitted, setSubmitted] = useState(false);
  const [error, setError] = useState('');

  const handleSubmit = async (e) => {
    e.preventDefault();
    if (!email.match(/^[^\s@]+@[^\s@]+\.[^\s@]+$/)) {
      setError('Vul een geldig e-mailadres in');
      return;
    }

    try {
      await onSubmit(email, product.id, product.variantId);
      setSubmitted(true);
      setError('');
    } catch (err) {
      setError('Er is iets misgegaan. Probeer het opnieuw.');
    }
  };

  return (
    <div className="bis-card">
      <div className="bis-header">
        <div className="bis-label"><span className="dot" /> Tijdelijk uitverkocht</div>
      </div>
      <div className="bis-title">Ontvang een melding wanneer dit product weer op voorraad is</div>
      <div className="bis-desc">
        Vul uw e-mailadres in en wij sturen u automatisch een bericht zodra dit product weer leverbaar is.
        {product.estimatedRestock && ` Verwachte leverdatum: ${product.estimatedRestock}`}
      </div>
      <form className="bis-form" onSubmit={handleSubmit}>
        <input
          className="bis-input"
          type="email"
          placeholder="Uw e-mailadres"
          value={email}
          onChange={(e) => setEmail(e.target.value)}
          required
        />
        <button className="bis-btn" type="submit">
          <Bell size={15} /> Meld mij
        </button>
      </form>
      {error && <div className="bis-error">{error}</div>}
      {submitted && (
        <div className="bis-success show">
          <CheckCircle size={16} /> U ontvangt een melding op het opgegeven adres.
        </div>
      )}
    </div>
  );
}
