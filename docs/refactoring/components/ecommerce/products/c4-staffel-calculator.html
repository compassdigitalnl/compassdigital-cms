📋 STEP 1: COMPONENT OVERVIEW

Component: C4 - Staffel Calculator

Category: E-commerce / Product

Complexiteit: High (live calculations, tier detection, smart hints)

Purpose:

Volume pricing calculator showing tiered pricing (staffelprijzen) with live updates. Encourages bulk purchases by showing savings and next-tier hints.

Key Features:

  • 4 tiered price levels (1-4, 5-9, 10-24, 25+)
  • Visual tier selection (radio-style with check icon)
  • Quantity stepper (− / + buttons)
  • Live total calculation with unit breakdown
  • Savings display (vs. base price)
  • Smart hint: "Bestel er X meer en betaal €Y" (next tier upsell)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • B2B e-commerce with volume discounts
  • Wholesale platforms
  • Bulk product sales

Best Practices:

  • Show savings as percentage AND absolute (€)
  • Always show next-tier hint (upsell opportunity)
  • Auto-select tier based on quantity
  • Make tiers clickable (quick jump to quantity)

💡 STEP 7: IMPLEMENTATION NOTES

// Tier Detection
function getTierForQty(qty, tiers) {
  return tiers.find(t => qty >= t.min && qty <= t.max);
}

// Savings Calculation
const tier = getTierForQty(qty, tiers);
const savings = (basePrice - tier.price) * qty;

// Next Tier Hint
const nextTier = tiers.find(t => t.min > qty);
const qtyNeeded = nextTier.min - qty;
