📋 STEP 1: COMPONENT OVERVIEW

Component: C9 - Compare Bar

Category: E-commerce / Product Discovery

Complexiteit: Medium (multi-product state, horizontal scroll, dynamic slots)

Purpose:

Bottom-fixed bar showing selected products for comparison. Users can add products from shop pages, view side-by-side specs, and make informed purchase decisions.

Key Features:

  • Navy background (dark theme)
  • Max 4 product slots (2 filled, 2 empty)
  • Product cards with thumbnail, name, remove button
  • Empty slots with dashed border
  • Compare button (teal) to view comparison table
  • Close button to hide bar
  • Horizontal scroll for overflow
  • Slide-up animation

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • Product categories with similar items (e.g., laptops, phones, medical supplies)
  • B2B catalogs with spec-heavy products
  • Shop pages with "Add to compare" checkbox on product cards

Best Practices:

  • Limit to 2-4 products (4 is optimal for desktop comparison)
  • Persist compare list in localStorage
  • Auto-open bar when first product added
  • Link to dedicated comparison page (/compare?ids=1,2,3)
  • Only allow same category products (enforce matching specs)
  • Show "Add to compare" on product cards (checkbox or icon button)

💡 STEP 7: IMPLEMENTATION NOTES

// React Component
export function CompareBar({ onCompare }) {
  const [products, setProducts] = useState([]);
  const maxProducts = 4;

  const addProduct = (product) => {
    if (products.length >= maxProducts) {
      toast.error('Maximaal 4 producten kunnen vergeleken worden');
      return;
    }
    setProducts([...products, product]);
    localStorage.setItem('compare_products', JSON.stringify([...products, product]));
  };

  const removeProduct = (productId) => {
    const updated = products.filter(p => p.id !== productId);
    setProducts(updated);
    localStorage.setItem('compare_products', JSON.stringify(updated));
  };

  return (
    <div className={`compare-bar ${products.length > 0 ? 'show' : ''}`}>
      <div className="compare-inner">
        <div className="compare-label">Vergelijken ({products.length}/{maxProducts})</div>
        <div className="compare-items">
          {products.map(p => (
            <div key={p.id} className="compare-item">
              <img src={p.image} className="ci-img" />
              <div className="ci-name">{p.name}</div>
              <button className="ci-remove" onClick={() => removeProduct(p.id)}>
                <X size={12} />
              </button>
            </div>
          ))}
          {Array.from({ length: maxProducts - products.length }).map((_, i) => (
            <div key={i} className="compare-slot-empty">+ product</div>
          ))}
        </div>
        <button className="compare-btn" onClick={() => onCompare(products.map(p => p.id))}>
          <Columns3 size={16} /> Vergelijken
        </button>
        <button className="compare-close" onClick={() => setProducts([])}>
          <X size={18} />
        </button>
      </div>
    </div>
  );
}
