📋 STEP 1: COMPONENT OVERVIEW

Component: C5 - QuickView Modal

Category: E-commerce / Product Preview

Complexiteit: High (modal overlay, product details, variant selection, add-to-cart)

Purpose:

Lightbox modal for quick product preview from shop/archive pages. Shows core product info, variant selection, and add-to-cart without leaving the current page. Improves conversion by reducing friction.

Key Features:

  • Full-screen overlay with backdrop blur
  • 2-column layout: image (left) + details (right)
  • Product image with badge overlay
  • Brand, name, SKU, stock indicator
  • Price with staffel pricing hint
  • Size/variant selector with visual feedback
  • Quantity stepper
  • Add-to-cart button
  • Link to full product page
  • ESC key to close, click backdrop to close
  • Scale-up animation (0.95 → 1.0)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • Product archive/shop pages with grid layout
  • Quick preview without page navigation
  • Simple products with limited variants (colors, sizes)
  • B2B catalogs with fast ordering workflow

Best Practices:

  • Trigger from "Quick view" button on product cards
  • Use only for products with simple variants (max 5-6 options)
  • For complex products (many options, long descriptions), link to full page
  • Auto-focus quantity input on open for keyboard users
  • Prevent body scroll when modal is open
  • Add to cart → show toast notification → optionally open minicart

export function QuickViewModal({ product, isOpen, onClose, onAddToCart }) {
  const [selectedVariant, setSelectedVariant] = useState(product.variants.find(v => v.default));
  const [quantity, setQuantity] = useState(1);

  useEffect(() => {
    if (isOpen) {
      document.body.style.overflow = 'hidden'; // Prevent body scroll
      document.getElementById('qv-qty-input')?.focus();
    } else {
      document.body.style.overflow = '';
    }
  }, [isOpen]);

  const handleAddToCart = () => {
    onAddToCart(product.id, selectedVariant.id, quantity);
    onClose();
    // Show toast notification
    // Optionally open minicart
  };

  return (
    <div className={`qv-overlay ${isOpen ? 'open' : ''}`}>
      <div className="qv-backdrop" onClick={onClose} />
      <div className="qv-modal" role="dialog" aria-modal="true">
        <button className="qv-close" onClick={onClose} aria-label="Sluit">
          <X size={18} />
        </button>
        <div className="qv-img">
          {product.badge && <div className="qv-badge">{product.badge}</div>}
          <img src={product.image} alt={product.name} />
        </div>
        <div className="qv-body">
          <div className="qv-brand">{product.brand}</div>
          <h2 className="qv-name">{product.name}</h2>
          <div className="qv-sku">Art. {product.sku}</div>
          {/* ... rest of content ... */}
        </div>
      </div>
    </div>
  );
}
