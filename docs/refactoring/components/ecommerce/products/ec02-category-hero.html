📋 STEP 1: Component Overview

Component ID: EC02

Component Name: CategoryHero

Category: E-commerce

Source File: /ecommerce-flows/ef1-shop-archive.html (lines 14-42)

Target Folder: /refactoring/components/ecommerce/

Description

Dark gradient hero section displayed at the top of category/shop archive pages. Features a category badge with icon, large title, description text, decorative teal glow overlay, and two stat counters (product count + brand count) on the right.

Key Features

  • ✅ Dark gradient background: Navy to navy-light (135deg)
  • ✅ Decorative teal glow: Radial gradient overlay (top-right, 8% opacity)
  • ✅ Category badge: Pill-style badge with icon + uppercase text
  • ✅ Large title: 36px Plus Jakarta Sans 800 weight
  • ✅ Description: 16px white text at 50% opacity, max-width 520px
  • ✅ Stats display: Product count + brand count with teal accent numbers
  • ✅ Responsive: Stacks vertically on mobile

🎯 STEP 2: When to Use

Use this component when:

  • ✅ Top of shop archive / category listing pages
  • ✅ Collection pages (e.g., "New Arrivals", "Best Sellers")
  • ✅ Brand landing pages (with brand name as title)
  • ✅ Search results pages (with query as title)
  • ✅ Product type pages (e.g., "Variable Products", "Bundles")

Design Patterns:

  • Badge icon: Use relevant Lucide icon (package, heart-pulse, wrench, etc.)
  • Badge text: Category type in uppercase (e.g., "PRODUCTCATEGORIE", "MEDISCH MATERIAAL")
  • Title: Category/collection name
  • Description: 1-2 sentence description of category contents
  • Stats: Always show product count, optionally show brand count

Don't use this component for:

  • ❌ Product detail pages (use product-specific hero)
  • ❌ Static content pages (use standard page hero)
  • ❌ Homepage (use dedicated homepage hero)

Auto-Calculate Product/Brand Counts

// In /collections/Categories.ts
{
  slug: 'categories',
  hooks: {
    afterRead: [
      async ({ doc, req }) => {
        // Count products in this category
        const productCount = await req.payload.count({
          collection: 'products',
          where: {
            category: {
              equals: doc.id
            }
          }
        })

        // Count unique brands
        const products = await req.payload.find({
          collection: 'products',
          where: {
            category: {
              equals: doc.id
            }
          },
          limit: 1000
        })

        const uniqueBrands = new Set(
          products.docs.map(p => p.brand?.id).filter(Boolean)
        )

        return {
          ...doc,
          productCount: productCount.totalDocs,
          brandCount: uniqueBrands.size
        }
      }
    ]
  },
  fields: [
    // ... fields
  ]
}

📐 STEP 5: Layout Rules

Desktop Layout (≥768px)

  • Container: 40px padding, 16px border-radius
  • Content layout: Flexbox row with justify-content: space-between
  • Text section: flex: 1 to take available space
  • Stats section: Fixed width (auto), aligned right

Mobile Layout (<768px)

  • Container: 32px 24px padding
  • Content layout: Flexbox column with align-items: flex-start
  • Title: Reduced to 28px font-size
  • Stats: Reduced gap to 24px
  • Stat values: Reduced to 24px font-size

Glow Overlay Positioning

  • Position: Absolute, top: -50%, right: -10%
  • Size: 400px × 400px
  • Background: radial-gradient(circle, rgba(0,137,123,0.08) 0%, transparent 70%)
  • Border-radius: 50% (perfect circle)
  • Pointer-events: none (doesn't block interactions)

Z-index Layering

  • Background gradient: Base layer (z-index: auto)
  • Glow overlay: ::before pseudo-element (no z-index needed)
  • Content: position: relative; z-index: 1 (above glow)

💻 STEP 7: Implementation Notes

React/Next.js Component Example

'use client'

import { LucideIcon } from 'lucide-react'
import * as Icons from 'lucide-react'

interface CategoryHeroProps {
  category: {
    name: string
    description?: string
    icon?: string
    badgeText?: string
  }
  productCount: number
  brandCount?: number
}

export function CategoryHero({
  category,
  productCount,
  brandCount
}: CategoryHeroProps) {
  // Dynamically get Lucide icon
  const IconComponent = category.icon
    ? (Icons[category.icon as keyof typeof Icons] as LucideIcon)
    : Icons.Package

  return (
    <section className="category-hero" role="banner" aria-labelledby="category-title">
      <div className="category-hero__content">
        <div className="category-hero__text">
          <div className="category-hero__badge" aria-hidden="true">
            {IconComponent && <IconComponent size={14} />}
            {category.badgeText || 'PRODUCTCATEGORIE'}
          </div>
          <h1 id="category-title" className="category-hero__title">
            {category.name}
          </h1>
          {category.description && (
            <p className="category-hero__description">
              {category.description}
            </p>
          )}
        </div>
        <div className="category-hero__stats" aria-label="Categorie statistieken">
          <div className="category-hero__stat">
            <div
              className="category-hero__stat-value"
              aria-label={`${productCount} producten`}
            >
              <span>{productCount}</span>
            </div>
            <div className="category-hero__stat-label" aria-hidden="true">
              Producten
            </div>
          </div>
          {brandCount !== undefined && (
            <div className="category-hero__stat">
              <div
                className="category-hero__stat-value"
                aria-label={`${brandCount} merken`}
              >
                <span>{brandCount}</span>
              </div>
              <div className="category-hero__stat-label" aria-hidden="true">
                Merken
              </div>
            </div>
          )}
        </div>
      </div>
    </section>
  )
}

Server Component Usage (Next.js App Router)

// app/shop/[category]/page.tsx
import { CategoryHero } from '@/components/ecommerce/CategoryHero'
import { getPayloadClient } from '@/lib/payload'

export default async function CategoryPage({
  params
}: {
  params: { category: string }
}) {
  const payload = await getPayloadClient()

  // Fetch category with auto-calculated counts
  const category = await payload.findByID({
    collection: 'categories',
    id: params.category
  })

  return (
    <>
      <CategoryHero
        category={{
          name: category.name,
          description: category.description,
          icon: category.icon,
          badgeText: category.badgeText
        }}
        productCount={category.productCount}
        brandCount={category.brandCount}
      />
      {/* Rest of page: filters, products, etc. */}
    </>
  )
}

Important Implementation Details

  • ✅ Dynamic icon loading: Use Lucide React's dynamic import to render icon by name
  • ✅ Fallback icon: Default to Package if no icon specified
  • ✅ Glow overlay: Use CSS ::before pseudo-element (no extra div needed)
  • ✅ Auto-counts: Calculate product/brand counts in Payload hooks, not on frontend
  • ✅ Optional brand count: Hide second stat if brandCount not provided
