📋 STEP 1: COMPONENT OVERVIEW

Component: C6 - Sticky Add-to-Cart Bar

Category: E-commerce / Conversion Optimization

Complexiteit: Medium (scroll detection, sticky positioning, responsive layout)

Purpose:

Bottom-fixed bar that appears when the main add-to-cart button scrolls out of view on product pages. Keeps CTA accessible at all times, improving conversion by reducing scroll-back friction.

Key Features:

  • Fixed bottom positioning (z-index 300)
  • Slides up when main CTA out of view, slides down when in view
  • Product thumbnail + name + meta info
  • Current price (large, bold)
  • Size/variant selector (compact)
  • Add-to-cart button with icon
  • Responsive: hides meta and sizes on mobile
  • Smooth slide animation (0.35s cubic-bezier)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • Product detail pages with long content (reviews, specs, related products)
  • Mobile product pages (small screens)
  • B2B catalogs with quick reordering workflow

Best Practices:

  • Trigger when main CTA scrolls past viewport top + 100px buffer
  • Sync selected variant with main product form
  • Hide on checkout/cart pages to avoid confusion
  • Use IntersectionObserver for performant scroll detection
  • Ensure enough contrast with page background (white bar + shadow)

💡 STEP 7: IMPLEMENTATION NOTES

// React Component
export function StickyBar({ product, mainCTARef }) {
  const [isVisible, setIsVisible] = useState(false);
  const [selectedVariant, setSelectedVariant] = useState(product.variants.find(v => v.default));

  useEffect(() => {
    const observer = new IntersectionObserver(
      ([entry]) => setIsVisible(!entry.isIntersecting),
      { threshold: 0, rootMargin: '-100px 0px 0px 0px' }
    );

    if (mainCTARef.current) {
      observer.observe(mainCTARef.current);
    }

    return () => observer.disconnect();
  }, [mainCTARef]);

  return (
    <div className={`sticky-bar ${isVisible ? 'show' : ''}`}>
      <div className="sticky-inner">
        <div className="sb-product">
          <img src={product.image} className="sb-img" />
          <div className="sb-info">
            <div className="sb-name">{product.name}</div>
            <div className="sb-meta">{product.brand} · {product.meta}</div>
          </div>
        </div>
        <div className="sb-price">€{product.price.toFixed(2)}</div>
        <div className="sb-sizes">
          {product.variants.map(v => (
            <div
              key={v.id}
              className={`sb-size ${selectedVariant.id === v.id ? 'active' : ''}`}
              onClick={() => setSelectedVariant(v)}
            >
              {v.name}
            </div>
          ))}
        </div>
        <button className="sb-add" onClick={() => onAddToCart(product.id, selectedVariant.id)}>
          <ShoppingCart size={17} /> Bestellen
        </button>
      </div>
    </div>
  );
}
