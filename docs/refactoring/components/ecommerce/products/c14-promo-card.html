📋 STEP 1: COMPONENT OVERVIEW

Component: C14 - Promo Card

Parent: C14-MegaNavigation (L3 Flyout Column) / Standalone

Category: Navigation / Promotional

Complexiteit: Low (simple card layout)

Purpose:

Highlight featured/bestselling products in mega menu flyouts, sidebars, or promotional sections. Dark gradient background with product image, badge, name, and price.

Variants:

  • Default: 64px image, 14px name, 16px price (mega menu L3 column)
  • Compact: 48px image, 12px name, 14px price (sidebars, narrow spaces)
  • Large: 80px image, 16px name, 18px price (hero sections, landing pages)

Visual Features:

  • Dark gradient background (navy → navy-mid 135deg)
  • Decorative teal glow (radial gradient, top-right corner)
  • Product image with semi-transparent background
  • Status badge (Bestseller, Nieuw, Populair, Actie)
  • Optional old price (strikethrough)
  • Hover effect (translateY -2px + shadow)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • Mega Menu Flyouts: L3 column bottom promo spot (most common)
  • Sidebar Widgets: Featured product in category sidebars
  • Dashboard Cards: Admin/account dashboard recommendations
  • Cross-Sell: "Related products" or "Customers also bought"
  • Campaign Pages: Hero section product highlight

Niet geschikt voor:

  • Product archives/grids (use standard ProductCard instead)
  • Cart line items (different purpose/layout)
  • Content-heavy pages where dark cards would clash

Common Placements:

  • Mega Nav L3: Bottom of third column (auto margin-top)
  • Category Sidebars: "Featured in this category"
  • Search Results: Sponsored/featured product slot
  • Empty States: Suggested product when filters return no results

💡 STEP 7: IMPLEMENTATION NOTES

Dynamic Badge Selection:

// Auto-select badge based on product metadata
function getBadge(product) {
  if (product.isNew && (Date.now() - product.createdAt) < 30 * 24 * 60 * 60 * 1000) {
    return { icon: 'zap', label: 'Nieuw' };
  }
  if (product.sales > 100) {
    return { icon: 'trending-up', label: 'Bestseller' };
  }
  if (product.discount > 15) {
    return { icon: 'tag', label: 'Actie' };
  }
  if (product.rating >= 4.5) {
    return { icon: 'sparkles', label: 'Populair' };
  }
  return null;  // No badge
}

Price Formatting:

// Format price with discount calculation
function formatPrice(price, oldPrice) {
  const formattedPrice = `€${price.toFixed(2).replace('.', ',')}`;

  if (oldPrice && oldPrice > price) {
    const discount = Math.round(((oldPrice - price) / oldPrice) * 100);
    const formattedOld = `€${oldPrice.toFixed(2).replace('.', ',')}`;

    return `
      ${formattedPrice}
      <span class="old" title="${discount}% korting">${formattedOld}</span>
    `;
  }

  return formattedPrice;
}

Analytics Tracking:

// Track promo card impressions and clicks
document.querySelectorAll('.promo-card').forEach(card => {
  // Impression tracking (when visible)
  const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        trackEvent('promo_impression', {
          product_id: card.dataset.productId,
          location: card.dataset.location,  // 'mega-menu', 'sidebar', etc.
        });
      }
    });
  });
  observer.observe(card);

  // Click tracking
  card.addEventListener('click', (e) => {
    trackEvent('promo_click', {
      product_id: card.dataset.productId,
      location: card.dataset.location,
    });
  });
});
