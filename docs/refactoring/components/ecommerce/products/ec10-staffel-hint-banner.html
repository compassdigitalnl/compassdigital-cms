Description: Amber notification banner displayed below cart line items to inform customers how many more units they need to order to qualify for volume pricing (staffel) discounts.

Key Features:

  • Amber background (#FFF8E1) with darker text (#7B6B00)
  • Trending-down icon (16px amber)
  • Smart messaging: "Bestel nog X stuks voor..."
  • Discount amount and percentage displayed
  • Success variant (green) when discount achieved
  • Attached below EC06: CartLineItem
  • Conditional display (only when close to next tier)

🎯 STEP 2: When to Use

Use this component when:

  • Product has volume pricing (staffel) tiers
  • Current quantity is close to next tier (within 50 units or 2x multiplier)
  • Attached below cart line item in cart page
  • In cart summary to show total potential savings

Common Design Patterns:

  • Cart Line Item: Attached below item, full-width banner
  • Product Page: Near quantity selector (optional)
  • Mini Cart: Compact version below item

Do NOT use when:

  • Product has no volume pricing
  • Current quantity already qualifies for best tier
  • Next tier is too far away (e.g., need 500 more units)
  • On checkout page (too late to modify cart)

Smart Display Logic

// Only show banner if customer is close to next tier
function shouldShowStaffelHint(
  currentQty: number,
  nextTierQty: number
): boolean {
  const gap = nextTierQty - currentQty

  // Show if within 50 units
  if (gap <= 50) return true

  // Or within 2x multiplier
  if (nextTierQty / currentQty <= 2) return true

  return false
}

{
  name: 'volumePricing',
  type: 'array',
  label: 'Volume Pricing (Staffel)',
  fields: [
    {
      name: 'minQuantity',
      type: 'number',
      required: true,
      admin: {
        description: 'Minimum quantity for this tier'
      }
    },
    {
      name: 'price',
      type: 'number',
      required: true,
      admin: {
        description: 'Price per unit at this tier'
      }
    },
    {
      name: 'discountPercent',
      type: 'number',
      admin: {
        description: 'Calculated discount % from base price (auto-calculated)',
        readOnly: true
      }
    }
  ],
  admin: {
    initCollapsed: false,
    components: {
      RowLabel: ({ data }) =>
        data?.minQuantity
          ? `${data.minQuantity}+ stuks: € ${data.price}`
          : 'Volume Tier'
    }
  },
  hooks: {
    beforeChange: [
      ({ data, siblingData }) => {
        // Auto-calculate discount percent
        const basePrice = siblingData.price
        if (data.price && basePrice) {
          data.discountPercent = Math.round(
            ((basePrice - data.price) / basePrice) * 100
          )
        }
        return data
      }
    ]
  }
}

💻 STEP 7: Implementation Notes

React/Next.js Component

'use client'

import { TrendingDown, CheckCircle } from 'lucide-react'
import { useMemo } from 'react'

export function StaffelHintBanner({
  currentQuantity,
  currentPrice,
  nextTierQuantity,
  nextTierPrice,
  nextTierDiscountPercent,
  variant = 'default',
  className = ''
}: StaffelHintBannerProps) {

  const { quantityGap, isAchieved } = useMemo(() => {
    const gap = nextTierQuantity - currentQuantity
    const achieved = currentQuantity >= nextTierQuantity

    return { quantityGap: gap, isAchieved: achieved }
  }, [currentQuantity, nextTierQuantity])

  // Don't show if too far away
  if (!isAchieved && quantityGap > 50 && nextTierQuantity / currentQuantity > 2) {
    return null
  }

  const Icon = isAchieved || variant === 'success' ? CheckCircle : TrendingDown
  const variantClass = isAchieved || variant === 'success'
    ? 'staffel-hint--success'
    : variant === 'compact'
    ? 'staffel-hint--compact'
    : ''

  return (
    <div
      className={`staffel-hint ${variantClass} ${className}`}
      role="status"
      aria-live="polite"
    >
      <Icon aria-hidden="true" />
      <span>
        {isAchieved || variant === 'success' ? (
          <>
            <strong>Staffelkorting actief!</strong> Je betaalt €{' '}
            {nextTierPrice.toFixed(2)} per stuk (−{nextTierDiscountPercent}%)
          </>
        ) : variant === 'compact' ? (
          <>
            +{quantityGap} stuks = −{nextTierDiscountPercent}% korting (€{' '}
            {nextTierPrice.toFixed(2)}/st)
          </>
        ) : (
          <>
            Bestel nog <strong>{quantityGap} stuks</strong> voor
            staffelkorting: € {nextTierPrice.toFixed(2)} per stuk (−
            {nextTierDiscountPercent}%)
          </>
        )}
      </span>
    </div>
  )
}

Integration with CartLineItem

// In CartLineItem component
export function CartLineItem({ product, quantity, ... }) {
  // Find next volume pricing tier
  const nextTier = product.volumePricing
    ?.filter(tier => tier.minQuantity > quantity)
    .sort((a, b) => a.minQuantity - b.minQuantity)[0]

  return (
    <div className="cart-line-item-wrapper">
      <div className="cart-line-item__main">
        {/* Product info */}
      </div>

      {nextTier && (
        <StaffelHintBanner
          currentQuantity={quantity}
          currentPrice={product.price}
          nextTierQuantity={nextTier.minQuantity}
          nextTierPrice={nextTier.price}
          nextTierDiscountPercent={nextTier.discountPercent}
        />
      )}
    </div>
  )
}

Important Implementation Details

  • Smart Display: Only show if customer is close to next tier (within 50 units or 2x multiplier)
  • Dynamic Calculation: Calculate gap in real-time as quantity changes
  • Success State: Auto-switch to green success banner when tier is reached
  • Formatting: Use .toFixed(2) for prices, show discount percentage
  • Multiple Tiers: Optionally show next 2 tiers if customer is very close
