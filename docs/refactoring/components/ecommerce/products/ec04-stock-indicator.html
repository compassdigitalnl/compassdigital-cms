📋 STEP 1: Component Overview

Component ID: EC04

Component Name: StockIndicator

Category: E-commerce

Source File: /ecommerce-flows/ef1-shop-archive.html (ProductCard component)

Target Folder: /refactoring/components/ecommerce/

Description

Small inline component displaying product stock status with color-coded dot and text. Shows 3 states: in-stock (green), low stock (amber), or out-of-stock (coral). Used in product cards, product detail pages, and cart items.

Key Features

  • ✅ 3 states: in-stock, low, out (color-coded)
  • ✅ Colored dot: 6px circle matching text color
  • ✅ Inline-flex layout: Works inline with other content
  • ✅ 3 sizes: Small (11px), default (12px), large (14px)
  • ✅ Accessibility: role="status" + aria-live="polite"
  • ✅ Lightweight: No dependencies, pure CSS

🎯 STEP 2: When to Use

Use this component when:

  • ✅ Product cards (bottom section)
  • ✅ Product detail pages (near add-to-cart)
  • ✅ Cart line items (below product info)
  • ✅ Quick order tables (stock column)
  • ✅ Wishlist items
  • ✅ Recently viewed products
  • ✅ Admin inventory management

Stock Status Logic:

  • In Stock (Green): stock >= 100 or configurable threshold
  • Low Stock (Amber): stock > 0 && stock < 100
  • Out of Stock (Coral): stock === 0

Text Patterns:

  • In Stock: "Op voorraad (X stuks)" — shows exact quantity
  • Low: "Laag op voorraad (X stuks)" — shows exact quantity
  • Out: "Tijdelijk uitverkocht" — no quantity shown
  • Pre-order (Future): "Pre-order (verwacht DD-MM-YYYY)"

Don't use this component for:

  • ❌ Generic status messages (use dedicated status badge)
  • ❌ Shipping status (use shipping tracker)
  • ❌ Payment status (use payment badge)

Auto-Calculate Status Hook

// In Payload Products Collection
{
  slug: 'products',
  hooks: {
    beforeChange: [
      ({ data }) => {
        // Auto-calculate stock status based on quantity
        const stock = data.stock || 0

        if (stock === 0) {
          data.stockStatus = 'out'
        } else if (stock > 0 && stock < 100) {
          data.stockStatus = 'low'
        } else {
          data.stockStatus = 'in-stock'
        }

        return data
      }
    ]
  },
  fields: [
    { name: 'stock', type: 'number', required: true },
    {
      name: 'stockStatus',
      type: 'select',
      options: ['in-stock', 'low', 'out'],
      admin: { readOnly: true }
    }
  ]
}

Configurable Threshold

// In environment config
const STOCK_THRESHOLDS = {
  low: process.env.LOW_STOCK_THRESHOLD || 100,
  out: 0
}

// In hook
if (stock === STOCK_THRESHOLDS.out) {
  data.stockStatus = 'out'
} else if (stock > 0 && stock < STOCK_THRESHOLDS.low) {
  data.stockStatus = 'low'
} else {
  data.stockStatus = 'in-stock'
}

  • ✅ Color alone not used to convey meaning (text also present)

  • ✅ Amber text on white: 3.8:1 ratio (large text OK)
  • ✅ Coral text on white: 4.2:1 ratio ✅ WCAG AA

💻 STEP 7: Implementation Notes

React/Next.js Component

type StockStatus = 'in-stock' | 'low' | 'out'
type StockSize = 'small' | 'default' | 'large'

interface StockIndicatorProps {
  status: StockStatus
  quantity?: number
  size?: StockSize
  customText?: string
}

export function StockIndicator({
  status,
  quantity,
  size = 'default',
  customText
}: StockIndicatorProps) {
  const getText = () => {
    if (customText) return customText

    switch (status) {
      case 'in-stock':
        return quantity
          ? `Op voorraad (${quantity.toLocaleString('nl-NL')} stuks)`
          : 'Op voorraad'
      case 'low':
        return quantity
          ? `Laag op voorraad (${quantity.toLocaleString('nl-NL')} stuks)`
          : 'Laag op voorraad'
      case 'out':
        return 'Tijdelijk uitverkocht'
      default:
        return ''
    }
  }

  const getAriaLabel = () => {
    if (customText) return customText

    switch (status) {
      case 'in-stock':
        return quantity
          ? `Op voorraad, ${quantity} stuks beschikbaar`
          : 'Op voorraad'
      case 'low':
        return quantity
          ? `Laag op voorraad, nog ${quantity} stuks beschikbaar`
          : 'Laag op voorraad'
      case 'out':
        return 'Tijdelijk uitverkocht'
      default:
        return ''
    }
  }

  return (
    <div
      className={`stock-indicator stock-indicator--${status} ${
        size !== 'default' ? `stock-indicator--${size}` : ''
      }`}
      role="status"
      aria-live="polite"
      aria-label={getAriaLabel()}
    >
      <div className="stock-indicator__dot" aria-hidden="true" />
      {getText()}
    </div>
  )
}

Important Implementation Details

  • ✅ Number formatting: Use toLocaleString('nl-NL') for thousands separator (2.400 not 2400)
