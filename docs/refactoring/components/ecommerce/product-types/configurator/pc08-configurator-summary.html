📋 STEP 1: Component Overview

PC08: ConfiguratorSummary is a summary panel that displays all selected product configuration options with a pricing breakdown and total. Updates in real-time as users make variant selections. Essential for complex configurable products to provide transparency and build trust.

Key Features

  • ✅ Live Updates: Real-time price recalculation as users select options
  • ✅ Clear Breakdown: Line-item display (base price + all modifications)
  • ✅ Price Indicators: Green "Incl." for included, navy "+ €X" for surcharges, green "− €X" for discounts
  • ✅ Bold Total: Large, prominent total price (20px, 800 weight)
  • ✅ Optional Elements: Edit buttons, savings badges, product thumbnails
  • ✅ Sticky Positioning: Can stick to viewport (sidebar placement)
  • ✅ Compact Variant: Smaller padding/fonts for mobile or tight spaces
  • ✅ Responsive: Adjusts font sizes and padding on mobile

Use Cases

  • 🛒 Complex configurable products (custom sneakers, laptops, furniture)
  • 🎨 Build-your-own products (bundles, kits, packages)
  • 💻 Product configurators (car, computer, home builder)
  • 📦 Subscription tiers with add-ons
  • 🎁 Gift builders with personalization options
  • ⚙️ Service packages with optional extras

🎯 STEP 2: When to Use This Component

✅ Use ConfigurationSummary When:

  • Multiple variant options — Products have 3+ configurable attributes (color, size, extras)
  • Price varies significantly — Add-ons or options change total price by 10%+
  • Complex configurations — Users need clarity on what they've selected
  • Trust is critical — Show full pricing transparency to reduce checkout abandonment
  • Sticky sidebar exists — Use sticky positioning for constant visibility
  • Mobile checkout — Use compact variant in responsive layouts

❌ Don't Use When:

  • Simple products — 1-2 options (color + size) don't need full summary
  • Fixed price — All options are included (show single price instead)
  • Space constrained — Very limited vertical space (use inline price instead)
  • Pre-configurated — Products have no customization options

⚙️ STEP 4: Props & Configuration

TypeScript Interface

interface ConfigurationRow { id: string; label: string; value: string; // Price formatted as "€ 12,50" type: 'base' | 'included' | 'surcharge' | 'discount'; image?: string; // Optional thumbnail (emoji or URL) editable?: boolean; // Show edit button onEdit?: () => void; } interface ConfigurationSummaryProps { title: string; titleIcon?: string; // Lucide icon name rows: ConfigurationRow[]; total: number; savings?: number; // Optional savings amount compact?: boolean; // Use compact variant (default: false) sticky?: boolean; // Sticky positioning (default: false) stickyOffset?: number; // Top offset when sticky (default: 90px) showImages?: boolean; // Show product thumbnails (default: false) }

Example Usage

const configRows: ConfigurationRow[] = [ { id: 'base', label: 'Basisprijs', value: '€ 159,95', type: 'base' }, { id: 'color', label: 'Kleur: Midnight Black', value: 'Incl.', type: 'included', editable: true, onEdit: () => scrollToVariantSection('color') }, { id: 'coating', label: 'Beschermende coating', value: '+ € 12,50', type: 'surcharge' } ]; <ConfigurationSummary title="Jouw configuratie" titleIcon="clipboard-list" rows={configRows} total={172.45} sticky={true} />

Auto-Calculation Logic

// Calculate total from selected variants function calculateTotal( basePrice: number, selectedVariants: Record, variantDefinitions: VariantDefinition[] ): number { let total = basePrice; Object.entries(selectedVariants).forEach(([variantId, optionId]) => { const variant = variantDefinitions.find(v => v.id === variantId); const option = variant?.options.find(o => o.id === optionId); if (option?.price) { total += option.price; } }); return total; } // Generate configuration rows function generateConfigRows( basePrice: number, selectedVariants: Record, variantDefinitions: VariantDefinition[] ): ConfigurationRow[] { const rows: ConfigurationRow[] = [ { id: 'base', label: 'Basisprijs', value: formatPrice(basePrice), type: 'base' } ]; Object.entries(selectedVariants).forEach(([variantId, optionId]) => { const variant = variantDefinitions.find(v => v.id === variantId); const option = variant?.options.find(o => o.id === optionId); if (variant && option) { rows.push({ id: variantId, label: `${variant.label}: ${option.name}`, value: option.price === 0 ? 'Incl.' : `+ ${formatPrice(option.price)}`, type: option.price === 0 ? 'included' : 'surcharge', editable: true, onEdit: () => scrollToVariantSection(variantId) }); } }); return rows; }

Default Values

  • Title: "Jouw configuratie"
  • Title Icon: "clipboard-list"
  • Compact: false (use full size)
  • Sticky: false (no sticky positioning)
  • Sticky Offset: 90px (accounts for fixed header)
  • Show Images: false (text-only rows)

Performance Considerations

  • Memoization: Use React.memo on ConfigurationSummary to prevent re-renders
  • Debouncing: Debounce price calculations if they're expensive (large product catalogs)
  • Sticky performance: Use position: sticky (CSS) instead of JS scroll listeners
