Component Documentation

STEP 1: Component Overview

VP11: VariantToolbar is a toolbar component that provides view controls and bulk actions for multi-variant selection interfaces. It sits above the variant grid/list and allows users to switch between layouts and perform bulk operations.

Key Features:

  • Variant count display (left side)
  • Grid/List view toggle (pill-style 2-button group)
  • Bulk select all button
  • Bulk deselect all button
  • Responsive: stacks vertically on mobile
  • Active state highlighting (teal bg for active view)
  • Disabled states for bulk buttons when not applicable

File Size: ~630 lines

Component ID: VP11

Related: VP08 (grid card), VP09 (list row), VP10 (sidebar), VP12 (grid container), VP13 (list container)

STEP 2: When to Use

✅ Use VP11 (Toolbar) When:

  • Displaying 10+ variants that benefit from view switching
  • Multi-variant selection (VP08/VP09) needs bulk actions
  • User should be able to toggle between grid and list layouts
  • Need "Select all" / "Deselect all" shortcuts
  • Archive/shop pages with many variant options
  • B2B context where bulk selection is common

❌ Don't Use VP11 When:

  • Fewer than 6 variants (toolbar overhead not worth it)
  • Single-variant selection (VP01-VP05) — no bulk actions needed
  • Fixed view mode (no grid/list toggle needed)
  • Mobile-only experience (toolbar takes too much vertical space)

Configuration Options

  • Variant count: Dynamic number display (24, 12, etc.)
  • View toggle visibility: Show/hide based on layout availability
  • Bulk buttons state: Enable "Select all" when <100% selected, "Deselect all" when >0% selected

  • Selection count: Optional "X geselecteerd" indicator in count area

STEP 7: Implementation Notes

JavaScript Behavior

// View toggle
document.querySelectorAll('.view-btn').forEach(btn => {
  btn.addEventListener('click', () => {
    const view = btn.dataset.view;

    // Update active state
    btn.parentElement.querySelectorAll('.view-btn').forEach(b =>
      b.classList.remove('active')
    );
    btn.classList.add('active');

    // Toggle grid/list visibility
    if (view === 'grid') {
      document.getElementById('gridView').classList.remove('hidden');
      document.getElementById('listView').classList.add('hidden');
    } else {
      document.getElementById('gridView').classList.add('hidden');
      document.getElementById('listView').classList.remove('hidden');
    }

    // Save preference to localStorage
    localStorage.setItem('variantView', view);
  });
});

// Select all
document.getElementById('selectAllBtn').addEventListener('click', () => {
  document.querySelectorAll('.variant-card, .variant-row').forEach(card => {
    const checkbox = card.querySelector('.variant-checkbox-compact');
    const qtyInput = card.querySelector('.qty-input-inline, .qty-input-compact');

    checkbox.classList.add('checked');
    checkbox.setAttribute('aria-checked', 'true');
    card.classList.add('selected');

    if (parseInt(qtyInput.value) === 0) {
      qtyInput.value = 1;
    }
  });

  updateBulkButtons();
  lucide.createIcons();
});

// Deselect all
document.getElementById('deselectAllBtn').addEventListener('click', () => {
  document.querySelectorAll('.variant-card, .variant-row').forEach(card => {
    const checkbox = card.querySelector('.variant-checkbox-compact');
    const qtyInput = card.querySelector('.qty-input-inline, .qty-input-compact');

    checkbox.classList.remove('checked');
    checkbox.setAttribute('aria-checked', 'false');
    card.classList.remove('selected');
    qtyInput.value = 0;
  });

  updateBulkButtons();
  lucide.createIcons();
});

// Update bulk button states
function updateBulkButtons() {
  const allCards = document.querySelectorAll('.variant-card, .variant-row');
  const selectedCards = document.querySelectorAll('.variant-card.selected, .variant-row.selected');

  const selectAllBtn = document.getElementById('selectAllBtn');
  const deselectAllBtn = document.getElementById('deselectAllBtn');

  // Disable "Select all" if all selected
  selectAllBtn.disabled = selectedCards.length === allCards.length;

  // Disable "Deselect all" if none selected
  deselectAllBtn.disabled = selectedCards.length === 0;
}

React/Next.js Example

import { Grid2X2, List, CheckSquare, Square } from 'lucide-react';

interface VariantToolbarProps {
  variantCount: number;
  selectedCount: number;
  currentView: 'grid' | 'list';
  onViewChange: (view: 'grid' | 'list') => void;
  onSelectAll: () => void;
  onDeselectAll: () => void;
  showSelectionCount?: boolean;
}

export function VariantToolbar({
  variantCount,
  selectedCount,
  currentView,
  onViewChange,
  onSelectAll,
  onDeselectAll,
  showSelectionCount = false
}: VariantToolbarProps) {
  return (
    <div className="variant-toolbar">
      <div className="toolbar-count">
        <span className="count-number">{variantCount}</span> varianten beschikbaar
        {showSelectionCount && selectedCount > 0 && (
          <span style={{ color: 'var(--teal)', marginLeft: '8px' }}>
            • {selectedCount} geselecteerd
          </span>
        )}
      </div>

      <div className="toolbar-actions">
        <div className="view-toggle">
          <button
            className={`view-btn ${currentView === 'grid' ? 'active' : ''}`}
            onClick={() => onViewChange('grid')}
            data-view="grid"
          >
            <Grid2X2 className="w-3.5 h-3.5" />
            Grid
          </button>
          <button
            className={`view-btn ${currentView === 'list' ? 'active' : ''}`}
            onClick={() => onViewChange('list')}
            data-view="list"
          >
            <List className="w-3.5 h-3.5" />
            Lijst
          </button>
        </div>

        <button
          className="bulk-btn"
          onClick={onSelectAll}
          disabled={selectedCount === variantCount}
        >
          <CheckSquare className="w-3 h-3" />
          Selecteer alle
        </button>

        <button
          className="bulk-btn"
          onClick={onDeselectAll}
          disabled={selectedCount === 0}
        >
          <Square className="w-3 h-3" />
          Deselecteer alle
        </button>
      </div>
    </div>
  );
}
