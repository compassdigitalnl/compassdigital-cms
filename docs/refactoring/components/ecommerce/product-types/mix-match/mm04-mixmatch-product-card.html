📦 MM04: MixMatchProductCard — Component Documentation

STEP 1: Component Overview

Component: MixMatchProductCard

Purpose: Selectable product card for Mix & Match boxes with visual selection states, badges, quantity controls, and nutritional info.

Key Features:

  • Visual selection state with teal border + glow + checkmark
  • Product badges (Popular, New, Vegan, Spicy)
  • Gradient background with emoji/image
  • Nutritional meta (calories, dietary info)
  • Price display (€X,XX or "✓ In box" when selected)
  • Add button → Quantity stepper (when selected)
  • Hover effects (border change, lift)

STEP 2: When to Use

Use when:

  • Mix & Match product pages
  • Build-your-own box interfaces
  • Multi-select product grids

Do NOT use when:

  • Shop archive/category pages (use EC01 ProductCard)
  • Single product pages (use full product layout)
  • Cart items (use EC06 CartLineItem)

STEP 3: Design Tokens

Colors:

  • Card bg: #FAFBFC (white)
  • Border (default): 1.5px #E8ECF1 (grey)
  • Border (hover): #00897B (teal)
  • Border (selected): #00897B + box-shadow 0 0 0 2px rgba(0,137,123,0.12)
  • Image bg: #F1F4F8 (grey-light) or gradient
  • Checkmark: #00897B (teal) circle, white icon
  • Tags: Amber/Teal/Green/Coral backgrounds
  • Price: JetBrains Mono, #0A1628 (navy)
  • Price (selected): DM Sans, #00C853 (green) "✓ In box"
  • Add button: #00897B (teal), hover #0A1628 (navy)

Typography:

  • Product name: 13px, 700 weight, DM Sans
  • Meta text: 10px, 400 weight, grey-mid
  • Price: 14px, 700 weight, JetBrains Mono
  • Price (selected): 12px, 700 weight, DM Sans
  • Tag: 10px, 700 weight
  • Button: 11px, 800 weight

Spacing:

  • Body padding: 10px 12px
  • Name margin-bottom: 2px
  • Meta margin-bottom: 6px, gap: 6px
  • Image height: 130px
  • Border radius: 16px (card), 8px (buttons/stepper), 4px (tags)
  • Checkmark: 26×26px circle, top: 6px, right: 6px
  • Tag: top: 6px, left: 6px, padding: 2px 8px

STEP 4: Props & Configuration

interface MixMatchProductCardProps {
  product: {
    id: string;
    name: string;
    price: number;
    image?: string;
    emoji?: string;
    gradient?: string;
    calories?: number;
    badges?: Array<'popular' | 'new' | 'vegan' | 'spicy'>;
    meta?: Array<{ icon: string; text: string }>;
  };
  isSelected: boolean;
  quantity?: number;
  onToggle: () => void;
  onQuantityChange?: (qty: number) => void;
  isDisabled?: boolean;
  maxReached?: boolean;
}

STEP 5-8: Summary

Layout: Vertical card with image (130px) + body (10/12px padding) + footer (space-between flex)

States:

  • Default: Grey border, price shown, "Toevoegen" button
  • Hover: Teal border, translateY(-2px), shadow
  • Selected: Teal border + glow, checkmark visible, "✓ In box" text, quantity stepper
  • Disabled: Reduced opacity (when max items reached)

Accessibility: ARIA role="checkbox", aria-checked={isSelected}, keyboard navigation (Space/Enter to toggle)

Related Components: MM03 (ProductGrid container), MM05 (SelectionSummary shows selected items), EC01 (ProductCard for shop)
