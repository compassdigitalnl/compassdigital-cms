📦 MM06: MixMatchCategoryFilter

Purpose: Horizontal category filter tabs for Mix & Match product grids. Shows product count per category with emoji/icon.

Key Features:

  • Pill-style buttons (100px border-radius)
  • Active state: teal bg + white text
  • Inactive state: white bg + grey border + grey text
  • Product count badges (semi-transparent on active, grey-light on inactive)
  • Icon/emoji + label layout
  • Hover: teal border + teal text (inactive tabs)
  • Flex-wrap for responsive stacking

Design Tokens:

  • Padding: 8px 16px
  • Gap: 4px (between tabs), 5px (icon to label)
  • Border-radius: 100px (pills)
  • Border: 1.5px solid #E8ECF1 (inactive) / #00897B (active/hover)
  • Font: 13px 700 DM Sans
  • Icon: 14×14px (Lucide) or emoji
  • Count badge: 10px 800, 1/6px padding, 100px radius
  • Active bg: #00897B (teal), text: white
  • Inactive bg: white, text: #94A3B8 (grey-mid)
  • Hover: border #00897B, text #00897B

TypeScript Interface:

interface MixMatchCategoryFilterProps {
  categories: Array<{
    id: string;
    label: string;
    icon?: string;  // Lucide icon name
    emoji?: string; // Emoji character
    count: number;
  }>;
  activeId: string;
  onChange: (categoryId: string) => void;
  showCounts?: boolean; // Default: true
}

React Example:

export const MixMatchCategoryFilter = ({ categories, activeId, onChange }) => (
  <div className="cat-tabs">
    {categories.map((cat) => (
      <button
        key={cat.id}
        className={`cat-tab ${activeId === cat.id ? 'active' : ''}`}
        onClick={() => onChange(cat.id)}
      >
        {cat.icon && <Icon name={cat.icon} size={14} />}
        {cat.emoji && cat.emoji}
        {cat.label}
        <span className="cat-count">{cat.count}</span>
      </button>
    ))}
  </div>
);

Usage Example:

const [activeCategory, setActiveCategory] = useState('all');

<MixMatchCategoryFilter
  categories={[
    { id: 'all', label: 'Alles', icon: 'grid', count: 32 },
    { id: 'salades', label: 'Salades', emoji: '🥗', count: 8 },
    { id: 'wraps', label: 'Wraps', emoji: '🌯', count: 6 },
  ]}
  activeId={activeCategory}
  onChange={setActiveCategory}
/>
