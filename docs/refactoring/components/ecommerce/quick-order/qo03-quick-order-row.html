QO03: QuickOrderRow

Component Identity: Individual table row for quick order entry with product search, quantity input, price display, and delete action

Category: E-commerce / B2B / Forms

Source File: ef5-quick-order.html (lines 52-90)

STEP 1: Component Overview

Purpose

The QuickOrderRow is a single row in the QuickOrderTable (QO02). It provides product search autocomplete, quantity input, price/total calculation, and a delete button. This is a reusable component that handles all row-level logic.

Key Features

  • Product autocomplete: Search by SKU or name, dropdown shows matches
  • SKU display: Shows product SKU after selection (monospace teal text)
  • Staffel hints: Amber text showing volume discount opportunities
  • Quantity input: Number input with monospace font, centered text
  • Price display: Read-only unit price (monospace)
  • Total calculation: Auto-calculated (quantity × price)
  • Delete button: 40×40px icon button (hidden on empty rows)

States

  • Empty: No product selected, grey text, delete button hidden
  • Searching: Input focused, teal border, autocomplete dropdown visible
  • Filled: Product selected, teal input bg, SKU displayed, total calculated
  • With staffel hint: Amber text below input showing discount opportunity

STEP 2: When to Use

Use This Component When:

  • Building the QuickOrderTable (QO02) component
  • You need a reusable row component with autocomplete
  • Users enter products by SKU or name (B2B flows)

Don't Use This Component When:

  • Building standalone product search (use InstantSearch C1 instead)
  • Users browse visually (use ProductCard instead)
  • Single product entry (use standard Add to Cart instead)

Content Guidelines

  • Placeholder: "Zoek op artikelnummer of naam..." (descriptive)
  • SKU format: "SKU: XXX-XXXX" (uppercase, monospace)
  • Staffel hint: "Nog X voor €Y (−Z%)" (amber, concise)
  • Empty price: Show "—" not "€ 0,00"

STEP 7: Implementation Notes

React/Next.js Component

'use client'

import { useState } from 'react'
import { Trash2, TrendingDown } from 'lucide-react'
import { useDebouncedCallback } from 'use-debounce'

export function QuickOrderRow({
  row,
  onUpdate,
  onDelete,
}: QuickOrderRowProps) {
  const [showAutocomplete, setShowAutocomplete] = useState(false)
  const [autocompleteResults, setAutocompleteResults] = useState([])

  const handleSearch = useDebouncedCallback(async (query: string) => {
    if (query.length < 2) {
      setShowAutocomplete(false)
      return
    }

    const results = await fetch(`/api/products/search?q=${query}`)
    const products = await results.json()

    setAutocompleteResults(products)
    setShowAutocomplete(true)
  }, 300)

  const handleProductSelect = (product: Product) => {
    onUpdate(row.id, {
      productId: product.id,
      productName: product.name,
      sku: product.sku,
      unitPrice: product.price,
      total: product.price * row.quantity,
    })

    setShowAutocomplete(false)
  }

  const handleQuantityChange = (qty: number) => {
    onUpdate(row.id, {
      quantity: qty,
      total: qty * row.unitPrice,
    })
  }

  return (
    <div className="quick-order-row" role="row">
      {/* Product Input */}
      <div className="quick-order-product-input-wrapper" role="cell">
        <input
          type="text"
          value={row.productQuery}
          onChange={(e) => {
            onUpdate(row.id, { productQuery: e.target.value })
            handleSearch(e.target.value)
          }}
          className={`quick-order-product-input ${row.productId ? 'filled' : ''}`}
          placeholder="Zoek op artikelnummer of naam..."
          aria-controls={`autocomplete-${row.id}`}
          aria-expanded={showAutocomplete}
        />

        {/* SKU Display */}
        {row.sku && (
          <div className="quick-order-sku">SKU: {row.sku}</div>
        )}

        {/* Staffel Hint */}
        {row.staffelHint && (
          <div className="quick-order-staffel-hint">
            <TrendingDown />
            Nog {row.staffelHint.quantityNeeded} voor € {row.staffelHint.nextTierPrice.toFixed(2)} (−{row.staffelHint.discount}%)
          </div>
        )}

        {/* Autocomplete Dropdown */}
        {showAutocomplete && (
          <div className="quick-order-autocomplete" id={`autocomplete-${row.id}`} role="listbox">
            {autocompleteResults.map(product => (
              <div
                key={product.id}
                className="quick-order-autocomplete-item"
                role="option"
                onClick={() => handleProductSelect(product)}
              >
                <div className="autocomplete-item-name">{product.name}</div>
                <div className="autocomplete-item-meta">
                  <span className="autocomplete-item-sku">SKU: {product.sku}</span>
                  <span className="autocomplete-item-price">€ {product.price.toFixed(2)}</span>
                </div>
              </div>
            ))}
          </div>
        )}
      </div>

      {/* Quantity Input */}
      <input
        type="number"
        value={row.quantity || ''}
        onChange={(e) => handleQuantityChange(parseInt(e.target.value) || 0)}
        className="quick-order-qty-input"
        placeholder="0"
        min="1"
        role="cell"
      />

      {/* Price Display */}
      <div className={`quick-order-price ${!row.unitPrice ? 'empty' : ''}`} role="cell">
        {row.unitPrice ? `€ ${row.unitPrice.toFixed(2)}` : '—'}
      </div>

      {/* Total Display */}
      <div className={`quick-order-total ${!row.total ? 'empty' : ''}`} role="cell">
        € {row.total.toFixed(2)}
      </div>

      {/* Delete Button */}
      <button
        onClick={() => onDelete(row.id)}
        className="quick-order-delete-button"
        aria-label="Verwijder product"
        role="cell"
      >
        <Trash2 />
      </button>
    </div>
  )
}

Performance Optimization

  • Use React.memo to prevent re-renders when other rows change
  • Debounce autocomplete search (300ms) to reduce API calls
  • Use controlled inputs with local state for smoother typing
  • Close autocomplete on click outside (useClickOutside hook)

Note: This component is designed to be used inside QO02 (QuickOrderTable). It relies on the table's grid layout for column sizing.
