QO04: CSVUploadButton

Component Identity: File upload button for CSV bulk import with loading, success, and error states

Category: Forms / File Upload

Source File: ef5-quick-order.html (lines 26-29, integrated with QO01)

STEP 1: Component Overview

Purpose

The CSVUploadButton allows B2B users to bulk import products via CSV file upload. It provides a accessible file input with visual feedback for upload progress, success, and errors.

Key Features

  • Accessible file input: Hidden native input + styled label button
  • CSV validation: Accepts only .csv files (configurable)
  • 4 states: Default, Loading, Success, Error
  • Feedback messages: Optional success/error text below button
  • Template download: Optional link to CSV template file
  • Keyboard accessible: Full keyboard navigation support

States

  • Default: Upload icon, "CSV uploaden" label, grey border
  • Loading: Spinner icon, "Verwerken..." label, disabled state
  • Success: Check icon, "CSV geüpload" label, green border/bg
  • Error: X icon, "Ongeldige CSV" label, coral border/bg

STEP 2: When to Use

Use This Component When:

  • Building quick order pages (B2B bulk entry)
  • Users need to import product lists from Excel/CSV
  • You want to enable bulk product uploads
  • Integrating with QuickOrderTable (QO02)

Don't Use This Component When:

  • Uploading images (use ImageUploadDropzone instead)
  • Single product entry (use QuickOrderRow QO03 instead)
  • Uploading non-tabular data

Content Guidelines

  • Default label: "CSV uploaden" (clear, short)
  • Loading label: "Verwerken..." (indicates progress)
  • Success label: "CSV geüpload" (past tense confirmation)
  • Error label: "Ongeldige CSV" or specific error
  • Feedback message: Include count (e.g., "42 producten geïmporteerd")

STEP 6: Accessibility

Semantic HTML

  • Use native <input type="file"> with unique ID
  • Use <label for="id"> to connect button to input
  • Add role="button" and tabindex="0" to label
  • Add accept attribute to limit file types

ARIA Attributes

  • aria-label="Upload CSV bestand" on label (if icon-only)
  • aria-busy="true" during loading state
  • aria-describedby linking to feedback message
  • aria-disabled="true" when disabled

Keyboard Navigation

  • Tab: Focus on label button
  • Enter/Space: Trigger file picker
  • Escape: Close file picker (native behavior)

Screen Reader Support

  • Label text announced: "CSV uploaden, knop"
  • State changes announced via aria-live regions
  • Feedback messages read automatically

React/Next.js Component

'use client'

import { useState, useRef } from 'react'
import { Upload, Loader2, CheckCircle, XCircle, Download } from 'lucide-react'

interface CSVUploadButtonProps {
  id: string
  onFileSelect: (file: File) => Promise<void>
  showTemplate?: boolean
  templateUrl?: string
}

export function CSVUploadButton({
  id,
  onFileSelect,
  showTemplate = true,
  templateUrl = '/templates/quick-order.csv',
}: CSVUploadButtonProps) {
  const [state, setState] = useState<'default' | 'loading' | 'success' | 'error'>('default')
  const [feedback, setFeedback] = useState('')
  const fileInputRef = useRef<HTMLInputElement>(null)

  const handleFileChange = async (e: React.ChangeEvent<HTMLInputElement>) => {
    const file = e.target.files?.[0]
    if (!file) return

    // Validate file type
    if (!file.name.endsWith('.csv')) {
      setState('error')
      setFeedback('Alleen CSV bestanden zijn toegestaan.')
      return
    }

    try {
      setState('loading')
      setFeedback('Bestand wordt verwerkt...')

      await onFileSelect(file)

      setState('success')
      setFeedback('CSV succesvol geüpload!')

      // Reset after 3 seconds
      setTimeout(() => {
        setState('default')
        setFeedback('')
      }, 3000)
    } catch (error) {
      setState('error')
      setFeedback(error instanceof Error ? error.message : 'Upload mislukt.')
    }
  }

  const iconMap = {
    default: Upload,
    loading: Loader2,
    success: CheckCircle,
    error: XCircle,
  }

  const labelMap = {
    default: 'CSV uploaden',
    loading: 'Verwerken...',
    success: 'CSV geüpload',
    error: 'Ongeldige CSV',
  }

  const Icon = iconMap[state]

  return (
    <div>
      <div className="csv-upload-wrapper">
        <input
          ref={fileInputRef}
          type="file"
          id={id}
          className="csv-upload-input"
          accept=".csv"
          onChange={handleFileChange}
          disabled={state === 'loading'}
        />
        <label
          htmlFor={id}
          className={`csv-upload-button ${state}`}
          role="button"
          tabIndex={0}
          aria-busy={state === 'loading'}
        >
          <Icon className={state === 'loading' ? 'animate-spin' : ''} />
          {labelMap[state]}
        </label>
      </div>

      {/* Feedback Message */}
      {feedback && (
        <div className={`csv-upload-feedback ${state}`}>
          <Icon />
          {feedback}
        </div>
      )}

      {/* Template Download Link */}
      {showTemplate && (
        <a href={templateUrl} download className="csv-template-link">
          <Download />
          Download CSV sjabloon
        </a>
      )}
    </div>
  )
}

CSV Parsing Example

import Papa from 'papaparse'

async function parseCSV(file: File): Promise<QuickOrderRow[]> {
  return new Promise((resolve, reject) => {
    Papa.parse(file, {
      header: true,
      skipEmptyLines: true,
      complete: (results) => {
        const rows = results.data.map((row: any) => ({
          id: Math.random().toString(),
          sku: row.SKU,
          quantity: parseInt(row.Quantity) || 0,
        }))
        resolve(rows)
      },
      error: (error) => reject(error),
    })
  })
}

// Usage
const handleCSVUpload = async (file: File) => {
  const rows = await parseCSV(file)

  // Fetch product details for each SKU
  const productsWithPrices = await Promise.all(
    rows.map(async (row) => {
      const product = await fetch(`/api/products?sku=${row.sku}`)
      const data = await product.json()

      return {
        ...row,
        productId: data.id,
        productName: data.name,
        unitPrice: data.price,
        total: data.price * row.quantity,
      }
    })
  )

  // Add to Quick Order Table
  addRowsToTable(productsWithPrices)
}

Library Recommendation: Use papaparse for robust CSV parsing with error handling and header detection.
