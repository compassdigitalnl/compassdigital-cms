QO05: ProTipBanner

Component Identity: Teal info banner with lightbulb icon for helpful tips and best practices

Category: UI Elements / Informational

Source File: ef5-quick-order.html (lines 112-118)

STEP 1: Component Overview

Purpose

The ProTipBanner displays helpful tips, best practices, or feature highlights to users. It uses a teal background to differentiate from alerts/warnings while maintaining a friendly, informative tone.

Key Features

  • Teal branding: Light teal background (#00897B08) with teal border and icon
  • Lightbulb icon: 20×20px icon indicating helpful information
  • "Pro tip:" label: Bold teal text to emphasize the tip nature
  • Rich text support: Allows bold text and links within content
  • Optional close button: Dismissible variant for non-critical tips
  • Compact variant: Smaller padding for inline use

Use Cases

  • Explaining advanced features (CSV upload, bestellijsten)
  • Suggesting time-saving shortcuts
  • Highlighting popular B2B features
  • Providing onboarding guidance

STEP 2: When to Use

Use This Component When:

  • Providing helpful tips or best practices
  • Explaining non-obvious features
  • Highlighting efficiency shortcuts
  • Guiding users toward advanced functionality
  • Displaying positive, informational messages

Don't Use This Component When:

  • Showing errors (use ErrorBanner or ToastSystem instead)
  • Displaying warnings (use WarningBanner instead)
  • Confirming actions (use SuccessBanner instead)
  • Critical system messages (use AlertBanner instead)

Content Guidelines

  • Label: Start with "Pro tip:", "Tip:", "Let op:", or similar indicator
  • Tone: Friendly, helpful, concise (max 2 sentences)
  • Action-oriented: Include actionable advice or links
  • Avoid jargon: Use plain language accessible to all users
  • Bold keywords: Emphasize important terms (bestellijst, CSV, etc.)

STEP 7: Implementation Notes

React/Next.js Component

'use client'

import { useState } from 'react'
import { Lightbulb, Info, Zap, Star, Award, X } from 'lucide-react'

interface ProTipBannerProps {
  label?: string
  content: string | React.ReactNode
  icon?: 'lightbulb' | 'info' | 'zap' | 'star' | 'award'
  variant?: 'default' | 'compact'
  dismissible?: boolean
}

export function ProTipBanner({
  label = 'Pro tip:',
  content,
  icon = 'lightbulb',
  variant = 'default',
  dismissible = false,
}: ProTipBannerProps) {
  const [isVisible, setIsVisible] = useState(true)

  const iconMap = {
    lightbulb: Lightbulb,
    info: Info,
    zap: Zap,
    star: Star,
    award: Award,
  }

  const Icon = iconMap[icon]

  if (!isVisible) return null

  return (
    <aside
      className={`pro-tip-banner ${variant}`}
      role="note"
    >
      <Icon className="pro-tip-icon" aria-hidden="true" />
      <div className="pro-tip-content">
        <strong className="pro-tip-label">{label}</strong> {content}
      </div>
      {dismissible && (
        <button
          onClick={() => setIsVisible(false)}
          className="pro-tip-close"
          aria-label="Sluit tip"
        >
          <X />
        </button>
      )}
    </aside>
  )
}

LocalStorage Persistence (Dismissible)

export function ProTipBanner({ id, dismissible, ...props }: Props) {
  const storageKey = `tip-dismissed-${id}`
  const [isVisible, setIsVisible] = useState(() => {
    if (typeof window === 'undefined') return true
    return !localStorage.getItem(storageKey)
  })

  const handleDismiss = () => {
    setIsVisible(false)
    if (id) {
      localStorage.setItem(storageKey, 'true')
    }
  }

  // ... rest of component
}
