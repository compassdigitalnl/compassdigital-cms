📋 STEP 1: COMPONENT OVERVIEW

Component: C11 - Notification Center

Category: Account / Communication

Complexiteit: High (dropdown, tabs, filtering, unread states, color-coded categories)

Purpose:

Notification center dropdown from header bell icon. Shows order updates, stock alerts, invoices, and system messages. Improves engagement by keeping users informed of important account events.

Key Features:

  • Bell trigger button with coral unread badge
  • Dropdown panel (400px width, max-height 380px)
  • 4 tabs: Alles, Bestellingen, Voorraad, Systeem (with count badges)
  • Color-coded notification icons (green/teal/blue/amber)
  • Unread indicator (teal dot + light teal background)
  • Notification title, description, timestamp
  • "Alles gelezen" button in header
  • Footer link to full notifications page
  • Click outside to close

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • E-commerce platforms (order tracking, stock alerts)
  • B2B accounts (invoices, credit notes, account updates)
  • Subscription services (renewal reminders, failed payments)
  • Any app with user-specific alerts

Best Practices:

  • Real-time updates via WebSocket or polling (every 30-60s)
  • Show max 5-10 notifications in dropdown (link to full page)
  • Auto-dismiss badge when dropdown opened
  • Persist read/unread state per user
  • Send email for critical notifications (failed payments, delivery issues)
  • Group similar notifications ("3 items back in stock")
  • Allow notification preferences in account settings

💡 STEP 7: IMPLEMENTATION NOTES

// React Component
export function NotificationCenter() {
  const [isOpen, setIsOpen] = useState(false);
  const [activeTab, setActiveTab] = useState('Alles');
  const { data: notifications, mutate } = useSWR('/api/notifications');

  const unreadCount = notifications?.filter(n => !n.read).length || 0;

  const filteredNotifications = useMemo(() => {
    if (activeTab === 'Alles') return notifications;
    return notifications.filter(n => n.type === tabTypeMap[activeTab]);
  }, [notifications, activeTab]);

  const handleMarkAllRead = async () => {
    await fetch('/api/notifications/mark-all-read', { method: 'POST' });
    mutate();
  };

  const handleNotificationClick = async (notification) => {
    if (!notification.read) {
      await fetch(`/api/notifications/${notification.id}/read`, { method: 'POST' });
      mutate();
    }
    router.push(notification.link);
    setIsOpen(false);
  };

  return (
    <div className="notif-wrap" ref={wrapRef}>
      <button className="notif-trigger" onClick={() => setIsOpen(!isOpen)}>
        <Bell size={20} />
        {unreadCount > 0 && <span className="notif-badge">{unreadCount}</span>}
      </button>

      <div className={`notif-dropdown ${isOpen ? 'open' : ''}`}>
        <div className="nd-header">
          <h3>Meldingen</h3>
          <button className="nd-header-btn" onClick={handleMarkAllRead}>
            Alles gelezen
          </button>
        </div>

        <div className="nd-tabs">
          {tabs.map(tab => (
            <div
              key={tab}
              className={`nd-tab ${activeTab === tab ? 'active' : ''}`}
              onClick={() => setActiveTab(tab)}
            >
              {tab}
              {getTabCount(tab) > 0 && (
                <span className="tab-badge">{getTabCount(tab)}</span>
              )}
            </div>
          ))}
        </div>

        <div className="nd-list">
          {filteredNotifications.map(notification => (
            <a
              key={notification.id}
              className={`nd-item ${!notification.read ? 'unread' : ''}`}
              onClick={() => handleNotificationClick(notification)}
            >
              <div className="nd-icon" style={{ background: getIconBg(notification.iconColor) }}>
                <Icon name={notification.icon} size={17} />
              </div>
              <div className="nd-body">
                <div className="nd-title">{notification.title}</div>
                <div className="nd-text">{notification.text}</div>
                <div className="nd-time">
                  <Clock size={11} /> {formatRelativeTime(notification.timestamp)}
                </div>
              </div>
            </a>
          ))}
        </div>

        <div className="nd-footer">
          <a href="/account/notifications">
            Alle meldingen bekijken <ArrowRight size={14} />
          </a>
        </div>
      </div>
    </div>
  );
}
