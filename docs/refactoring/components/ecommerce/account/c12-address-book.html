📋 STEP 1: COMPONENT OVERVIEW

Component: C12 - Address Book Manager

Category: Account / Checkout

Complexiteit: High (CRUD operations, primary address logic, inline editing, validation)

Purpose:

Address management interface for user accounts. Allows adding, editing, deleting, and setting default shipping/billing addresses. Used in My Account and during checkout for address selection.

Key Features:

  • 2-column grid layout (responsive to 1-col on mobile)
  • Address cards with type icons (building, file-text, home)
  • Primary address indicator (teal border + "Standaard" badge)
  • 3 action buttons: Edit, Set as Default, Delete
  • Duplicate button for primary address (copy & modify)
  • Dashed "Add new" card with plus icon
  • Inline edit form (modal or expandable panel)
  • Validation for postal code, required fields

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • E-commerce checkout (shipping/billing address selection)
  • Account settings (/account/addresses)
  • B2B accounts (multiple delivery locations)
  • Subscription services (different billing addresses)

Best Practices:

  • Always have exactly ONE primary/default address
  • Prevent deletion of last address (require min 1)
  • Validate postal codes with regex or API (e.g., PostcodeAPI)
  • Auto-fill city from postal code (UX improvement)
  • Show address preview before saving
  • Confirm before deleting address
  • Allow "Duplicate" for creating similar addresses

💡 STEP 7: IMPLEMENTATION NOTES

// React Component
export function AddressBook() {
  const [addresses, setAddresses] = useState([]);
  const [editingId, setEditingId] = useState(null);

  const handleSetPrimary = async (addressId) => {
    await fetch(`/api/addresses/${addressId}/set-primary`, { method: 'POST' });
    // Update local state: unset old primary, set new primary
    setAddresses(prev => prev.map(a => ({
      ...a,
      isPrimary: a.id === addressId
    })));
  };

  const handleDelete = async (addressId) => {
    if (addresses.length === 1) {
      toast.error('U moet minimaal 1 adres hebben');
      return;
    }

    if (confirm('Weet u zeker dat u dit adres wilt verwijderen?')) {
      await fetch(`/api/addresses/${addressId}`, { method: 'DELETE' });
      setAddresses(prev => prev.filter(a => a.id !== addressId));
    }
  };

  const handleDuplicate = (addressId) => {
    const address = addresses.find(a => a.id === addressId);
    setEditingId('new');
    setFormData({ ...address, id: null, isPrimary: false });
  };

  const handleSave = async (formData) => {
    if (formData.id) {
      // Update existing
      await fetch(`/api/addresses/${formData.id}`, {
        method: 'PATCH',
        body: JSON.stringify(formData)
      });
    } else {
      // Create new
      await fetch('/api/addresses', {
        method: 'POST',
        body: JSON.stringify(formData)
      });
    }
    setEditingId(null);
    refetch();
  };

  const sortedAddresses = useMemo(() => {
    return [...addresses].sort((a, b) => {
      if (a.isPrimary) return -1;
      if (b.isPrimary) return 1;
      return new Date(b.createdAt) - new Date(a.createdAt);
    });
  }, [addresses]);

  return (
    <div className="addr-manager">
      <div className="addr-header">
        <h2>Mijn adressen ({addresses.length})</h2>
      </div>

      <div className="addr-grid">
        {sortedAddresses.map(address => (
          <AddressCard
            key={address.id}
            address={address}
            onEdit={() => setEditingId(address.id)}
            onDelete={() => handleDelete(address.id)}
            onSetPrimary={() => handleSetPrimary(address.id)}
            onDuplicate={() => handleDuplicate(address.id)}
          />
        ))}
        <div className="addr-add" onClick={() => setEditingId('new')}>
          <div className="addr-add-icon"><Plus /></div>
          <div className="addr-add-text">Nieuw adres toevoegen</div>
        </div>
      </div>

      {editingId && (
        <AddressEditForm
          address={editingId === 'new' ? null : addresses.find(a => a.id === editingId)}
          onSave={handleSave}
          onCancel={() => setEditingId(null)}
        />
      )}
    </div>
  );
}
