📋 STEP 1: COMPONENT OVERVIEW

Component: C13 - Recently Viewed Products

Category: Account / Personalization

Complexiteit: Medium (horizontal scroll, localStorage persistence, time formatting)

Purpose:

Horizontal scroll carousel displaying recently viewed products with time indicators. Helps users quickly return to products they've browsed. Increases conversion by reducing friction in the purchase journey.

Key Features:

  • Horizontal scroll carousel with scroll-snap behavior
  • Navigation arrows (left/right chevrons)
  • Product cards (180px width) with emoji placeholders
  • Time stamp overlay ("10 min", "1 uur", "Gisteren")
  • Brand label (teal, uppercase)
  • Product name (2-line clamp)
  • Price + direct add-to-cart button
  • Hover-reveal favorite button (heart icon)
  • "Clear history" link with trash icon
  • Smooth scroll behavior on navigation clicks

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • Homepage (below hero/featured products)
  • Product pages (cross-sell section at bottom)
  • Account dashboard ("Verder winkelen" widget)
  • Cart page (continue browsing section)
  • 404/empty search results (fallback content)

Best Practices:

  • Track up to 12-20 recently viewed products (don't store indefinitely)
  • Persist in localStorage (cookie-free, 30-day expiry)
  • Remove duplicates (keep most recent timestamp)
  • Show relative time: "10 min", "1 uur", "Gisteren", "X dagen"
  • Remove deleted/out-of-stock products from history
  • Add "View more" link if >7 products
  • Respect privacy: delete on logout (optional based on UX)
  • On mobile: hide nav arrows, use touch swipe

💡 STEP 7: IMPLEMENTATION NOTES

// React Component with localStorage persistence
import { useState, useEffect, useRef } from 'react';
import { Clock, ChevronLeft, ChevronRight, Trash2, Heart, ShoppingCart } from 'lucide-react';

export function RecentlyViewed({ maxProducts = 12 }) {
  const [products, setProducts] = useState([]);
  const scrollRef = useRef(null);

  useEffect(() => {
    // Load from localStorage
    const stored = localStorage.getItem('recently_viewed');
    if (stored) {
      const parsed = JSON.parse(stored);
      // Sort by viewedAt desc, take maxProducts
      const sorted = parsed
        .sort((a, b) => new Date(b.viewedAt) - new Date(a.viewedAt))
        .slice(0, maxProducts);
      setProducts(sorted);
    }
  }, [maxProducts]);

  const handleScroll = (direction) => {
    if (scrollRef.current) {
      scrollRef.current.scrollBy({
        left: direction * 200,
        behavior: 'smooth'
      });
    }
  };

  const handleClearHistory = () => {
    if (confirm('Weet u zeker dat u uw browsegeschiedenis wilt wissen?')) {
      localStorage.removeItem('recently_viewed');
      setProducts([]);
      // Analytics
      analytics.track('Recently Viewed Cleared');
    }
  };

  const formatTimestamp = (timestamp) => {
    const now = new Date();
    const viewed = new Date(timestamp);
    const diffMs = now - viewed;
    const diffMins = Math.floor(diffMs / 60000);
    const diffHours = Math.floor(diffMs / 3600000);
    const diffDays = Math.floor(diffMs / 86400000);

    if (diffMins < 60) return `${diffMins} min`;
    if (diffHours < 24) return `${diffHours} uur`;
    if (diffDays === 1) return 'Gisteren';
    return `${diffDays} dagen`;
  };

  if (products.length === 0) return null;

  return (
    <div className="rv-section">
      <div className="rv-header">
        <h2 className="rv-title">
          <Clock size={20} /> Recent bekeken
        </h2>
        <div className="rv-nav">
          <button
            className="rv-nav-btn"
            onClick={() => handleScroll(-1)}
            aria-label="Vorige producten"
          >
            <ChevronLeft size={16} />
          </button>
          <button
            className="rv-nav-btn"
            onClick={() => handleScroll(1)}
            aria-label="Volgende producten"
          >
            <ChevronRight size={16} />
          </button>
        </div>
      </div>

      <div className="rv-scroll" ref={scrollRef} role="list">
        {products.map((product, i) => (
          <a
            key={product.id}
            href={`/product/${product.slug}`}
            className="rv-card"
            role="listitem"
            onClick={() => {
              analytics.track('Recently Viewed Product Clicked', {
                productId: product.id,
                position: i
              });
            }}
          >
            <button
              className="rv-card-fav"
              onClick={(e) => {
                e.preventDefault();
                toggleFavorite(product.id);
              }}
              aria-label="Voeg toe aan favorieten"
            >
              <Heart size={14} />
            </button>

            <div className="rv-card-img">
              <img src={product.image} alt={product.name} />
              <div className="rv-card-time">
                <Clock size={10} /> {formatTimestamp(product.viewedAt)}
              </div>
            </div>

            <div className="rv-card-body">
              <div className="rv-card-brand">{product.brand}</div>
              <div className="rv-card-name">{product.name}</div>
              <div className="rv-card-bottom">
                <span className="rv-card-price">€{product.price.toFixed(2)}</span>
                <button
                  className="rv-card-cart"
                  onClick={(e) => {
                    e.preventDefault();
                    addToCart(product.id);
                  }}
                  aria-label="Voeg toe aan winkelwagen"
                >
                  <ShoppingCart size={13} />
                </button>
              </div>
            </div>
          </a>
        ))}
      </div>

      <button
        className="rv-clear"
        onClick={handleClearHistory}
        role="button"
        tabIndex={0}
      >
        <Trash2 size={13} /> Geschiedenis wissen
      </button>
    </div>
  );
}

// Track product views (call on product page mount)
export function trackProductView(product) {
  const stored = localStorage.getItem('recently_viewed');
  const products = stored ? JSON.parse(stored) : [];

  // Remove duplicate if exists
  const filtered = products.filter(p => p.id !== product.id);

  // Add to front with current timestamp
  filtered.unshift({
    id: product.id,
    name: product.name,
    brand: product.brand,
    price: product.price,
    image: product.image,
    slug: product.slug,
    viewedAt: new Date().toISOString()
  });

  // Keep max 20 products
  const capped = filtered.slice(0, 20);

  localStorage.setItem('recently_viewed', JSON.stringify(capped));
}

  • Practical limit: 12-20 products to avoid localStorage bloat
  • Use IntersectionObserver to lazy-load product images as they scroll into view
  • Debounce scroll events if tracking scroll position for analytics
  • Consider IndexedDB for very large history (100+ products) with image caching
