Description: Sidebar component displaying order totals breakdown (subtotal, discount, shipping, tax) with grand total and checkout CTA buttons. Used in cart pages, checkout reviews, and order confirmations.

Key Features:

  • Line-by-line breakdown of costs
  • Discount display (green, negative value)
  • Free shipping indicator
  • Tax calculation (21% BTW)
  • Grand total (large, bold, Jakarta Sans 28px 800)
  • Tax-inclusive note
  • Primary CTA: "Naar betalen" (gradient teal button)
  • Secondary CTA: "Offerte aanvragen" (outline button)
  • Sticky positioning option (for cart sidebar)

🎯 STEP 2: When to Use

Use this component when:

  • Displaying cart totals on cart page (right sidebar)
  • Showing order breakdown on checkout page
  • Reviewing order before confirmation
  • Displaying final order totals on confirmation page
  • In mini cart flyout (condensed version)

Common Design Patterns:

  • Cart Sidebar: Sticky sidebar at top: 90px (below header)
  • Checkout: Right column, static positioning
  • Order Confirmation: Read-only (no CTAs), center-aligned
  • Mini Cart: Compact version at bottom of flyout

Do NOT use when:

  • Cart is empty (show empty state instead)
  • On product pages (use price display only)
  • For single-item quick orders (too much overhead)

Auto-Calculation Hook

// Hook to calculate order totals from cart items
export function useOrderSummary(cart: Cart, discountCode?: string) {
  const subtotal = cart.items.reduce((sum, item) =>
    sum + (item.price * item.quantity), 0
  )

  // Apply discount
  let discount = 0
  if (discountCode) {
    const coupon = await fetchCoupon(discountCode)
    discount = coupon.type === 'percentage'
      ? subtotal * (coupon.value / 100)
      : coupon.value
  }

  const subtotalAfterDiscount = subtotal - discount

  // Calculate shipping
  const shipping = subtotalAfterDiscount >= 60 ? 0 : 6.95

  // Calculate tax (21%)
  const tax = (subtotalAfterDiscount + shipping) * 0.21

  const total = subtotalAfterDiscount + shipping + tax

  return {
    subtotal,
    discount,
    shipping,
    tax,
    total
  }
}

💻 STEP 7: Implementation Notes

React/Next.js Component

'use client'

import { Receipt, Lock, FileText } from 'lucide-react'

export function OrderSummary({
  subtotal,
  discount,
  discountCode,
  shipping,
  tax,
  total,
  showQuoteButton = true,
  sticky = false,
  readonly = false,
  taxRate = 21,
  currencySymbol = '€',
  locale = 'nl-NL',
  onCheckout,
  onRequestQuote
}: OrderSummaryProps) {

  const formatCurrency = (amount: number) => {
    return new Intl.NumberFormat(locale, {
      style: 'currency',
      currency: 'EUR',
      minimumFractionDigits: 2
    }).format(amount)
  }

  const isShippingFree = shipping === 'free' || shipping === 0

  return (
    <aside
      className={`order-summary ${sticky ? 'order-summary--sticky' : ''}`}
      aria-label="Order summary"
    >
      <h2 className="order-summary__header">
        <Receipt aria-hidden="true" />
        Overzicht
      </h2>

      <dl className="order-summary__breakdown">
        <div className="order-summary__row">
          <dt className="order-summary__label">Subtotaal</dt>
          <dd className="order-summary__value">
            {formatCurrency(subtotal)}
          </dd>
        </div>

        {discount && discount > 0 && (
          <div className="order-summary__row">
            <dt className="order-summary__label">
              Korting {discountCode && `(${discountCode})`}
            </dt>
            <dd className="order-summary__value order-summary__value--discount">
              −{formatCurrency(discount)}
            </dd>
          </div>
        )}

        <div className="order-summary__row">
          <dt className="order-summary__label">Verzendkosten</dt>
          <dd className="order-summary__value order-summary__value--discount">
            {isShippingFree ? 'Gratis' : formatCurrency(shipping as number)}
          </dd>
        </div>

        <div className="order-summary__row">
          <dt className="order-summary__label">BTW ({taxRate}%)</dt>
          <dd className="order-summary__value">{formatCurrency(tax)}</dd>
        </div>
      </dl>

      <hr className="order-summary__divider" />

      <div className="order-summary__total">
        <span className="order-summary__total-label">Totaal</span>
        <span className="order-summary__total-value">
          {currencySymbol} {Math.floor(total)}
          <small>,{((total % 1) * 100).toFixed(0).padStart(2, '0')}</small>
        </span>
      </div>
      <div className="order-summary__tax-note">Inclusief BTW</div>

      {!readonly && (
        <div className="order-summary__actions">
          <button
            className="order-summary__btn order-summary__btn--primary"
            onClick={onCheckout}
          >
            <Lock size={20} />
            Naar betalen
          </button>

          {showQuoteButton && (
            <button
              className="order-summary__btn order-summary__btn--secondary"
              onClick={onRequestQuote}
            >
              <FileText size={18} />
              Offerte aanvragen
            </button>
          )}
        </div>
      )}
    </aside>
  )
}

Important Implementation Details

  • Currency Formatting: Use Intl.NumberFormat for locale-aware formatting
  • Sticky Positioning: Only on desktop, use top: 90px to account for header
  • Free Shipping: Accept 'free' string or 0 number, display "Gratis"
  • Discount Display: Always show negative sign (−) and green color
  • Total Formatting: Split euros and cents with <small> tag
  • Semantic HTML: Use <aside>, <dl>, <dt>, <dd>
