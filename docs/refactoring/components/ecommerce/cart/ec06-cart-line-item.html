📋 STEP 1: Component Overview

Property Value

Priority CRITICAL

Description: Individual cart item row displaying product details, quantity controls, pricing, and item actions. Core component for cart pages, mini cart flyouts, and checkout reviews.

Key Features:

  • Product image (100px × 100px, rounded)
  • Brand label (uppercase teal)
  • Product title and SKU
  • Variable attributes display (size, color, etc.)
  • Stock status indicator (3 states: in-stock, low, out)
  • Quantity stepper with +/− buttons
  • Price calculation (quantity × unit price)
  • Unit price display
  • Action buttons (add to list, remove)
  • Responsive layout (stacks on mobile)

🎯 STEP 2: When to Use

Use this component when:

  • Displaying items in a shopping cart
  • Showing cart contents in a mini cart flyout
  • Reviewing order items on checkout page
  • Displaying saved cart items or wishlists
  • Building order history/detail views

Common Design Patterns:

  • Cart Page: Stacked list of line items with full details
  • Mini Cart: Compact version with smaller image (60px)
  • Checkout Review: Read-only version (no qty stepper)
  • Order Confirmation: Static display with ordered quantities

Do NOT use when:

  • Displaying products in a catalog (use EC01: ProductCard instead)
  • Showing single-line order summaries (use OrderSummary instead)
  • Building quick-order forms (use QuickOrderRow instead)

⚙️ STEP 4: Props & Configuration

TypeScript Interface

interface CartLineItemProps {
  // Product Data
  id: string
  product: {
    id: string
    name: string
    slug: string
    sku: string
    brand: {
      name: string
      slug: string
    }
    image: {
      url: string
      alt: string
    }
  }

  // Pricing
  price: number              // Snapshot price at time of add
  unitPrice: number          // Price per single unit
  unit?: string              // "per stuk", "per 100 stuks"

  // Quantity
  quantity: number
  minQuantity?: number       // Default: 1
  maxQuantity?: number       // Stock-based limit

  // Attributes (for variable products)
  attributes?: {
    name: string           // "Maat", "Kleur"
    value: string          // "L", "Blauw"
    icon?: string          // Lucide icon name
  }[]

  // Stock Status
  stock: number
  stockStatus: 'in-stock' | 'low' | 'out'
  stockText?: string       // Custom stock message

  // Actions
  onQuantityChange?: (newQuantity: number) => void
  onAddToList?: (productId: string) => void
  onRemove?: (itemId: string) => void

  // Display Options
  readonly?: boolean       // Disable quantity/actions (checkout review)
  compact?: boolean        // Smaller image/spacing (mini cart)
  className?: string
}

Payload CMS Cart Schema

{
  slug: 'carts',
  fields: [
    { name: 'user', type: 'relationship', relationTo: 'users' },
    { name: 'sessionId', type: 'text' }, // For guest users
    {
      name: 'items',
      type: 'array',
      fields: [
        {
          name: 'product',
          type: 'relationship',
          relationTo: 'products',
          required: true
        },
        {
          name: 'quantity',
          type: 'number',
          required: true,
          min: 1
        },
        {
          name: 'selectedAttributes',
          type: 'json',
          admin: {
            description: 'For variable products: {size: "L", color: "Blue"}'
          }
        },
        {
          name: 'price',
          type: 'number',
          required: true,
          admin: {
            description: 'Snapshot price at time of add-to-cart'
          }
        },
        {
          name: 'addedAt',
          type: 'date',
          defaultValue: () => new Date()
        }
      ]
    }
  ],
  hooks: {
    afterRead: [
      async ({ doc, req }) => {
        // Populate product data with current stock status
        for (const item of doc.items) {
          const product = await req.payload.findByID({
            collection: 'products',
            id: item.product.id
          })
          item.product.stock = product.stock
          item.product.stockStatus = product.stockStatus
        }
        return doc
      }
    ]
  }
}

💻 STEP 7: Implementation Notes

React/Next.js Component

'use client'

import { useState } from 'react'
import Image from 'next/image'
import {
  Box, Ruler, CheckCircle, AlertTriangle,
  XCircle, ListPlus, Trash2
} from 'lucide-react'

export function CartLineItem({
  id,
  product,
  price,
  unitPrice,
  unit,
  quantity: initialQuantity,
  stock,
  stockStatus,
  attributes = [],
  readonly = false,
  compact = false,
  onQuantityChange,
  onAddToList,
  onRemove
}: CartLineItemProps) {

  const [quantity, setQuantity] = useState(initialQuantity)

  const handleDecrease = () => {
    if (quantity > 1) {
      const newQty = quantity - 1
      setQuantity(newQty)
      onQuantityChange?.(newQty)
    }
  }

  const handleIncrease = () => {
    if (quantity < stock) {
      const newQty = quantity + 1
      setQuantity(newQty)
      onQuantityChange?.(newQty)
    }
  }

  const handleInputChange = (e: React.ChangeEvent<HTMLInputElement>) => {
    const value = parseInt(e.target.value)
    if (!isNaN(value) && value >= 1 && value <= stock) {
      setQuantity(value)
      onQuantityChange?.(value)
    }
  }

  const totalPrice = unitPrice * quantity

  const StockIcon = stockStatus === 'in-stock' ? CheckCircle :
                    stockStatus === 'low' ? AlertTriangle : XCircle

  const stockText = stockStatus === 'in-stock'
    ? `Op voorraad${stock ? ` (${stock.toLocaleString('nl-NL')} stuks)` : ''}`
    : stockStatus === 'low'
    ? `Laag op voorraad${stock ? ` (${stock} stuks)` : ''}`
    : 'Tijdelijk uitverkocht'

  return (
    <article className="cart-line-item" aria-label={`Cart item: ${product.name}`}>
      <div className="cart-line-item__main">
        <div className={`cart-line-item__image ${compact ? 'cart-line-item__image--compact' : ''}`}>
          <Image
            src={product.image.url}
            alt={product.image.alt}
            width={compact ? 60 : 100}
            height={compact ? 60 : 100}
          />
        </div>

        <div className="cart-line-item__details">
          <div className="cart-line-item__brand">{product.brand.name}</div>
          <div className="cart-line-item__title">{product.name}</div>

          <div className="cart-line-item__meta">
            <span className="cart-line-item__meta-item">
              <Box size={12} />
              SKU: {product.sku}
            </span>
            {attributes.map((attr, i) => (
              <span key={i} className="cart-line-item__meta-item">
                {attr.icon && <Ruler size={12} />}
                {attr.name}: {attr.value}
              </span>
            ))}
          </div>

          <div className={`cart-line-item__stock cart-line-item__stock--${stockStatus}`}>
            <StockIcon size={12} />
            {stockText}
          </div>
        </div>

        {!readonly && (
          <div className="cart-line-item__qty-wrapper">
            <div className="cart-line-item__qty-stepper" role="group" aria-label="Aantal aanpassen">
              <button
                className="cart-line-item__qty-btn"
                onClick={handleDecrease}
                disabled={quantity <= 1 || stockStatus === 'out'}
                aria-label="Verminder aantal"
              >
                −
              </button>
              <input
                type="number"
                className="cart-line-item__qty-input"
                value={quantity}
                onChange={handleInputChange}
                min={1}
                max={stock}
                disabled={stockStatus === 'out'}
                aria-label="Aantal"
              />
              <button
                className="cart-line-item__qty-btn"
                onClick={handleIncrease}
                disabled={quantity >= stock || stockStatus === 'out'}
                aria-label="Verhoog aantal"
              >
                +
              </button>
            </div>
          </div>
        )}

        <div className="cart-line-item__price-wrapper">
          <div className="cart-line-item__price">
            € {Math.floor(totalPrice)}<small>,{((totalPrice % 1) * 100).toFixed(0).padStart(2, '0')}</small>
          </div>
          <div className="cart-line-item__unit-price">
            € {unitPrice.toFixed(2)} {unit || 'per stuk'}
          </div>

          {!readonly && (
            <div className="cart-line-item__actions">
              <button
                className="cart-line-item__action-btn"
                onClick={() => onAddToList?.(product.id)}
                aria-label="Voeg toe aan lijst"
              >
                <ListPlus size={14} />
                Lijst
              </button>
              <button
                className="cart-line-item__action-btn cart-line-item__action-btn--delete"
                onClick={() => onRemove?.(id)}
                aria-label="Verwijder item"
              >
                <Trash2 size={14} />
                Verwijder
              </button>
            </div>
          )}
        </div>
      </div>
    </article>
  )
}

Important Implementation Details

  • Price Snapshot: Store price at time of add-to-cart (prevents price changes affecting existing carts)
  • Qty Validation: Min 1, max = stock level, disabled when out of stock
  • Total Calculation: Real-time calculation: unitPrice × quantity
  • Disabled State: Disable qty stepper and actions when out of stock
  • Number Input Spinners: Hide default browser spinners with CSS
  • Responsive Wrapping: Use flex-wrap: wrap on mobile
