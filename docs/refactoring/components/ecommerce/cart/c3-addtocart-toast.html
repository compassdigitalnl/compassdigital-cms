📋 STEP 1: COMPONENT OVERVIEW

Component: C3 - AddToCart Toast

Category: E-commerce / Feedback

Complexiteit: Medium (toast management, auto-dismiss, stacking)

Purpose:

Toast notification shown when item added to cart. Provides instant feedback with product info, CTAs (view cart/continue shopping), and auto-dismiss timer.

Key Features:

  • Slide-in from right (bounce-style animation)
  • Product image, success message, name, metadata
  • Two action buttons: "Naar winkelwagen" (primary), "Verder winkelen" (outline)
  • Close button (X) top-right
  • Auto-dismiss after 5 seconds with progress bar
  • Stack management (max 3 visible toasts)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • All e-commerce "Add to cart" actions
  • Non-intrusive feedback (doesn't block UI)
  • Multi-product browsing (user adds multiple items)

Best Practices:

  • Show toast + open minicart simultaneously
  • Auto-dismiss after 5s (user can dismiss manually)
  • Stack max 3 toasts (oldest auto-dismiss first)
  • Include "Continue shopping" for smooth UX

💡 STEP 7: IMPLEMENTATION NOTES

// React Component
export function ToastManager() {
  const [toasts, setToasts] = useState([]);

  const addToast = (product) => {
    const toast = { id: crypto.randomUUID(), product, createdAt: Date.now() };
    setToasts(prev => [toast, ...prev].slice(0, 3)); // Max 3

    setTimeout(() => {
      setToasts(prev => prev.filter(t => t.id !== toast.id));
    }, 5000);
  };

  return (
    <div className="toast-container">
      {toasts.map(toast => (
        <Toast key={toast.id} {...toast} onClose={() => removeToast(toast.id)} />
      ))}
    </div>
  );
}
