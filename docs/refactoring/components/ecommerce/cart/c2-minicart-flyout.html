📋 STEP 1: COMPONENT OVERVIEW

Component: C2 - MiniCart Flyout

Category: E-commerce

Complexiteit: High (slide-over panel, state management, calculations)

Purpose:

Right-side slide-over panel showing cart contents with quantity controls, free shipping progress, cross-sell suggestions, and checkout button. Allows quick cart management without leaving current page.

Key Features:

  • Slide-in from right (smooth 0.35s animation)
  • Free shipping progress bar (dynamic, goal-based)
  • Cart item cards with image, brand, name, variant, qty stepper, price
  • Hover-revealed remove button (soft delete interaction)
  • Cross-sell section ("Vaak samen gekocht")
  • Order summary with subtotal, discount, shipping, total
  • Prominent checkout button
  • Backdrop with blur effect (click to close)

🎯 STEP 2: WHEN TO USE

Ideaal voor:

  • All e-commerce sites (essential component)
  • B2B platforms with multi-item orders
  • Sites with frequent add-to-cart actions
  • Mobile-responsive shops (slide-over works well on mobile)

Niet geschikt voor:

  • Single-product checkout flows (no cart needed)
  • Subscription-only sites (direct to checkout)
  • Sites with extremely simple carts (use dropdown instead)

Best Practices:

  • Auto-open on "Add to cart" (with toast notification)
  • Show free shipping progress (encourages higher order value)
  • Display staffel/volume discounts applied
  • Include cross-sell (1-2 relevant products max)
  • Make remove button subtle (hover-only) to prevent accidents
  • Show product variants (size, color, quantity) clearly
  • Update totals instantly on quantity change

State Management (Example: Zustand):

import { create } from 'zustand';

interface CartStore {
  items: CartItem[];
  isOpen: boolean;
  addItem: (product: Product, quantity: number) => void;
  updateQty: (itemId: string, quantity: number) => void;
  removeItem: (itemId: string) => void;
  openCart: () => void;
  closeCart: () => void;
  getSubtotal: () => number;
  getTotalQty: () => number;
}

const useCartStore = create<CartStore>((set, get) => ({
  items: [],
  isOpen: false,
  addItem: (product, quantity) => {
    set(state => ({
      items: [...state.items, {
        id: crypto.randomUUID(),
        productId: product.id,
        name: product.name,
        brand: product.brand,
        price: product.price,
        quantity,
        lineTotal: product.price * quantity
      }]
    }));
  },
  updateQty: (itemId, quantity) => {
    set(state => ({
      items: state.items.map(item =>
        item.id === itemId
          ? { ...item, quantity, lineTotal: item.price * quantity }
          : item
      )
    }));
  },
  removeItem: (itemId) => {
    set(state => ({
      items: state.items.filter(item => item.id !== itemId)
    }));
  },
  openCart: () => set({ isOpen: true }),
  closeCart: () => set({ isOpen: false }),
  getSubtotal: () => {
    return get().items.reduce((sum, item) => sum + item.lineTotal, 0);
  },
  getTotalQty: () => {
    return get().items.reduce((sum, item) => sum + item.quantity, 0);
  }
}));

💡 STEP 7: IMPLEMENTATION NOTES

React Component (TypeScript):

import { ShoppingCart, X, Truck, Plus, Trash2, ArrowRight } from 'lucide-react';

export function MiniCart() {
  const {
    items,
    isOpen,
    updateQty,
    removeItem,
    closeCart
  } = useCartStore();

  const subtotal = items.reduce((sum, item) => sum + item.lineTotal, 0);
  const freeShippingThreshold = 150;
  const freeShippingProgress = Math.min(1, subtotal / freeShippingThreshold);
  const remaining = freeShippingThreshold - subtotal;

  // Prevent body scroll when open
  useEffect(() => {
    if (isOpen) {
      document.body.style.overflow = 'hidden';
    } else {
      document.body.style.overflow = '';
    }
    return () => { document.body.style.overflow = ''; };
  }, [isOpen]);

  // ESC key to close
  useEffect(() => {
    const handleEsc = (e: KeyboardEvent) => {
      if (e.key === 'Escape' && isOpen) {
        closeCart();
      }
    };
    document.addEventListener('keydown', handleEsc);
    return () => document.removeEventListener('keydown', handleEsc);
  }, [isOpen, closeCart]);

  return (
    <>
      <div
        className={`mc-backdrop ${isOpen ? 'open' : ''}`}
        onClick={closeCart}
      />
      <div className={`mc-panel ${isOpen ? 'open' : ''}`}>
        <div className="mc-header">
          <div className="mc-title">
            <ShoppingCart size={20} />
            Winkelwagen
            <span className="mc-count-badge">{items.length} items</span>
          </div>
          <button className="mc-close" onClick={closeCart}>
            <X size={18} />
          </button>
        </div>

        {remaining > 0 && (
          <div className="mc-shipping">
            <div className="mc-shipping-text">
              <Truck size={14} />
              Nog <strong>€{remaining.toFixed(2)}</strong> tot gratis verzending
            </div>
            <div className="mc-shipping-bar">
              <div
                className="mc-shipping-fill"
                style={{ width: `${freeShippingProgress * 100}%` }}
              />
            </div>
          </div>
        )}

        <div className="mc-body">
          {items.map(item => (
            <CartItem
              key={item.id}
              item={item}
              onUpdateQty={(qty) => updateQty(item.id, qty)}
              onRemove={() => removeItem(item.id)}
            />
          ))}
        </div>

        <div className="mc-footer">
          <div className="mc-summary">
            <div className="mc-summary-row">
              <span>Subtotaal</span>
              <span>€{subtotal.toFixed(2)}</span>
            </div>
            <div className="mc-summary-row total">
              <span>Totaal</span>
              <span>€{(subtotal + 6.95).toFixed(2)}</span>
            </div>
          </div>
          <button className="mc-btn-checkout">
            Afrekenen <ArrowRight size={18} />
          </button>
        </div>
      </div>
    </>
  );
}
