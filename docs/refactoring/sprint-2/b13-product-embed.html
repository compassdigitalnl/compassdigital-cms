Product Embed Block

Block Purpose: Featured product showcase met image, details, specs, en ATC button. Inline product highlight binnen content voor blog posts, landing pages, of productgidsen. Ondersteunt horizontale (2-kolom) en verticale (gestapelde) layouts.

// src/branches/blocks/ecommerce/product/ProductEmbed/types.ts
import { Product } from '@/payload-types'

export interface ProductEmbedBlock {
  blockType: 'productembed'
  product: string | Product
  layout: 'horizontal' | 'vertical'
  showSpecs: boolean
  customDescription?: any // Lexical richText
  showBadge: boolean
  customBadge?: {
    text: string
    variant: 'success' | 'info' | 'warning' | 'error' | 'pro'
  }
  buttonText: string
  showPrice: boolean
  specFields?: Array<{
    label: string
    field: string
  }>
}

export interface ProductEmbedProps {
  product: Product
  layout: 'horizontal' | 'vertical'
  showSpecs: boolean
  customDescription?: any
  showBadge: boolean
  customBadge?: {
    text: string
    variant: 'success' | 'info' | 'warning' | 'error' | 'pro'
  }
  buttonText: string
  showPrice: boolean
  specFields?: Array<{
    label: string
    field: string
  }>
}

export interface ProductBadge {
  text: string
  variant: 'success' | 'info' | 'warning' | 'error' | 'pro'
}

export interface ProductSpec {
  label: string
  value: string
}

Type Safety: Product kan zowel een ID (string) als een populated Product object zijn. Badge variant type enforces de 5 kleuren. SpecFields array voor dynamische spec display.

    // Custom badge takes precedence
    if (customBadge?.text) {

    }

    // Auto-detect badge from product
    if (product.badge) {
      return {
        text: product.badge.text,
        variant: product.badge.variant as ProductBadge['variant'],
      }
    }

Step 7: Troubleshooting

Veelvoorkomende problemen en oplossingen.

Issue Symptom Solution
Product niet gevonden Block rendert leeg, console error: "Product undefined" Controleer of product relationship correct is populated. Voeg depth: 1 toe aan findByID call in server component. Verifieer dat product ID bestaat in database.
Badge toont niet showBadge=true maar geen badge zichtbaar Check product.badge field in database. Verifieer customBadge?.text of product.isBestseller is set. Auto-detect logic vereist product.createdAt field voor "Nieuw" badge.
Specs leeg ondanks showSpecs=true Spec card toont niet, specs.length === 0 Controleer specFields array in block data. Verifieer dat product fields (bijv: product.material, product.thickness) daadwerkelijk bestaan en gevuld zijn. Check field names match exactly (case-sensitive).
Layout breekt op mobile Horizontal layout toont verkeerd op mobile Verifieer md:grid-cols-[300px_1fr] Tailwind class. Op mobile moet grid terugvallen naar 1 kolom (geen md: prefix). Check responsive breakpoints in tailwind.config.ts.
RichText rendert niet customDescription toont raw JSON of helemaal niet Importeer RichText component van @/components/RichText. Controleer Lexical editor configuratie in Payload config. Verifieer customDescription field type is 'richText' (niet 'textarea').
AddToCartButton werkt niet Button klikken doet niets, geen cart update Verifieer AddToCartButton component bestaat en correct geïmporteerd. Check ecommerce context provider is wrapped rond app. Controleer product.id en product.price zijn beschikbaar. Test cart API endpoint manually.
Image placeholder toont emoji ipv thumbnail Emoji 🩺 toont altijd, zelfs met product.thumbnail Check product.thumbnail?.url bestaat en niet null/undefined. Verifieer Next.js Image component heeft fill prop en parent heeft relative class. Check image URL is accessible (CORS, permissions).
Conditional fields niet zichtbaar in admin customBadge of specFields fields verschijnen niet Check condition functions in config.ts. Verifieer siblingData.showBadge === true voor customBadge visibility. Voor specFields: siblingData.showSpecs === true. Test met browser devtools Payload admin debug mode.
