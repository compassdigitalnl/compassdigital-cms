Principes v5.0

  • Sityzr standaardprijzen — Platform pricing conform Sityzr Pricing Model Q2 2026
  • Multistore Hub = admin-only site — dedicated beheersite (alleen /admin), geen frontend
  • Aboland + Magvilla = webshops — E-commerce Compleet (Aboland + Publishing module, Magvilla puur e-commerce)
  • Per-site facturatie — elke site = eigen tenant, eigen factuur
  • Email = volume-based — afrekenen op verzonden emails, niet op profielen
  • Modulaire add-ons voor titels — basispakket + kies wat je nodig hebt

Uitgeverslicenties

Aangesloten Titels

Elke titel = klant van Aboland. Duidelijke bundels met abonnee-limieten. Aboland verkoopt door met eigen marge.

Bundels in plaats van losse modules

Elke bundel bevat alles wat een uitgever nodig heeft: website + THOR koppeling + e-commerce + email. Geen losse modules kiezen — gewoon de bundel die past bij de omvang van de titel.

Prijs is gebaseerd op aantal actieve abonnees — de metric die uitgevers begrijpen.
